# Tambahkan Watermark ke Dokumen dengan GroupDocs.Viewer

Selamat datang di pusat tutorial GroupDocs.Viewer. Baik Anda sedang membangun aplikasi web, desktop, atau seluler, Anda akan segera menemukan betapa mudahnya **menambahkan watermark ke file dokumen** dan merendernya secara sempurna di semua platform. Dengan dukungan untuk PDF, Microsoft Office, gambar CAD, pesan email, dan lainnya, GroupDocs.Viewer memungkinkan Anda mengonversi Word ke PDF, merender PDF di Java, dan menampilkan gambar CAD—semua sambil menjaga dokumen Anda tetap aman dan konsisten dengan merek.

## Mengapa menambahkan watermark ke dokumen menggunakan GroupDocs.Viewer?

Menambahkan watermark melindungi hak kekayaan intelektual, memperkuat branding, dan memenuhi persyaratan kepatuhan. GroupDocs.Viewer menyediakan API sederhana yang memungkinkan Anda menimpa watermark teks atau gambar selama proses rendering, sehingga file asli tetap tidak tersentuh sementara setiap halaman yang dirender menampilkan tanda khusus Anda.

{{% alert color="primary" %}}
Berdayakan aplikasi .NET Anda dengan kemampuan tampilan dokumen berfidelitas tinggi. Tutorial GroupDocs.Viewer untuk .NET kami menyediakan semua yang Anda perlukan untuk mengintegrasikan penampil dokumen yang kuat. Pelajari cara merender lebih dari 170 format dokumen ke HTML, JPEG, PNG, dan PDF. Jelajahi topik lanjutan seperti merender tata letak spesifik pada gambar CAD, menangani lampiran dokumen, dan mengoptimalkan kinerja. Mulailah membangun pengalaman tampilan dokumen yang kokoh dan profesional dalam C#, ASP.NET, dan kerangka kerja .NET lainnya.
{{% /alert %}}
