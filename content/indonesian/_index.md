# Tambahkan Watermark ke Dokumen dengan GroupDocs.Viewer

Selamat datang di pusat tutorial GroupDocs.Viewer. Di sini Anda akan menemukan koleksi lengkap tutorial dan panduan untuk membantu Anda menguasai API penampil dokumen kami yang kuat untuk .NET dan Java. Baik Anda membangun aplikasi web, aplikasi desktop, atau solusi seluler, GroupDocs.Viewer memungkinkan Anda merender dan menampilkan berbagai format dokumen secara mulus, termasuk PDF, Microsoft Office (Word, Excel, PowerPoint), CAD, gambar, dan lainnya. **Anda juga dapat dengan mudah menambahkan watermark ke dokumen** untuk melindungi konten Anda dan memperkuat merek.

## Cara menambahkan watermark ke dokumen menggunakan GroupDocs.Viewer

Menambahkan watermark adalah kebutuhan umum bagi perusahaan yang perlu melindungi informasi kepemilikan atau menandai setiap halaman yang diekspor. Dengan GroupDocs.Viewer Anda dapat menerapkan watermark teks atau gambar selama proses rendering tanpa mengubah file asli. Pendekatan ini bekerja untuk PDF, file Word, spreadsheet, dan bahkan gambar CAD, memberikan lapisan perlindungan yang konsisten di semua format yang didukung.

Di bawah ini kami akan menelusuri bagian utama dari perpustakaan tutorial kami. Setiap bagian berisi panduan langkah‑demi‑langkah, potongan kode, dan tip praktik terbaik, sehingga Anda dapat dengan cepat mulai merender, mengonversi, dan menambahkan watermark pada dokumen dalam aplikasi Anda.
