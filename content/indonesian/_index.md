### Cara memulai GroupDocs Viewer di .NET
To get started with GroupDocs Viewer in .NET, install the `GroupDocs.Viewer` NuGet package, add a valid license, and reference the namespace in your project. The `Viewer` class is the core component that loads a document and provides rendering methods. Then you can instantiate the Viewer class and begin rendering documents to HTML, images, or PDF.  

Before diving into the detailed guides, make sure you have the following prerequisites:

- .NET 5/6 atau .NET Framework 4.6+ terpasang  
- Lisensi GroupDocs Viewer yang valid (atau percobaan gratis)  
- Paket NuGet `GroupDocs.Viewer` ditambahkan ke proyek Anda

**T: Apakah saya perlu menginstal perangkat lunak tambahan untuk menggunakan GroupDocs Viewer?**  
A: Tidak ada perangkat lunak eksternal yang diperlukan; API menangani semua rendering secara internal.

A: Ya, Anda dapat memberikan kata sandi saat memuat dokumen melalui API.

**T: Bagaimana caching meningkatkan kinerja?**  
A: Caching menyimpan halaman atau gambar yang telah dirender, mengurangi waktu pemrosesan untuk permintaan berikutnya.

**T: Apakah memungkinkan menambahkan watermark ke halaman yang dirender?**  
A: Tentu—tutorial khusus menunjukkan cara menambahkan watermark teks atau gambar selama proses rendering.

**T: Format file apa yang didukung secara bawaan?**  
A: Lebih dari 170 format, termasuk PDF, DOCX, XLSX, PPTX, DWG, DWF, SVG, dan banyak jenis gambar.
