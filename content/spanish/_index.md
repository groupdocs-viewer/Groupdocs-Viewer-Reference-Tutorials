# Agregar marca de agua a documento con GroupDocs.Viewer

Bienvenido al centro de tutoriales de GroupDocs.Viewer. Ya sea que estés creando una aplicación web, de escritorio o móvil, pronto descubrirás lo fácil que es **agregar marca de agua a documentos** y renderizarlos sin problemas en todas las plataformas. Con soporte para PDF, Microsoft Office, dibujos CAD, mensajes de correo electrónico y más, GroupDocs.Viewer te permite convertir Word a PDF, renderizar PDF en Java y mostrar dibujos CAD, todo mientras mantienes tus documentos seguros y coherentes con la marca.

## ¿Por qué agregar marca de agua a documentos usando GroupDocs.Viewer?

Agregar una marca de agua protege la propiedad intelectual, refuerza la marca y cumple con los requisitos de cumplimiento. GroupDocs.Viewer ofrece una API sencilla que te permite superponer marcas de agua de texto o imagen durante el proceso de renderizado, de modo que el archivo original permanezca intacto mientras cada página renderizada lleva tu marca personalizada.
