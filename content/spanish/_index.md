# Añadir marca de agua a documento con GroupDocs.Viewer

Bienvenido al centro de tutoriales de GroupDocs.Viewer. Aquí encontrará una colección completa de tutoriales y guías para ayudarle a dominar nuestras potentes API de visualización de documentos para .NET y Java. Ya sea que esté creando una aplicación web, una aplicación de escritorio o una solución móvil, GroupDocs.Viewer le permite renderizar y mostrar sin problemas una amplia variedad de formatos de documentos, incluidos PDF, Microsoft Office (Word, Excel, PowerPoint), CAD, imágenes y más. **También puede agregar fácilmente una marca de agua al documento** para proteger su contenido y reforzar la marca.

## Cómo agregar marca de agua a documento usando GroupDocs.Viewer

Agregar una marca de agua es un requisito común para las empresas que necesitan proteger información propietaria o marcar cada página exportada. Con GroupDocs.Viewer puede aplicar marcas de agua de texto o imagen durante el proceso de renderizado sin alterar el archivo original. Este enfoque funciona para PDFs, archivos Word, hojas de cálculo e incluso dibujos CAD, brindándole una capa de protección consistente en todos los formatos compatibles.

A continuación, recorreremos las secciones principales de nuestra biblioteca de tutoriales. Cada sección contiene guías paso a paso, fragmentos de código y consejos de mejores prácticas, para que pueda comenzar rápidamente a renderizar, convertir y agregar marcas de agua a documentos en sus aplicaciones.
