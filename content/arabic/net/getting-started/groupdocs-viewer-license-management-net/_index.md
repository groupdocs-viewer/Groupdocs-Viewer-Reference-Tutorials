# إتقان إدارة التراخيص في GroupDocs.Viewer لـ .NET
## دليل شامل

### مقدمة
قد يكون دمج حل عرض مستندات قوي في تطبيقات .NET أمرًا صعبًا، ولكن مع GroupDocs.Viewer لـ .NET، ستحصل على مكتبة على مستوى المؤسسة توفر إمكانيات عرض مستندات سلسة. سيرشدك هذا البرنامج التعليمي خلال إعداد وإدارة التراخيص باستخدام مسارات الملفات والموارد المضمنة في C#. بنهاية هذه المقالة، ستتقن:

![إتقان إدارة التراخيص باستخدام GroupDocs.Viewer لـ .NET](/viewer/getting-started/license-management.png)

- تعيين ترخيص GroupDocs.Viewer .NET من مسار الملف
- تحميل ترخيص من مورد مضمن ضمن مجموعة التطبيق لديك
- فهم خيارات الترخيص المختلفة لـ GroupDocs.Viewer

اكتشف كيف يمكن لهذه الأساليب تبسيط عملية التكامل الخاصة بك.

### المتطلبات الأساسية
قبل الغوص في البرنامج التعليمي، تأكد من أن لديك:

- **إطار عمل .NET** 4.7.2 أو أحدث، مطلوب بواسطة GroupDocs.Viewer.
- فهم أساسي لبنية مشروع C# و.NET.
- تم تثبيت Visual Studio لإدارة بيئة التطوير الخاصة بك بكفاءة.

## إعداد GroupDocs.Viewer لـ .NET
لبدء استخدام GroupDocs.Viewer، يجب عليك أولاً تثبيت المكتبة في تطبيق .NET. يمكنك القيام بذلك بسهولة عبر مدير حزم NuGet أو واجهة سطر أوامر .NET:

**وحدة تحكم مدير الحزم NuGet**
```bash
Install-Package GroupDocs.Viewer -Version 25.3.0
```

**.NET CLI**
```bash
dotnet add package GroupDocs.Viewer --version 25.3.0
```

### الحصول على ترخيص
قبل تهيئة المكتبة، تأكد من حصولك على الترخيص المناسب:

- **نسخة تجريبية مجانية**:احصل على ترخيص تجريبي مجاني لتقييم كافة الميزات.
- **رخصة مؤقتة**:اطلب ترخيصًا مؤقتًا لفترات تقييم أطول.
- **شراء**:للاستخدام طويل الأمد والوصول إلى الميزات الكاملة، فكر في شراء ترخيص دائم.

لتهيئة GroupDocs.Viewer باستخدام طريقة الترخيص التي اخترتها، قم بتضمين الإعداد الأساسي التالي في C#:

```csharp

    {
```

## دليل التنفيذ
### إعداد الترخيص من الملف
تتيح لك هذه الطريقة تعيين ترخيص باستخدام مسار ملف، وهي مثالية للتطبيقات التي يتم تخزين ملف الترخيص فيها بشكل منفصل أو يحتاج إلى إدارته عبر بيئات متعددة.

#### ملخص
يتضمن إعداد ترخيص من ملف التحقق من وجود ملف الترخيص ثم تطبيقه باستخدام GroupDocs.Viewer `License` فصل.

##### خطوات التنفيذ
**1. تحديد مسار الترخيص**
ابدأ بتحديد المسار إلى ملف الترخيص الخاص بك:

```csharp
string licensePath = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "YourLicense.lic");
```

**2. التحقق من وجود الملف**
تأكد من وجود ملف الترخيص قبل محاولة تعيينه:

```csharp
if (File.Exists(licensePath))
{
    License license = new License();
    license.SetLicense(licensePath);
    Console.WriteLine("License set successfully from file.");
}
else
{
    Console.WriteLine("License file not found.");
}
```

### إعداد الترخيص من المورد المضمن
بالنسبة للتطبيقات التي تريد فيها تجميع كل شيء داخل التجميع الخاص بك، فإن تحميل الترخيص كمورد مضمن هو الحل الأمثل.

#### ملخص
تقوم هذه الطريقة بتضمين ملف الترخيص في موارد مشروعك وتحميلها في وقت التشغيل.

##### خطوات التنفيذ
**1. تحديد اسم المورد**
تأكد من تعيين ملف الترخيص الخاص بك كمورد مضمن في مشروعك:

```csharp
string resourceName = "YourAssemblyName.YourLicense.lic";
```

**2. تحميل التدفق من المورد المضمن**
استرداد مجرى الموارد باستخدام الانعكاس:

```csharp
using System.Reflection;
using System.IO;

Assembly assembly = Assembly.GetExecutingAssembly();
using (Stream stream = assembly.GetManifestResourceStream(resourceName))
{
    if (stream != null)
    {
        License license = new License();
        license.SetLicense(stream);
        Console.WriteLine("License set successfully from an embedded resource.");
    }
    else
    {
        Console.WriteLine("License file not found in embedded resources.");
    }
}
```
