## دليل التنفيذ

### تصغير مستندات HTML

يؤدي تصغير حجم HTML إلى تقليل حجم الملف وتحسين أوقات التحميل، مما يجعله خطوة حاسمة لتحسين الويب.

#### الخطوة 1: تحديد دليل الإخراج
ابدأ بتحديد المكان الذي سيتم حفظ ملفاتك المصغرة فيه:

```csharp
string outputDirectory = @"YOUR_OUTPUT_DIRECTORY/";
string pageFilePathFormat = Path.Combine(outputDirectory, "page_{0}.html");
```

#### الخطوة 2: تهيئة العارض باستخدام خيار التصغير

قم بتحميل المستند وتكوين خيارات عرض HTML لتمكين التصغير:

```csharp
using (Viewer viewer = new Viewer(@"YOUR_DOCUMENT_DIRECTORY/SAMPLE_DOCX"))
{
    HtmlViewOptions options = HtmlViewOptions.ForEmbeddedResources(pageFilePathFormat);
    options.Minify = true; // تمكين تصغير HTML
    
    viewer.View(options);  // عرض المستند مع التصغير
}
```
في هذا الإعداد:
- `HtmlViewOptions` يحدد كيفية عرض المستندات.
- جلسة `options.Minify = true` يقوم بتنشيط تصغير HTML.

#### نصائح استكشاف الأخطاء وإصلاحها
- تأكد من تحديد مسارات الملفات بشكل صحيح لتجنب الاستثناءات.
- تحقق من وجود أي مشكلات تتعلق بتوافق الإصدارات بين GroupDocs وإطار عمل .NET الخاص بك.
