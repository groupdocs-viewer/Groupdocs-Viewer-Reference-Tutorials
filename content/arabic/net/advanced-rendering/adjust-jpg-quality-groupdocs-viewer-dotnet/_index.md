## دليل التنفيذ

### ضبط جودة JPG

#### ملخص
تؤثر إمكانية ضبط جودة JPG بشكل كبير على مظهر صورك. تضمن لك هذه الميزة التحكم في التوازن بين وضوح الصورة وحجم الملف.

#### دليل خطوة بخطوة
**1. تكوين خيارات العرض**
ابدأ بإنشاء مثيل لـ `JpgViewOptions`، والذي يسمح بتخصيص إعدادات الإخراج:

```csharp
using GroupDocs.Viewer;
using GroupDocs.Viewer.Options;

string outputDirectory = "YOUR_OUTPUT_DIRECTORY";
string filePath = "your-document-path";

// تهيئة العارض
using (Viewer viewer = new Viewer(filePath))
{
    JpgViewOptions options = new JpgViewOptions(outputDirectory + "/output.jpg");

    // ضبط جودة صورة JPG الناتجة
    options.Quality = 90; // تتراوح الجودة من 0 إلى 100

    viewer.View(options);
}
```
**توضيح**: 
- `JpgViewOptions`:يحدد كيفية عرض ملفات JPG.
- `Quality`: لضبط مستوى الضغط. كلما زادت القيمة، زادت جودة الملف وحجمه.

**2. تقديم المستند**
بعد تكوين خياراتك، يمكنك عرض المستند عن طريق استدعاء `View` الطريقة على `Viewer` هدف:

```csharp
viewer.View(options);
```
تعمل هذه الدعوة على معالجة المستند وتطبيق إعدادات الجودة المحددة على صورة JPG الناتجة.
