## دليل التنفيذ

### تحميل المستندات باستخدام ترميز محدد
إدارة الترميزات المختلفة أمرٌ بالغ الأهمية لعرض البيانات بدقة. لنشرح الخطوات:

#### نظرة عامة على الميزة
تتيح لك هذه الميزة تحديد الترميز عند تحميل مستند، مما يضمن عرض الأحرف بشكل صحيح.

#### تنفيذ الكود

##### الخطوة 1: إعداد المسارات ومجموعة الأحرف
أولاً، حدّد مسار ملفك ومجلد الإخراج. حدّد مجموعة الأحرف لترميز مستندك:

```java
import java.nio.charset.Charset;
import java.nio.file.Path;

String filePath = "YOUR_DOCUMENT_DIRECTORY/sample.txt"; // استبدله بمسار الملف الفعلي الخاص بك
Path outputDirectory = Path.of("YOUR_OUTPUT_DIRECTORY", "LoadDocumentsWithEncoding");
Path pageFilePathFormat = outputDirectory.resolve("page_{0}.html");

// تحديد ترميز الأحرف للمستند
Charset charset = Charset.forName("shift_jis"); 
```

##### الخطوة 2: تكوين LoadOptions
إنشاء وتكوين `LoadOptions` لاستخدام مجموعة الأحرف المحددة:

```java
import com.groupdocs.viewer.options.LoadOptions;

LoadOptions loadOptions = new LoadOptions();
loadOptions.setCharset(charset);
```

يُعلم هذا GroupDocs.Viewer كيفية تفسير نص المستند.

##### الخطوة 3: تهيئة العارض باستخدام خيارات التحميل
تهيئة `Viewer` باستخدام مسار الملف الخاص بك و `LoadOptions`. وهذا يضمن معالجة مشكلات الترميز منذ البداية:

```java
import com.groupdocs.viewer.Viewer;
import com.groupdocs.viewer.options.HtmlViewOptions;

try (Viewer viewer = new Viewer(filePath, loadOptions)) {
    HtmlViewOptions options = HtmlViewOptions.forEmbeddedResources(pageFilePathFormat);
    viewer.view(options); // عرض المستند باستخدام خيارات العرض المحددة
}
```

### شرح المعلمات
- **LoadOptions.setCharset(Charset charset):** تحدد هذه الطريقة ترميز الأحرف للمستند الخاص بك.
- **HtmlViewOptions.forEmbeddedResources(مسار pageFilePathFormat):** يقوم بتكوين كيفية عرض المستندات بتنسيق HTML باستخدام الموارد المضمنة.

### نصائح استكشاف الأخطاء وإصلاحها
- تأكد من أن الترميز المحدد يتطابق مع الترميز الفعلي للمستند الخاص بك لتجنب النص المشوه.
- تأكد من مسارات الملفات وأذونات الدليل إذا واجهت استثناءات الإدخال/الإخراج.
