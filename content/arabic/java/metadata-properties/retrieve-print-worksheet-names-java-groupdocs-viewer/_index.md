# استخراج أسماء أوراق العمل Java باستخدام GroupDocs.Viewer API

إدارة عدة أوراق عمل داخل ملفات جداول البيانات يمكن أن تكون صعبة، خاصةً عند التعامل مع مجموعات بيانات كبيرة أو أتمتة إنشاء التقارير. في هذا الدليل ستتعلم **كيفية استخراج أسماء أوراق العمل java** باستخدام GroupDocs.Viewer for Java API، وهي طريقة موثوقة لتبسيط سير عمل أتمتة المستندات.

## كيفية استخراج أسماء أوراق Excel (قائمة أوراق جدول البيانات)

فيما يلي دليل خطوة بخطوة يوضح لك كيفية استخراج أسماء أوراق العمل. يبقى الكود دون تغيير عن المثال الأصلي، مما يضمن تشغيله كما هو.

### الخطوة 1: تهيئة Viewer

ابدأ بتهيئة `Viewer` باستخدام مسار المستند الخاص بك:

```java
try (Viewer viewer = new Viewer("YOUR_DOCUMENT_DIRECTORY/three_sheets.xlsx")) {
    // Initialization code here...
}
```

هذا الكود يضبط Viewer للعمل مع ملف محدد، مع ضمان إدارة الموارد بشكل صحيح باستخدام try‑with‑resources.

### الخطوة 2: تكوين ViewInfoOptions

قم بتعيين `ViewInfoOptions` لاسترجاع معلومات العرض بصيغة HTML:

```java
ViewInfoOptions viewInfoOptions = ViewInfoOptions.forHtmlView();
viewInfoOptions.setSpreadsheetOptions(SpreadsheetOptions.forOnePagePerSheet());
```

هذا التكوين يضمن عرض كل ورقة عمل بشكل منفصل، مما يسهل التكرار على الأوراق الفردية.

### الخطوة 3: استرجاع وعرض أسماء أوراق العمل

احصل على كائن `ViewInfo` للحصول على تفاصيل حول صفحات المستند (أوراق العمل):

```java
ViewInfo viewInfo = viewer.getViewInfo(viewInfoOptions);

for (Page page : viewInfo.getPages()) {
    System.out.println(" - Worksheet " + page.getNumber() + ": '" + page.getName() + "'");
}
```

هذه الحلقة تتكرر عبر كل ورقة عمل، وتطبع رقمها واسمها. إنها جوهر عملية **java extract xlsx sheets**.
