# استخراج أسماء أوراق العمل وعرضها في Java باستخدام واجهة برمجة تطبيقات GroupDocs.Viewer

## مقدمة

قد تُشكّل إدارة أوراق عمل متعددة داخل ملفات جداول البيانات تحديًا، خاصةً عند التعامل مع مجموعات بيانات كبيرة أو أتمتة إنشاء التقارير. تُبسّط واجهة برمجة تطبيقات GroupDocs.Viewer لـ Java هذه المهمة من خلال السماح لك باسترجاع أسماء أوراق العمل برمجيًا، مما يوفر الوقت ويُحسّن سير عمل الأتمتة. سيُرشدك هذا البرنامج التعليمي خلال عملية استخدام GroupDocs.Viewer لـ Java لاستخراج أسماء أوراق العمل وعرضها من مستند جدول بيانات.

**النقاط الرئيسية:**
- إعداد بيئتك باستخدام GroupDocs.Viewer
- تهيئة العارض وتكوين الخيارات
- تقنيات لاسترجاع البيانات وتكرارها من خلال أوراق العمل بكفاءة
- أفضل الممارسات لتحسين الأداء

## المتطلبات الأساسية

لمتابعة هذا البرنامج التعليمي، تأكد من أن لديك:

- **المكتبات والتبعيات:** قم بتثبيت GroupDocs.Viewer الإصدار 25.2 أو الأحدث.
- **إعداد البيئة:** استخدم بيئة تطوير Java مثل IntelliJ IDEA أو Eclipse.
- **قاعدة المعرفة:** إن الفهم الأساسي لـ Java والتعرف على Maven لإدارة التبعيات أمر ضروري.

## إعداد GroupDocs.Viewer لـ Java

يتوفر GroupDocs.Viewer عبر Maven، مما يُسهّل تضمينه في مشاريعك. أضف التكوين التالي إلى ملفك: `pom.xml` ملف:

```xml
<repositories>
   <repository>
      <id>repository.groupdocs.com</id>
      <name>GroupDocs Repository</name>
      <url>https://releases.groupdocs.com/viewer/java/</url>
   </repository>
</repositories>
<dependencies>
   <dependency>
      <groupId>com.groupdocs</groupId>
      <artifactId>groupdocs-viewer</artifactId>
      <version>25.2</version>
   </dependency>
</dependencies>
```

### الحصول على الترخيص

يقدم GroupDocs خيارات ترخيص متنوعة، بما في ذلك نسخة تجريبية مجانية وتراخيص مؤقتة لأغراض التقييم. للوصول الكامل، يُنصح بشراء ترخيص من موقعهم الرسمي.

## دليل التنفيذ

### الميزة: استخراج أسماء أوراق العمل

توضح هذه الميزة كيفية استخراج أسماء أوراق العمل من جدول بيانات باستخدام GroupDocs.Viewer.

#### الخطوة 1: تهيئة العارض

ابدأ بالتهيئة `Viewer` مع مسار المستند الخاص بك:

```java
try (Viewer viewer = new Viewer("YOUR_DOCUMENT_DIRECTORY/three_sheets.xlsx")) {
    // كود التهيئة هنا...
}
```

تعمل هذه الكتلة على إعداد العارض للعمل مع ملف محدد، مما يضمن إدارة الموارد بشكل صحيح باستخدام try-with-resources.

#### الخطوة 2: تكوين ViewInfoOptions

تعيين `ViewInfoOptions` لاسترجاع معلومات عرض HTML:

```java
ViewInfoOptions viewInfoOptions = ViewInfoOptions.forHtmlView();
viewInfoOptions.setSpreadsheetOptions(SpreadsheetOptions.forOnePagePerSheet());
```

يضمن هذا التكوين عرض كل ورقة عمل بشكل منفصل، مما يسهل التكرار على الأوراق الفردية.

#### الخطوة 3: استرداد أسماء أوراق العمل وعرضها

احصل على `ViewInfo` كائن للحصول على تفاصيل حول صفحات المستند (أوراق العمل):

```java
ViewInfo viewInfo = viewer.getViewInfo(viewInfoOptions);

for (Page page : viewInfo.getPages()) {
    System.out.println(" - Worksheet " + page.getNumber() + ": '" + page.getName() + "'");
}
```

تتكرر هذه الحلقة في كل ورقة عمل، وتطبع الفهرس والاسم الخاص بها.
