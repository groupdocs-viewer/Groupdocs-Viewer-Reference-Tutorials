# 使用 GroupDocs.Viewer 为文档添加水印

欢迎来到 GroupDocs.Viewer 教程中心。在这里您可以找到全面的教程和指南，帮助您掌握我们强大的文档查看器 API，支持 .NET 和 Java。无论您是在构建 Web 应用、桌面应用还是移动解决方案，GroupDocs.Viewer 都能让您无缝渲染和显示多种文档格式，包括 PDF、Microsoft Office（Word、Excel、PowerPoint）、CAD、图像等。**您还可以轻松为文档添加水印**，以保护您的内容并强化品牌形象。

## 使用 GroupDocs.Viewer 为文档添加水印的方式

为文档添加水印是企业常见的需求，用于保护专有信息或为每个导出页面加上品牌标识。使用 GroupDocs.Viewer，您可以在渲染过程中应用文本或图像水印，而无需更改原始文件。此方法适用于 PDF、Word 文件、电子表格，甚至 CAD 图纸，为所有受支持的格式提供一致的保护层。

下面我们将逐步浏览教程库的主要章节。每个章节都包含一步一步的指南、代码片段和最佳实践提示，帮助您快速在应用中开始渲染、转换和添加水印。
