- **Which formats support watermarks?** 所有由 GroupDocs.Viewer 渲染的格式均支持水印，包括 PDF、DOCX、XLSX、CAD 和电子邮件消息。  
- **Do I need a license?** 生产使用需要有效的 GroupDocs.Viewer 许可证；同时提供免费试用。  
- **Can I combine watermarks with other features?** 是的——水印可以与缓存、自定义渲染和安全设置一起使用。  
- **Is it compatible with .NET Core?** 完全兼容——GroupDocs.Viewer 可在 .NET Framework、.NET Core 以及 .NET 5/6+ 上运行。

## 使用 GroupDocs.Viewer 添加水印文档

添加水印通常用于品牌标识、保密或合规需求。使用 GroupDocs.Viewer，您可以在将文档渲染为 HTML、PDF 或图像格式时应用水印。过程非常简单：

1. **创建 `WatermarkOptions` 对象** 并设置文本、颜色、不透明度和旋转角度。  
2. **传递选项** 到相应的渲染方法（例如 `RenderToPdf`、`RenderToHtml` 或 `RenderToImage`）。  
3. **渲染文档**——输出将在每页上包含水印。

此方法适用于所有受支持的文件类型，包括 **render pdf .net** 场景、CAD 图纸和电子邮件消息。

## 常见问题

**Q: 我可以在渲染为图像的文档上添加水印吗？**  
A: 是的——使用 `WatermarkOptions` 与 `RenderToImage` 结合，在每个生成的图像页上嵌入水印。

**Q: 添加水印会影响渲染性能吗？**  
A: 影响很小；GroupDocs.Viewer 会优化叠加过程，尤其在结合缓存时。

**Q: 在渲染电子邮件消息时支持水印吗？**  
A: 当然——可以在渲染的电子邮件消息的 HTML 或 PDF 输出中应用水印。

**Q: 我如何自定义水印外观？**  
A: 您可以设置字体族、大小、颜色、不透明度、旋转角度，甚至使用图像作为水印。

**Q: 能否对不同页面应用不同的水印？**  
A: 标准 API 会应用统一的水印，但您可以实现自定义渲染逻辑，以实现每页不同的水印。
