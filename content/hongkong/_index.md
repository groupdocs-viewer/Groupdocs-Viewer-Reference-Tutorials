# 在 GroupDocs.Viewer 中為文件添加水印

歡迎來到 GroupDocs.Viewer 教學中心。無論您是開發 Web、桌面或行動應用程式，都很快會發現 **為文件添加水印** 有多麼簡單，且能在各平台上完美呈現。GroupDocs.Viewer 支援 PDF、Microsoft Office、CAD 圖紙、電郵訊息等多種格式，讓您可以將 Word 轉換為 PDF、在 Java 中渲染 PDF，並顯示 CAD 圖紙——同時確保文件安全且符合品牌形象。

## 為何使用 GroupDocs.Viewer 為文件添加水印？

添加水印可保護智慧財產權、強化品牌形象，並符合合規要求。GroupDocs.Viewer 提供簡易的 API，讓您在渲染過程中覆蓋文字或圖片水印，原始檔案保持不變，而每一頁渲染結果都會帶有您的自訂標記。

為您的 .NET 應用程式賦能高保真文件檢視功能。我們的 GroupDocs.Viewer for .NET 教學提供整合強大文件檢視器所需的全部資訊。學習如何將超過 170 種文件格式渲染為 HTML、JPEG、PNG 與 PDF。探索進階主題，如在 CAD 圖紙中渲染特定版面、處理文件附件以及效能優化。立即開始在 C#、ASP.NET 及其他 .NET 框架中打造穩健且專業的文件檢視體驗。
