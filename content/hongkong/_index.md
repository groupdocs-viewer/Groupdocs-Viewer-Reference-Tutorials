## 常見問題

**Q: 使用 GroupDocs Viewer 是否需要安裝其他軟體？**  
A: 不需要額外的軟體；API 會在內部處理所有渲染。

**Q: 我可以渲染受密碼保護的文件嗎？**  
A: 可以，您可以在透過 API 載入文件時傳入密碼。

**Q: 快取如何提升效能？**  
A: 快取會儲存已渲染的頁面或圖像，減少後續請求的處理時間。

**Q: 能否在渲染的頁面上加入浮水印？**  
A: 當然可以 — 專門的教學示範如何在渲染時覆蓋文字或圖像浮水印。

**Q: 預設支援哪些檔案格式？**  
A: 超過 170 種格式，包括 PDF、DOCX、XLSX、PPTX、DWG、DWF、SVG 以及多種圖像類型。
