# GroupDocs.Viewer 教學

歡迎來到 GroupDocs.Viewer 教學中心。此處您將找到完整的教學與指南，協助您 **add watermark document** 並精通我們功能強大的 .NET 與 Java 文件檢視 API。無論您是開發 Web 應用、桌面程式或行動解決方案，GroupDocs.Viewer 都能讓您無縫渲染與顯示各種文件格式，包括 PDF、Microsoft Office（Word、Excel、PowerPoint）、CAD、圖像等。

## 快速解答
- **「add watermark document」是什麼意思？** 它指的是在已渲染文件的每一頁上覆蓋半透明的文字或圖像。  
- **哪些格式支援浮水印？** 所有由 GroupDocs.Viewer 渲染的格式，包括 PDF、DOCX、XLSX、CAD 以及電子郵件訊息。  
- **我需要授權嗎？** 生產環境必須使用有效的 GroupDocs.Viewer 授權；亦提供免費試用版。  
- **我可以將浮水印與其他功能結合使用嗎？** 可以——浮水印可與快取、自訂渲染及安全設定等一起套用。  
- **它相容於 .NET Core 嗎？** 絕對相容——GroupDocs.Viewer 支援 .NET Framework、.NET Core 以及 .NET 5/6 以上版本。

## 如何使用 GroupDocs.Viewer 加入浮水印文件

加入浮水印通常是為了品牌識別、機密性或符合法規要求。使用 GroupDocs.Viewer，您可以在將文件渲染為 HTML、PDF 或圖像格式時套用浮水印。此流程相當簡單：

1. **建立 `WatermarkOptions` 物件**，並設定文字、顏色、不透明度與旋轉角度。  
2. **將選項傳遞**給相應的渲染方法（例如 `RenderToPdf`、`RenderToHtml` 或 `RenderToImage`）。  
3. **渲染文件**——輸出結果將在每一頁包含浮水印。

此方法適用於所有支援的檔案類型，包括 **render pdf .net** 情境、CAD 圖紙與電子郵件訊息。

## 常見問題

**Q: 我可以在渲染為圖像的文件上加入浮水印嗎？**  
A: 可以——使用 `WatermarkOptions` 搭配 `RenderToImage`，即可在每一個產生的圖像頁面上嵌入浮水印。

**Q: 加入浮水印會影響渲染效能嗎？**  
A: 影響極小；GroupDocs.Viewer 會最佳化疊加過程，特別是在結合快取時。

**Q: 在渲染電子郵件訊息時支援浮水印嗎？**  
A: 絕對支援——浮水印可套用於渲染後的電子郵件訊息之 HTML 或 PDF 輸出。

**Q: 我該如何自訂浮水印外觀？**  
A: 您可以設定字型、大小、顏色、不透明度、旋轉角度，甚至使用圖像作為浮水印。

**Q: 能否對不同頁面套用不同的浮水印？**  
A: 標準 API 會套用統一的浮水印，但您可以自行實作自訂渲染邏輯，以在各頁使用不同的浮水印。
