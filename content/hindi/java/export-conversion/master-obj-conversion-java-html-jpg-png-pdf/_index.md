## कार्यान्वयन मार्गदर्शिका

GroupDocs.Viewer Java API का उपयोग करके OBJ फ़ाइलों को विभिन्न प्रारूपों में प्रस्तुत करने का तरीका जानें।

### OBJ को HTML में प्रस्तुत करना

**अवलोकन:** एम्बेडेड संसाधनों के साथ 3D मॉडल को इंटरैक्टिव, वेब-अनुकूल HTML पृष्ठों में परिवर्तित करें।

#### चरण-दर-चरण मार्गदर्शिका:
1. **आउटपुट निर्देशिका सेट करें**
   
   ```java
   Path outputDirectory = Paths.get("YOUR_OUTPUT_DIRECTORY");
   Path pageFilePathFormat = outputDirectory.resolve("obj_result.html");
   ```

2. **व्यूअर इंस्टेंस बनाएं**
   
   ```java
   try (Viewer viewer = new Viewer("YOUR_DOCUMENT_DIRECTORY/SAMPLE_OBJ")) {
       // रेंडरिंग के लिए कोड यहाँ जाएगा
   }
   ```

3. **HTML दृश्य विकल्प कॉन्फ़िगर करें**
   
   ```java
   HtmlViewOptions options = HtmlViewOptions.forEmbeddedResources(pageFilePathFormat);
   ```

4. **OBJ दस्तावेज़ प्रस्तुत करें**
   
   ```java
   viewer.view(options);
   ```

**स्पष्टीकरण:** The `HtmlViewOptions` क्लास यह सुनिश्चित करता है कि संसाधन सीधे HTML में एम्बेड किए जाएं, जिससे देखने का सहज अनुभव मिले।

### OBJ को JPG में प्रस्तुत करना

**अवलोकन:** आसान साझाकरण और प्रदर्शन के लिए 3D मॉडल को उच्च गुणवत्ता वाली JPEG छवियों में परिवर्तित करें।


3. **JPG दृश्य विकल्प कॉन्फ़िगर करें**

**अवलोकन:** 3D मॉडल को वितरण और मुद्रण के लिए उपयुक्त पेशेवर PDF दस्तावेज़ों में परिवर्तित करें।
