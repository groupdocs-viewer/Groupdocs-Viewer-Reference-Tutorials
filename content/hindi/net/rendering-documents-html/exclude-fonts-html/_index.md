# रेंडर किए गए HTML से फ़ॉन्ट्स को बाहर निकालें

## परिचय
GroupDocs.Viewer for .NET एक शक्तिशाली दस्तावेज़ रेंडरिंग लाइब्रेरी है जो डेवलपर्स को बाहरी निर्भरता की आवश्यकता के बिना अपने .NET अनुप्रयोगों में 50 से अधिक दस्तावेज़ प्रारूप प्रदर्शित करने की अनुमति देता है। इस ट्यूटोरियल में, हम GroupDocs.Viewer की एक विशिष्ट विशेषता पर ध्यान केंद्रित करेंगे: रेंडर किए गए HTML आउटपुट से फ़ॉन्ट को बाहर करना। 
## आवश्यक शर्तें
आरंभ करने से पहले, सुनिश्चित करें कि आपके पास निम्नलिखित हैं:
1. C# और .NET विकास की बुनियादी समझ।
2. .NET के लिए GroupDocs.Viewer स्थापित। आप इसे यहाँ से डाउनलोड कर सकते हैं [यहाँ](https://releases.groupdocs.com/viewer/net/).
3. C# विकास के लिए विजुअल स्टूडियो या कोई अन्य IDE.

## नामस्थान आयात करें
अपने C# कोड में, आवश्यक नामस्थानों को शामिल करना सुनिश्चित करें:
```csharp
using System;
using System.IO;
using GroupDocs.Viewer.Options;
```

## चरण 1: आउटपुट निर्देशिका परिभाषित करें
वह निर्देशिका सेट करें जहां आप रेंडर की गई HTML फ़ाइलों को सहेजना चाहते हैं.
```csharp
string outputDirectory = "Your Document Directory";
```
## चरण 2: पृष्ठ फ़ाइल पथ प्रारूप निर्धारित करें
प्रस्तुत दस्तावेज़ के अलग-अलग पृष्ठों के फ़ाइल पथों के लिए प्रारूप निर्दिष्ट करें।
```csharp
string pageFilePathFormat = Path.Combine(outputDirectory, "page_{0}.html");
```
## चरण 3: व्यूअर ऑब्जेक्ट को आरंभ करें
व्यूअर ऑब्जेक्ट को उस दस्तावेज़ के साथ इंस्टैंसिएट करें जिसे आप रेंडर करना चाहते हैं।
```csharp
using (Viewer viewer = new Viewer("YourDocumentPath"))
{
    // आपका कोड यहां जाएगा
}
```
## चरण 4: HTML दृश्य विकल्प सेट करें
HTML रेंडरिंग के लिए विकल्प परिभाषित करें, जिसमें एम्बेडेड संसाधनों और बहिष्कृत किए जाने वाले फ़ॉन्ट का प्रारूप शामिल हो।
```csharp
HtmlViewOptions options = HtmlViewOptions.ForEmbeddedResources(pageFilePathFormat);
options.FontsToExclude.Add("Arial");
```
## चरण 5: दस्तावेज़ प्रस्तुत करें
दस्तावेज़ को रेंडर करने के लिए HTML दृश्य विकल्पों को व्यूअर ऑब्जेक्ट में पास करें।
```csharp
viewer.View(options);
```
## चरण 6: आउटपुट रेंडर किए गए दस्तावेज़ का स्थान
उपयोगकर्ता को उस स्थान के बारे में सूचित करें जहां रेंडर की गई HTML फ़ाइलें सहेजी गई हैं।
```csharp
Console.WriteLine($"\nSource document rendered successfully.\nCheck output in {outputDirectory}.");
```

## निष्कर्ष
इस ट्यूटोरियल में, हमने सीखा है कि रेंडर किए गए HTML आउटपुट से फ़ॉन्ट को बाहर करने के लिए .NET के लिए GroupDocs.Viewer का उपयोग कैसे करें। ऊपर बताए गए चरणों का पालन करके, आप अपनी विशिष्ट आवश्यकताओं को पूरा करने के लिए रेंडरिंग प्रक्रिया को अनुकूलित कर सकते हैं, जिससे आपके अनुप्रयोगों में दस्तावेज़ों का इष्टतम प्रदर्शन सुनिश्चित हो सके।
## अक्सर पूछे जाने वाले प्रश्न
### क्या मैं रेंडर किए गए HTML से एकाधिक फ़ॉन्ट्स को बाहर कर सकता हूँ?
हां, आप इसमें एकाधिक फ़ॉन्ट नाम जोड़ सकते हैं `FontsToExclude` HTML दृश्य विकल्पों में सूची बनाएँ.
### क्या GroupDocs.Viewer सभी .NET फ्रेमवर्क के साथ संगत है?
हां, GroupDocs.Viewer .NET Framework 4.6.1 और उच्चतर का समर्थन करता है।
### क्या मैं दूरस्थ भंडारण स्थानों से दस्तावेज़ प्रस्तुत कर सकता हूँ?
हां, GroupDocs.Viewer स्थानीय भंडारण के साथ-साथ दूरस्थ भंडारण स्थानों और धाराओं से दस्तावेज़ों को प्रस्तुत करने का समर्थन करता है।
### क्या GroupDocs.Viewer HTML आउटपुट के लिए उत्तरदायी डिज़ाइन का समर्थन करता है?
हां, आप HTML दृश्य विकल्पों को तदनुसार समायोजित करके उत्तरदायी रेंडरिंग सक्षम कर सकते हैं।
### क्या GroupDocs.Viewer के लिए तकनीकी सहायता उपलब्ध है?
हां, आप सहायता ले सकते हैं और चर्चा में भाग ले सकते हैं। [GroupDocs.व्यूअर फ़ोरम](https://forum.groupdocs.com/c/viewer/9).
