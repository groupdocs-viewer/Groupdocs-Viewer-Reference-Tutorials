## परिचय

इस गाइड के अंत तक, आप अपने अनुप्रयोगों में इस सुविधा को लागू करने और अनुकूलित करने के लिए सुसज्जित हो जाएँगे। आइए पहले कुछ पूर्वापेक्षाएँ संबोधित करें।

## आउटलुक डेटा फ़ाइलों से दृश्य जानकारी पुनर्प्राप्त करना

यह सुविधा आपको आउटलुक डेटा फ़ाइलों से सीधे फ़ाइल प्रकार और पृष्ठ संख्या जैसी महत्वपूर्ण जानकारी निकालने की अनुमति देती है।

### 1. व्यूअर ऑब्जेक्ट को प्रारंभ करें

इसका एक उदाहरण बनाएं `Viewer` अपने दस्तावेज़ पथ के साथ क्लास:

```csharp
string documentPath = @"YOUR_DOCUMENT_DIRECTORY\/";
using (Viewer viewer = new Viewer(documentPath))
{
    // आगे की प्रक्रिया यहां होगी
}
```

### 2. दृश्य जानकारी विकल्प कॉन्फ़िगर करें

विशिष्ट दृश्य जानकारी प्राप्त करने के लिए, कॉन्फ़िगर करें `ViewInfoOptions` HTML रेंडरिंग के लिए:

```csharp
ViewInfoOptions options = ViewInfoOptions.ForHtmlView();
```

### 3. OutlookViewInfo प्राप्त करें

उपयोग `GetViewInfo()` दृश्य जानकारी प्राप्त करने और उसे डालने की विधि `OutlookViewInfo`:

```csharp
OutlookViewInfo rootFolderInfo = viewer.GetViewInfo(options) as OutlookViewInfo;
```

### 4. फ़ाइल प्रकार और पृष्ठ संख्या तक पहुँच

फ़ाइल प्रकार और पृष्ठ जानकारी निकालें:

```csharp
string fileType = "File type is: " + rootFolderInfo.FileType;
string pagesCount = "Pages count: " + rootFolderInfo.Pages.Count;
```

### 5. फ़ोल्डरों के माध्यम से पुनरावृति करें

Outlook डेटा फ़ाइल के भीतर फ़ोल्डरों के माध्यम से लूप करें:

```csharp
foreach (string folder in rootFolderInfo.Folders)
{
    // प्रत्येक फ़ोल्डर को आवश्यकतानुसार संसाधित करें
}
```
