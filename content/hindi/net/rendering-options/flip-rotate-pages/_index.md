# पेज पलटें और घुमाएँ

## परिचय
इस ट्यूटोरियल में, हम .NET के लिए Groupdocs.Viewer की कार्यक्षमताओं पर गहराई से चर्चा करेंगे, विशेष रूप से पृष्ठों को फ़्लिप करने और घुमाने पर ध्यान केंद्रित करेंगे। .NET के लिए Groupdocs.Viewer एक शक्तिशाली उपकरण है जिसे .NET अनुप्रयोगों के भीतर विभिन्न प्रारूपों में दस्तावेज़ों को प्रस्तुत करने के लिए डिज़ाइन किया गया है। चाहे आप कोई दस्तावेज़ प्रबंधन प्रणाली विकसित कर रहे हों या अपने सॉफ़्टवेयर में दस्तावेज़ देखने की क्षमताओं को एकीकृत करने की आवश्यकता हो, .NET के लिए Groupdocs.Viewer एक कुशल समाधान प्रदान करता है।
## आवश्यक शर्तें
शुरू करने से पहले, सुनिश्चित करें कि आपके पास निम्नलिखित पूर्वापेक्षाएँ निर्धारित हैं:
### .NET के लिए Groupdocs.Viewer स्थापित करना
.NET के लिए Groupdocs.Viewer का उपयोग करने के लिए, आपको NuGet पैकेज मैनेजर के माध्यम से पैकेज स्थापित करना होगा। आप विस्तृत स्थापना निर्देश यहाँ पा सकते हैं [प्रलेखन](https://tutorials.groupdocs.com/viewer/net/).

## नामस्थान आयात करें
सुनिश्चित करें कि आपने .NET के लिए Groupdocs.Viewer का प्रभावी ढंग से उपयोग करने के लिए अपने प्रोजेक्ट में आवश्यक नामस्थान आयात किए हैं।
```csharp
using System;
using System.IO;
using GroupDocs.Viewer.Options;
```

आइए .NET के लिए Groupdocs.Viewer का उपयोग करके पृष्ठों को फ़्लिप करने और घुमाने की प्रक्रिया को सरल चरणों में विभाजित करें:
## चरण 1: आउटपुट निर्देशिका और फ़ाइल पथ सेट करें
वह निर्देशिका निर्धारित करें जहां आप आउटपुट फ़ाइल को सहेजना चाहते हैं तथा आउटपुट फ़ाइल पथ निर्दिष्ट करें।
```csharp
string outputDirectory = "Your Document Directory";
string outputFilePath = Path.Combine(outputDirectory, "output.pdf");
```
## चरण 2: व्यूअर ऑब्जेक्ट को आरंभ करें
जिस दस्तावेज़ को आप देखना चाहते हैं उसका पथ पास करके Viewer वर्ग का एक उदाहरण बनाएँ।
```csharp
using (Viewer viewer = new Viewer(TestFiles.SAMPLE_DOCX))
```
## चरण 3: दृश्य विकल्प कॉन्फ़िगर करें
दृश्य विकल्प सेट करें, जैसे आउटपुट फ़ाइल प्रारूप निर्दिष्ट करना और पृष्ठ रोटेशन जैसी कोई अतिरिक्त सेटिंग।
```csharp
PdfViewOptions viewOptions = new PdfViewOptions(outputFilePath);
viewOptions.RotatePage(1, Rotation.On90Degree);
```
## चरण 4: दस्तावेज़ प्रस्तुत करें
Viewer ऑब्जेक्ट की View विधि को लागू करें और दृश्य विकल्प पास करें।
```csharp
viewer.View(viewOptions);
```
## चरण 5: सफलता संदेश प्रदर्शित करें
उपयोगकर्ता को सूचित करें कि दस्तावेज़ सफलतापूर्वक प्रस्तुत किया गया है और सत्यापन के लिए आउटपुट निर्देशिका निर्दिष्ट करें।
```csharp
Console.WriteLine($"\nSource document rendered successfully.\nCheck output in {outputDirectory}.");
```

## निष्कर्ष
निष्कर्ष में, .NET के लिए Groupdocs.Viewer दस्तावेज़ों को प्रस्तुत करने के लिए शक्तिशाली क्षमताएँ प्रदान करता है, जिसमें पृष्ठों को फ़्लिप करना और घुमाना शामिल है। इस ट्यूटोरियल में बताए गए चरणों का पालन करके, आप इन सुविधाओं को अपने .NET अनुप्रयोगों में सहजता से एकीकृत कर सकते हैं, जिससे आपके उपयोगकर्ताओं के लिए दस्तावेज़ देखने का अनुभव बेहतर हो सकता है।
