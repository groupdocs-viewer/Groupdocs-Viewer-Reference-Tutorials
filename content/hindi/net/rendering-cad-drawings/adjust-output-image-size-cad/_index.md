# CAD चित्रों के लिए आउटपुट छवि का आकार समायोजित करें

## परिचय
CAD ड्रॉइंग को अक्सर इष्टतम दृश्य और प्रस्तुति के लिए विशिष्ट समायोजन की आवश्यकता होती है। GroupDocs.Viewer for .NET CAD ड्रॉइंग आउटपुट को प्रबंधित और अनुकूलित करने के लिए एक शक्तिशाली टूलसेट प्रदान करता है। इस ट्यूटोरियल में, हम आपको CAD ड्रॉइंग के लिए आउटपुट इमेज आकार को चरण दर चरण समायोजित करने की प्रक्रिया के माध्यम से मार्गदर्शन करेंगे।
## आवश्यक शर्तें
आरंभ करने से पहले, सुनिश्चित करें कि आपके पास निम्नलिखित पूर्वापेक्षाएँ हैं:
1. .NET के लिए GroupDocs.Viewer: .NET के लिए GroupDocs.Viewer को डाउनलोड करें और इंस्टॉल करें [यहाँ](https://releases.groupdocs.com/viewer/net/).
2. दस्तावेज़ निर्देशिका: वह निर्देशिका तैयार करें जहाँ आपका दस्तावेज़ स्थित है।
3. बुनियादी समझ: .NET प्रोग्रामिंग की बुनियादी अवधारणाओं से खुद को परिचित कराएं।

## नामस्थान आयात करें
सबसे पहले, GroupDocs.Viewer कार्यक्षमताओं तक पहुँचने के लिए आवश्यक नामस्थानों को आयात करना सुनिश्चित करें:
```csharp
using System;
using System.IO;
using GroupDocs.Viewer.Options;
```
## चरण 1: आउटपुट निर्देशिका सेट करें
वह निर्देशिका निर्धारित करें जहां आप CAD चित्रों की आउटपुट छवियों को संग्रहीत करना चाहते हैं:
```csharp
string outputDirectory = "Your Document Directory";
```
## चरण 2: पृष्ठ फ़ाइल पथ प्रारूप निर्धारित करें
पेज फ़ाइल पथों के लिए प्रारूप सेट करें। इस प्रारूप का उपयोग अलग-अलग पेजों को HTML फ़ाइलों के रूप में नाम देने और सहेजने के लिए किया जाएगा:
```csharp
string pageFilePathFormat = Path.Combine(outputDirectory, "page_{0}.html");
```
## चरण 3: छवि का आकार समायोजित करें
व्यूअर ऑब्जेक्ट के लिए उपयोग ब्लॉक के अंदर, उपयुक्त विकल्प सेट करके CAD रेखाचित्रों के लिए छवि का आकार समायोजित करें:
```csharp
using (Viewer viewer = new Viewer(TestFiles.SAMPLE_DWG_WITH_LAYOUTS_AND_LAYERS))
{
    HtmlViewOptions options = HtmlViewOptions.ForEmbeddedResources(pageFilePathFormat);
    options.CadOptions = CadOptions.ForRenderingByScaleFactor(0.5f);
    
    viewer.View(options);
}
```
## चरण 4: आउटपुट निर्देशिका प्रदर्शित करें
दस्तावेज़ को रेंडर करने के बाद, सफल रेंडरिंग को इंगित करने वाला एक संदेश प्रदर्शित करें और आउटपुट निर्देशिका का स्थान प्रदान करें:
```csharp
Console.WriteLine($"\nSource document rendered successfully.\nCheck output in {outputDirectory}.");
```

## निष्कर्ष
CAD ड्रॉइंग के लिए आउटपुट इमेज का आकार समायोजित करना उनकी दृश्यता और उपयोगिता को बढ़ाने के लिए महत्वपूर्ण है। .NET के लिए GroupDocs.Viewer के साथ, यह प्रक्रिया सुव्यवस्थित और कुशल हो जाती है, जिससे आप अपनी विशिष्ट आवश्यकताओं के अनुसार आउटपुट को अनुकूलित कर सकते हैं।
## अक्सर पूछे जाने वाले प्रश्न
### क्या मैं CAD चित्रों के अलावा अन्य प्रकार के दस्तावेज़ों के लिए आउटपुट छवि का आकार समायोजित कर सकता हूँ?
हां, .NET के लिए GroupDocs.Viewer विभिन्न दस्तावेज़ प्रकारों का समर्थन करता है, और आप अधिकांश दस्तावेज़ प्रारूपों के लिए आउटपुट छवि आकार समायोजित कर सकते हैं।
### क्या GroupDocs.Viewer for .NET .NET फ्रेमवर्क के विभिन्न संस्करणों के साथ संगत है?
हां, GroupDocs.Viewer for .NET .NET फ्रेमवर्क के कई संस्करणों के साथ संगत है, जो विभिन्न वातावरणों में लचीलापन और प्रयोज्यता सुनिश्चित करता है।
### क्या .NET के लिए GroupDocs.Viewer के लिए कोई लाइसेंसिंग विकल्प उपलब्ध हैं?
हां, आप अपनी आवश्यकताओं के अनुरूप अस्थायी लाइसेंस और वाणिज्यिक लाइसेंस सहित विभिन्न लाइसेंसिंग विकल्पों पर विचार कर सकते हैं।
### क्या मैं प्रस्तुत दस्तावेजों के आउटपुट प्रारूप को अनुकूलित कर सकता हूँ?
बिल्कुल, .NET के लिए GroupDocs.Viewer विभिन्न अनुकूलन विकल्प प्रदान करता है, जिससे आप अपने ptutorialss के अनुसार आउटपुट प्रारूप को तैयार कर सकते हैं।
### मैं .NET के लिए GroupDocs.Viewer के साथ अतिरिक्त समर्थन या सहायता कहां पा सकता हूं?
आप GroupDocs.Viewer फ़ोरम पर जा सकते हैं [यहाँ](https://forum.groupdocs.com/c/viewer/9) समर्थन प्राप्त करने, प्रश्न पूछने और समुदाय के साथ जुड़ने के लिए।
