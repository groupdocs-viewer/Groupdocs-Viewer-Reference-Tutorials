# GroupDocs.Viewer .NET का उपयोग करके Apple Numbers को HTML, JPG, PNG, PDF में बदलें

## परिचय

क्या आप अपने Apple Numbers दस्तावेज़ों को HTML, JPG, PNG, और PDF जैसे विभिन्न स्वरूपों में सहजता से परिवर्तित करना चाहते हैं? यह मार्गदर्शिका उन डेवलपर्स के लिए तैयार की गई है जिन्हें दस्तावेज़ रेंडरिंग क्षमताओं को एकीकृत करने की आवश्यकता है और जो डेटा एक्सेसिबिलिटी का विस्तार करने में रुचि रखते हैं। GroupDocs.Viewer .NET के साथ, आप Apple Numbers फ़ाइलों को कुशलतापूर्वक रेंडर कर सकते हैं।

![.NET के लिए GroupDocs.Viewer के साथ Apple Numbers को HTML, JPG, PNG, PDF में कनवर्ट करें](/viewer/export-conversion/convert-apple-numbers-html-jpg-png-pdf.png)

**आप क्या सीखेंगे:**
- .NET के लिए GroupDocs.Viewer कैसे सेट करें।
- Apple Numbers दस्तावेज़ों को HTML, JPG, PNG, और PDF प्रारूपों में परिवर्तित करने के चरण-दर-चरण निर्देश।
- आपके दस्तावेज़ रेंडरिंग प्रक्रिया को अनुकूलित करने के लिए प्रमुख कॉन्फ़िगरेशन विकल्प।
- वास्तविक दुनिया के परिदृश्यों में इस सुविधा के व्यावहारिक अनुप्रयोग।

सबसे पहले, आइए कार्यान्वयन विवरण में जाने से पहले आवश्यक पूर्वापेक्षाओं की समीक्षा करें।

## कार्यान्वयन मार्गदर्शिका
अब जब आपने GroupDocs.Viewer .NET सेट कर लिया है, तो आइए जानें कि Apple Numbers दस्तावेज़ों को विभिन्न फ़ॉर्मेट में कैसे रेंडर किया जाए। प्रत्येक अनुभाग आपको एक विशिष्ट सुविधा के माध्यम से मार्गदर्शन करेगा।

### Apple Numbers दस्तावेज़ को HTML में प्रस्तुत करें
**अवलोकन:** आसानी से देखने और साझा करने के लिए अपनी Apple Numbers फ़ाइल को एम्बेडेड संसाधनों के साथ HTML दस्तावेज़ में परिवर्तित करें।

#### चरण 1: आउटपुट निर्देशिका परिभाषित करें
```csharp
string outputDirectory = "YOUR_OUTPUT_DIRECTORY";
```

#### चरण 2: व्यूअर इंस्टेंस सेट अप करें
```csharp
using (Viewer viewer = new Viewer("YOUR_DOCUMENT_DIRECTORY/SAMPLE_NUMBERS"))
{
    // कोड स्निपेट जारी है...
}
```
*क्यों?* एक का उपयोग करना `using` यह कथन संसाधनों के उचित निपटान को सुनिश्चित करता है, तथा मेमोरी लीक को रोकता है।

#### चरण 3: HTML दृश्य विकल्प कॉन्फ़िगर करें
```csharp
HtmlViewOptions options = HtmlViewOptions.ForEmbeddedResources(pageFileFullPath);
```

#### चरण 4: दस्तावेज़ प्रस्तुत करें
```csharp
viewer.View(options);
```
*क्यों?* यह चरण आपके दस्तावेज़ को एक सुलभ वेब प्रारूप में बदल देता है।

**समस्या निवारण सुझाव:** रनटाइम त्रुटियों से बचने के लिए सुनिश्चित करें कि सभी फ़ाइल पथ सही और सुलभ हैं।

### Apple Numbers दस्तावेज़ को JPG में प्रस्तुत करें
**अवलोकन:** अपनी Apple Numbers फ़ाइल को उच्च गुणवत्ता वाली JPEG छवि में बदलें, जो प्रस्तुतियों या साझा करने के लिए आदर्श है।

#### चरण 1: आउटपुट पथ परिभाषित करें
```csharp
string pageFileFullPath = Path.Combine("YOUR_OUTPUT_DIRECTORY", "Numbers_result.jpg");
```

#### चरण 2: व्यूअर और विकल्प सेट करें
```csharp
using (Viewer viewer = new Viewer("YOUR_DOCUMENT_DIRECTORY/SAMPLE_NUMBERS"))
{
    JpgViewOptions options = new JpgViewOptions(pageFileFullPath);
    // कोड स्निपेट जारी है...
}
```

#### चरण 3: दस्तावेज़ प्रस्तुत करें
```csharp
viewer.View(options);
```
*क्यों?* JPG में कनवर्ट करने से विभिन्न प्लेटफार्मों और उपकरणों पर संगतता सुनिश्चित होती है।

### Apple Numbers दस्तावेज़ को PNG में प्रस्तुत करें
**अवलोकन:** दोषरहित गुणवत्ता के लिए अपने दस्तावेज़ को PNG छवि में परिवर्तित करें।

#### चरण:
1. **आउटपुट पथ परिभाषित करें**
2. **सेटअप व्यूअर और विकल्प**
3. **दस्तावेज़ प्रस्तुत करें**

```csharp
PngViewOptions options = new PngViewOptions(pageFileFullPath);
viewer.View(options);
```

### Apple Numbers दस्तावेज़ को PDF में प्रस्तुत करें
**अवलोकन:** अपने Apple Numbers दस्तावेज़ से एक पोर्टेबल PDF फ़ाइल बनाएँ।

#### चरण:
1. **आउटपुट पथ परिभाषित करें**
2. **सेटअप व्यूअर और विकल्प**
3. **दस्तावेज़ प्रस्तुत करें**

```csharp
PdfViewOptions options = new PdfViewOptions(pageFileFullPath);
viewer.View(options);
```
