# Λήψη συντεταγμένων κειμένου για απόδοση εικόνας

## Εισαγωγή
Το GroupDocs.Viewer για .NET είναι ένα ισχυρό API απόδοσης εγγράφων που επιτρέπει στους προγραμματιστές να αποδίδουν απρόσκοπτα έγγραφα σε διάφορες μορφές όπως PDF, Microsoft Office και πολλές άλλες. Μία από τις βασικές λειτουργίες του είναι η δυνατότητα εξαγωγής συντεταγμένων κειμένου για ακριβή απόδοση εικόνας.

## Βήμα 3: Επανάληψη σελίδων
Περιηγηθείτε σε κάθε σελίδα του εγγράφου για να αποκτήσετε πρόσβαση σε γραμμές κειμένου, λέξεις και χαρακτήρες.
```csharp
foreach (Page page in viewInfo.Pages)
{
    Console.WriteLine($"Page: {page.Number}");
    Console.WriteLine("Text lines/words/characters:");
    foreach (Line line in page.Lines)
    {
        Console.WriteLine(line);
        foreach (Word word in line.Words)
        {
            Console.WriteLine("\t" + word);
            foreach (Character character in word.Characters)
                Console.WriteLine("\t\t" + character);
        }
    }
}
```
