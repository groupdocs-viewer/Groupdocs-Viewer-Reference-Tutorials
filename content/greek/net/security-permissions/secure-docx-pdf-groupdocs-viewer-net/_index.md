## Οδηγός Εφαρμογής

### Απόδοση ενός DOCX σε ένα προστατευμένο PDF
Το κύριο χαρακτηριστικό που εξερευνούμε είναι η απόδοση αρχείων DOCX σε PDF με ενσωματωμένη προστασία. Αυτό περιλαμβάνει τον ορισμό κωδικών πρόσβασης για το άνοιγμα του εγγράφου και τον ορισμό δικαιωμάτων, όπως η άρνηση εκτύπωσης.

#### Βήμα 1: Ορισμός καταλόγων εξόδου και εισόδου
Ορίστε κατάλληλα τις διαδρομές των αρχείων σας:

```csharp
string outputDirectory = "YOUR_OUTPUT_DIRECTORY";
string filePath = Path.Combine(outputDirectory, "output.pdf");
```

#### Βήμα 2: Αρχικοποίηση του Viewer με ένα έγγραφο DOCX
Χρησιμοποιήστε το `Viewer` κλάση για να φορτώσετε το έγγραφό σας:

```csharp
using (Viewer viewer = new Viewer("YOUR_DOCUMENT_DIRECTORY/SAMPLE_DOCX"))
{
    // Περαιτέρω επεξεργασία θα γίνει εδώ.
}
```

#### Βήμα 3: Ρύθμιση ρυθμίσεων ασφαλείας
Διαμορφώστε λειτουργίες ασφαλείας, όπως κωδικούς πρόσβασης και δικαιώματα:

```csharp
Security security = new Security
{
    DocumentOpenPassword = "o123", // Απαιτείται κωδικός πρόσβασης για το άνοιγμα του PDF
    PermissionsPassword = "p123",  // Κωδικός πρόσβασης δικαιωμάτων
    Permissions = Permissions.AllowAll ^ Permissions.DenyPrinting // Άρνηση άδειας εκτύπωσης
};
```

#### Βήμα 4: Ορίστε τις επιλογές προβολής για την απόδοση σε PDF με τις ρυθμίσεις ασφαλείας
Καθορίστε τις προτιμήσεις απόδοσης και τις διαμορφώσεις ασφαλείας σας:

```csharp
PdfViewOptions options = new PdfViewOptions(filePath)
{
    Security = security
};
```

#### Βήμα 5: Απόδοση του εγγράφου σε προστατευμένο αρχείο PDF
Τέλος, εκτελέστε τη μέθοδο προβολής για να αποδώσετε και να προστατεύσετε το έγγραφό σας:

```csharp
viewer.View(options);
```
