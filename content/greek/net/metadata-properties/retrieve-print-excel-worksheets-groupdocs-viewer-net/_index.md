Θα αναλύσουμε τη διαδικασία ανάκτησης και εκτύπωσης ονομάτων φύλλων εργασίας σε διαχειρίσιμα βήματα.

### Λειτουργία: Ανάκτηση και εκτύπωση ονομάτων φύλλων εργασίας
Αυτή η λειτουργία εστιάζει στην εξαγωγή και εμφάνιση όλων των ονομάτων φύλλων εργασίας από ένα έγγραφο Excel χρησιμοποιώντας το GroupDocs.Viewer για .NET. Ακολουθήστε τα παρακάτω βήματα υλοποίησης:

#### Βήμα 1: Αρχικοποίηση του Viewer με μια διαδρομή αρχείου
Ξεκινήστε αρχικοποιώντας το `Viewer` αντικείμενο με τη διαδρομή αρχείου του υπολογιστικού φύλλου σας.
```csharp
string filePath = "YOUR_DOCUMENT_DIRECTORY/sample.xlsx";
using (Viewer viewer = new Viewer(filePath))
{
    // Προχωρήστε στο επόμενο βήμα...
}
```

#### Βήμα 2: Ρύθμιση του ViewInfoOptions για την προβολή HTML
Ρύθμιση παραμέτρων `ViewInfoOptions` για να ρυθμίσετε την προβολή HTML του υπολογιστικού φύλλου σας. Αυτή η ρύθμιση παραμέτρων είναι απαραίτητη για τη σωστή απόδοση του εγγράφου.
```csharp
ViewInfoOptions viewInfoOptions = ViewInfoOptions.ForHtmlView();
viewInfoOptions.SpreadsheetOptions = SpreadsheetOptions.ForOnePagePerSheet(); // Κάθε φύλλο ως μία σελίδα.
```

#### Βήμα 3: Ανάκτηση πληροφοριών προβολής
Αποκτήστε το `ViewInfo` αντικείμενο, το οποίο περιέχει λεπτομέρειες σχετικά με τη δομή και τις σελίδες του εγγράφου.
```csharp
ViewInfo viewInfo = viewer.GetViewInfo(viewInfoOptions);
Console.WriteLine("Worksheets:");
```

#### Βήμα 4: Επαναλάβετε κάθε σελίδα και εκτυπώστε τα ονόματα των φύλλων εργασίας
Τέλος, επαναλάβετε κάθε σελίδα για να εξαγάγετε και να εκτυπώσετε ονόματα φύλλων εργασίας.
```csharp
foreach (Page page in viewInfo.Pages)
{
    Console.WriteLine($" - Worksheet {page.Number} name '{page.Name}'");
}
```
