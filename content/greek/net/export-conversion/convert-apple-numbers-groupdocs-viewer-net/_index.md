**Επισκόπηση:** Μετατρέψτε το αρχείο Apple Numbers σε έγγραφο HTML με ενσωματωμένους πόρους για εύκολη προβολή και κοινή χρήση.

#### Βήμα 1: Ορισμός καταλόγου εξόδου
```csharp
string outputDirectory = "YOUR_OUTPUT_DIRECTORY";
```

#### Βήμα 2: Ρύθμιση στιγμιότυπου προβολής
```csharp
using (Viewer viewer = new Viewer("YOUR_DOCUMENT_DIRECTORY/SAMPLE_NUMBERS"))
{
    // Το απόσπασμα κώδικα συνεχίζεται...
}
```
*Γιατί;* Χρησιμοποιώντας ένα `using` Η δήλωση διασφαλίζει την ορθή διάθεση των πόρων, αποτρέποντας τις διαρροές μνήμης.

#### Βήμα 3: Ρύθμιση παραμέτρων επιλογών προβολής HTML
```csharp

```

#### Βήμα 4: Απόδοση του εγγράφου
```csharp

```
*Γιατί;* Αυτό το βήμα μετατρέπει το έγγραφό σας σε μια προσβάσιμη μορφή ιστού.

**Συμβουλή αντιμετώπισης προβλημάτων:** Βεβαιωθείτε ότι όλες οι διαδρομές αρχείων είναι σωστές και προσβάσιμες για να αποφύγετε σφάλματα χρόνου εκτέλεσης.

### Απόδοση εγγράφου Apple Numbers σε JPG
**Επισκόπηση:** Μετατρέψτε το αρχείο Apple Numbers σε εικόνα JPEG υψηλής ποιότητας, ιδανική για παρουσιάσεις ή κοινή χρήση.

#### Βήμα 1: Ορισμός διαδρομής εξόδου
```csharp
string pageFileFullPath = Path.Combine("YOUR_OUTPUT_DIRECTORY", "Numbers_result.jpg");
```

#### Βήμα 2: Ρύθμιση Προβολής και Επιλογών
```csharp

{

    // Το απόσπασμα κώδικα συνεχίζεται...
}
```

#### Βήμα 3: Απόδοση του εγγράφου
```csharp
viewer.View(options);
```
*Γιατί;* Η μετατροπή σε JPG διασφαλίζει τη συμβατότητα σε διάφορες πλατφόρμες και συσκευές.

### Απόδοση εγγράφου Apple Numbers σε PNG
**Επισκόπηση:** Μετατρέψτε το έγγραφό σας σε εικόνα PNG για ποιότητα χωρίς απώλειες.
