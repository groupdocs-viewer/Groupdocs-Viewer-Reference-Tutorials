### Απόδοση περιοχών εκτύπωσης στο Excel

#### Επισκόπηση

Η απόδοση συγκεκριμένων περιοχών εκτύπωσης βελτιώνει την απόδοση εστιάζοντας σε συγκεκριμένα τμήματα δεδομένων, βελτιώνοντας την εμπειρία χρήστη για μεγάλα σύνολα δεδομένων.

#### Βήμα προς βήμα εφαρμογή

**1. Ρυθμίστε το περιβάλλον σας**

Αρχικά, βεβαιωθείτε ότι οι διαδρομές εξόδου και εγγράφου έχουν οριστεί σωστά:

```csharp
string outputDirectory = Path.Combine("YOUR_OUTPUT_DIRECTORY");
string pageFilePathFormat = Path.Combine(outputDirectory, "page_{0}.html");
```

**2. Αρχικοποίηση του αντικειμένου προβολής**

Δημιουργήστε ένα `Viewer` αντικείμενο για το αρχείο Excel σας:

```csharp
using (Viewer viewer = new Viewer(Path.Combine("YOUR_DOCUMENT_DIRECTORY", "SAMPLE_XLSX_WITH_PRINT_AREAS")))
{
    // Συνέχεια με την εγκατάσταση...
}
```

**3. Ρύθμιση παραμέτρων επιλογών προβολής HTML**

Στήνω `HtmlViewOptions` για ενσωματωμένους πόρους:

```csharp
HtmlViewOptions options = HtmlViewOptions.ForEmbeddedResources(pageFilePathFormat);
```

**4. Περιοχές εκτύπωσης μόνο για απόδοση**

Ρυθμίστε τις επιλογές για απόδοση μόνο των περιοχών εκτύπωσης χρησιμοποιώντας `SpreadsheetOptions`:

```csharp
options.SpreadsheetOptions = SpreadsheetOptions.ForRenderingPrintArea();
```

**5. Εκτέλεση απόδοσης**

Χρησιμοποιήστε το `View` μέθοδος για τη δημιουργία της εξόδου:

```csharp
viewer.View(options);
```

#### Συμβουλές αντιμετώπισης προβλημάτων
- Βεβαιωθείτε ότι οι διαδρομές έχουν οριστεί σωστά τόσο για τους καταλόγους εισόδου όσο και για τους καταλόγους εξόδου.
- Βεβαιωθείτε ότι το αρχείο Excel περιέχει καθορισμένες περιοχές εκτύπωσης, εάν θέλετε να αποδώσετε μόνο συγκεκριμένες ενότητες.
