## Οδηγός Εφαρμογής
### Λήψη πληροφοριών προβολής εγγράφου
Ανακτήστε αναλυτικές λεπτομέρειες για συγκεκριμένες προβολές, όπως τον αριθμό σελίδων και τις διαθέσιμες επιλογές προβολής, από το έγγραφό σας χρησιμοποιώντας το GroupDocs.Viewer για Java.

#### Επισκόπηση
Ο στόχος είναι η λήψη λεπτομερών πληροφοριών σχετικά με τις προβολές ενός εγγράφου.

#### Βήμα προς βήμα εφαρμογή
**1. Αρχικοποίηση του προγράμματος προβολής**
Ρυθμίστε το `Viewer` κλάση με τη διαδρομή προς το έγγραφό σας:

```java
import com.groupdocs.viewer.Viewer;
import com.groupdocs.viewer.options.ViewInfoOptions;
import com.groupdocs.viewer.results.ViewInfo;

public class FeatureGetViewInfo {
    public static void main(String[] args) {
        // Καθορίστε τη διαδρομή προς το έγγραφο εισαγωγής.
        String filePath = "YOUR_DOCUMENT_DIRECTORY/SAMPLE_PDF";
        
        // Αρχικοποιήστε το ViewInfoOptions για προβολή HTML.
        ViewInfoOptions viewInfoOptions = ViewInfoOptions.forHtmlView();

        try (Viewer viewer = new Viewer(filePath)) {
            // Ανακτήστε πληροφορίες προβολής σχετικά με το έγγραφο χρησιμοποιώντας τις καθορισμένες επιλογές.
            ViewInfo info = viewer.getViewInfo(viewInfoOptions);
            
            // Το αντικείμενο info περιέχει πλέον λεπτομέρειες όπως τον αριθμό σελίδων και τις διαθέσιμες προβολές.
        }
    }
}
```

**2. Κατανόηση Παραμέτρων και Μεθόδων**
- **`ViewInfoOptions.forHtmlView()`:** Ρυθμίζει τις παραμέτρους της προβολής για την ανάκτηση πληροφοριών που αφορούν συγκεκριμένα την HTML.
- **`viewer.getViewInfo(viewInfoOptions)`:** Ανακτά λεπτομερείς πληροφορίες προβολής με βάση τις παρεχόμενες επιλογές.

#### Βασικές επιλογές διαμόρφωσης
- Τροποποιώ `ViewInfoOptions` για διαφορετικές μορφές όπως PDF ή προβολές εικόνων χρησιμοποιώντας αντίστοιχες μεθόδους όπως `.forPdfView()`.

### Συμβουλές αντιμετώπισης προβλημάτων
- Βεβαιωθείτε ότι η διαδρομή του εγγράφου είναι σωστή για να αποφύγετε σφάλματα "το αρχείο δεν βρέθηκε".
- Επαληθεύστε ότι οι εξαρτήσεις Maven έχουν ρυθμιστεί σωστά για να αποτρέψετε προβλήματα διαδρομής κλάσης.
