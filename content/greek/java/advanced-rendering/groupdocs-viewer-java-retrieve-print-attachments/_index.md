## Πώς να Ανακτήσετε Συνημμένα Java

### Βήμα 1: Αρχικοποίηση του Αντικειμένου Viewer
Αρχικά, δημιουργήστε μια παρουσία `Viewer` που δείχνει στο έγγραφο που περιέχει τα συνημμένα. Η χρήση ενός μπλοκ *try‑with‑resources* εγγυάται ότι ο viewer κλείνει αυτόματα, διατηρώντας την εφαρμογή σας καθαρή και αποτρέποντας διαρροές μνήμης.

```java
import com.groupdocs.viewer.Viewer;
import java.util.List;

// Define the path to your document containing attachments
String documentPath = "YOUR_DOCUMENT_DIRECTORY/SAMPLE_MSG_WITH_ATTACHMENTS";

try (Viewer viewer = new Viewer(documentPath)) {
    // Code for retrieving and printing attachments will go here
} catch (Exception e) {
    e.printStackTrace();
}
```

### Βήμα 2: Ανάκτηση Συνημμένων
Με τον viewer έτοιμο, καλέστε `getAttachments()` για να εξάγετε κάθε ενσωματωμένο αρχείο από το πηγαίο έγγραφο. Η μέθοδος επιστρέφει ένα `List<Attachment>` που μπορείτε να διασχίσετε, φιλτράρετε ή μεταβιβάσετε απευθείας σε άλλες υπηρεσίες.

```java
// Retrieve all attachments from the specified document
List<Attachment> attachments = viewer.getAttachments();
```

### Βήμα 3: Εκτύπωση Λεπτομερειών Συνημμένου
Πριν την εκτύπωση, είναι χρήσιμο να καταγράψετε τα μεταδεδομένα κάθε συνημμένου — όνομα, μέγεθος και τύπο περιεχομένου — ώστε να γνωρίζετε ακριβώς με τι εργάζεστε.

```java
// Iterate through each attachment and print its details
for (Attachment attachment : attachments) {
    System.out.println(attachment);
}
```
