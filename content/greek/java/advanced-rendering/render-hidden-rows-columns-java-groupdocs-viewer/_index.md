## Πώς να Μετατρέψετε Excel σε HTML με Κρυφά Δεδομένα
Αυτή η ενότητα σας καθοδηγεί βήμα προς βήμα στις ακριβείς ενέργειες για τη **μετατροπή Excel σε HTML** διασφαλίζοντας ότι οι κρυμμένες γραμμές και στήλες εμφανίζονται.

### Βήμα 1: Ορισμός Διαδρομής Καταλόγου Εξόδου
Ξεκινήστε ορίζοντας πού θα αποθηκευτούν τα αποδοθέντα αρχεία σας:
```java
import java.nio.file.Path;
import java.nio.file.Paths;

Path outputDirectory = Paths.get("YOUR_OUTPUT_DIRECTORY", "RenderHiddenRowsAndColumns");
```

### Βήμα 2: Διαμόρφωση HTMLViewOptions
Στη συνέχεια, ρυθμίστε το `HtmlViewOptions` ώστε να ενσωματώνει τους πόρους απευθείας στα παραγόμενα αρχεία HTML:
```java
import com.groupdocs.viewer.options.HtmlViewOptions;

// Create a file path format for rendering each page.
Path pageFilePathFormat = outputDirectory.resolve("page_{0}.html");

HtmlViewOptions viewOptions = HtmlViewOptions.forEmbeddedResources(pageFilePathFormat);
```

### Βήμα 3: Ενεργοποίηση Απόδοσης Κρυμμένων Στηλών και Γραμμών
Διαμορφώστε το `SpreadsheetOptions` για την απόδοση των κρυμμένων στοιχείων:
```java
// Enable rendering of hidden columns and rows.
viewOptions.getSpreadsheetOptions().setRenderHiddenColumns(true);
viewOptions.getSpreadsheetOptions().setRenderHiddenRows(true);
```

### Βήμα 4: Αρχικοποίηση Viewer με Έγγραφο και Απόδοση
Τέλος, αρχικοποιήστε το GroupDocs.Viewer με τη διαδρομή του εγγράφου σας και αποδώστε το περιεχόμενο:
```java
import com.groupdocs.viewer.Viewer;

try (Viewer viewer = new Viewer("YOUR_DOCUMENT_DIRECTORY/SAMPLE_XLSX_WITH_HIDDEN_ROW_AND_COLUMN")) {
    // Render the document to HTML using the specified view options.
    viewer.view(viewOptions);
} catch (Exception e) {
    System.out.println("Error rendering document: " + e.getMessage());
}
```

**Συμβουλές Επίλυσης Προβλημάτων**: Βεβαιωθείτε ότι οι διαδρομές είναι σωστά ορισμένες και ότι οι εξαρτήσεις περιλαμβάνονται σωστά στο `pom.xml` σας.
