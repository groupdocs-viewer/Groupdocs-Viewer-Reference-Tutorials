## Οδηγός Εφαρμογής

### Δυνατότητα: Προσαρμογή ποιότητας εικόνων JPG σε PDF

Αυτή η λειτουργία εστιάζει στην τροποποίηση της ανάλυσης και της ποιότητας των εικόνων JPG κατά τη μετατροπή εγγράφων όπως παρουσιάσεων σε μορφή PDF χρησιμοποιώντας το GroupDocs.Viewer.

#### Βήμα 1: Ορισμός διαδρομής καταλόγου εξόδου

Ξεκινήστε επιλύοντας τον κατάλογο εξόδου όπου θα αποθηκευτεί το PDF που έχετε μετατρέψει:

```java
import java.nio.file.Path;
import java.nio.file.Paths;

public class FeatureResolveOutputDirectoryPath {
    public static Path getOutputDirectoryPath(String subdirectory) {
        String directory = Paths.get("YOUR_OUTPUT_DIRECTORY", "AdjustQualityOfJpgImages", subdirectory).toString();
        
        try {
            return Paths.get(directory);
        } catch (IOException e) {
            throw new RuntimeException("Failed to create output directory.", e);
        }
    }
}
```

#### Βήμα 2: Ρύθμιση παραμέτρων του PdfViewOptions

Δημιουργήστε μια παρουσία του `PdfViewOptions` και καθορίστε την επιθυμητή ποιότητα για εικόνες JPG:

```java
import com.groupdocs.viewer.Viewer;
import com.groupdocs.viewer.options.PdfViewOptions;

public class FeatureAdjustQualityOfJpgImages {
    public static void run() {
        Path outputDirectory = FeatureResolveOutputDirectoryPath.getOutputDirectoryPath("YOUR_DOCUMENT_DIRECTORY");
        Path filePath = outputDirectory.resolve("output.pdf");

        PdfViewOptions viewOptions = new PdfViewOptions(filePath);
        
        // Ορίστε την επιθυμητή ποιότητα JPG (κλίμακα 0-100)
        byte quality = 10;
        viewOptions.setJpgQuality(quality);

        try (Viewer viewer = new Viewer("YOUR_DOCUMENT_DIRECTORY/source.pptx")) {
            viewer.view(viewOptions);
        }
    }
}
```

**Εξήγηση:**
- `setJpgQuality(byte quality)`: Προσαρμόζει την ποιότητα των εικόνων JPG στο PDF εξόδου. Οι χαμηλότερες τιμές μειώνουν το μέγεθος του αρχείου αλλά και την καθαρότητα της εικόνας.
