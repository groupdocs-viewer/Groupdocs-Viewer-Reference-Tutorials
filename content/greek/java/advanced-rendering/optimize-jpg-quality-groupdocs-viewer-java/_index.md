# Μείωση Μεγέθους PDF Java – Βελτιστοποίηση Ποιότητας JPG με το GroupDocs Viewer

Η ισορροπία μεταξύ του μεγέθους αρχείου και της οπτικής πιστότητας είναι μια κοινή πρόκληση όταν εργάζεστε με PDFs. Σε αυτόν τον οδηγό θα ανακαλύψετε πώς να **reduce PDF size Java** προσαρμόζοντας την ποιότητα των εικόνων JPG μέσα σε έγγραφα PDF χρησιμοποιώντας το GroupDocs.Viewer for Java. Θα περάσουμε από τη ρύθμιση, την υλοποίηση κώδικα και πρακτικές συμβουλές ώστε να μπορείτε με σιγουριά να συμπιέζετε τις εικόνες PDF χωρίς να θυσιάζετε την αναγνωσιμότητα.

![Βελτιστοποίηση Ποιότητας JPG σε PDFs με το GroupDocs.Viewer for Java](/viewer/advanced-rendering/optimize-jpg-quality-in-pdfs.png)

## Γρήγορες Απαντήσεις
- **Τι σημαίνει “reduce PDF size Java”;** Προσαρμογή της ποιότητας εικόνας, της συμπίεσης και της διαχείρισης πόρων για τη δημιουργία μικρότερων αρχείων PDF σε εφαρμογές Java.  
- **Ποια ρύθμιση ελέγχει την ποιότητα JPG;** `PdfViewOptions.setJpgQuality(byte quality)` όπου η τιμή κυμαίνεται από 0 (χαμηλότερη) έως 100 (υψηλότερη).  
- **Μπορώ επίσης να μετατρέψω PPTX σε PDF στην ίδια ροή;** Ναι—απλώς δείξτε το `Viewer` σε μια πηγή `.pptx` και οι ίδιες επιλογές εφαρμόζονται.  
- **Ποιο επίπεδο ποιότητας είναι τυπικό για δημοσίευση στο web;** Μια τιμή γύρω στα **50‑70** παρέχει καλή ισορροπία για τις περισσότερες περιπτώσεις web.  
- **Χρειάζομαι άδεια για αυτή τη λειτουργία;** Μια δωρεάν δοκιμή λειτουργεί για αξιολόγηση· απαιτείται μόνιμη άδεια για χρήση σε παραγωγή.

## Τι είναι το “reduce PDF size Java”;
Η μείωση του μεγέθους PDF σε Java περιλαμβάνει τη βελτιστοποίηση των πόρων μέσα σε ένα PDF—ιδιαίτερα των εικόνων—ώστε το τελικό αρχείο να καταλαμβάνει λιγότερο χώρο αποθήκευσης και να φορτώνει πιο γρήγορα. Με τη μείωση της ποιότητας JPG, συμπιέζετε αποτελεσματικά **PDF images**, που συχνά αποτελούν το μεγαλύτερο μέρος του μεγέθους ενός εγγράφου.

## Γιατί να προσαρμόσετε την ποιότητα JPG με το GroupDocs Viewer;
- **Σημαντική μείωση μεγέθους**: Η μείωση της ποιότητας εικόνας μπορεί να μειώσει τα PDFs κατά 30‑70 % ανάλογα με την αρχική ανάλυση.  
- **Μετατροπή σε μία διεργασία**: Δεν χρειάζεται ξεχωριστό βήμα επεξεργασίας εικόνας· το GroupDocs το διαχειρίζεται κατά τη δημιουργία του PDF.  
- **Ευελιξία**: Μπορείτε να ρυθμίσετε ακριβώς την τιμή ποιότητας `byte` ανάλογα με τις απαιτήσεις του έργου (π.χ., εκτύπωση υψηλής ποιότητας vs. ελαφρύ προεπισκόπηση web).

## Οδηγός Υλοποίησης

### Βήμα 1: Επίλυση της Διαδρομής του Καταλόγου Εξόδου
Δημιουργήστε μια βοηθητική κλάση που δημιουργεί το φάκελο εξόδου όπου θα αποθηκευτεί το PDF.

```java
import java.nio.file.Path;
import java.nio.file.Paths;

public class FeatureResolveOutputDirectoryPath {
    public static Path getOutputDirectoryPath(String subdirectory) {
        String directory = Paths.get("YOUR_OUTPUT_DIRECTORY", "AdjustQualityOfJpgImages", subdirectory).toString();
        
        try {
            return Paths.get(directory);
        } catch (IOException e) {
            throw new RuntimeException("Failed to create output directory.", e);
        }
    }
}
```

### Βήμα 2: Διαμόρφωση του `PdfViewOptions` με την Επιθυμητή Ποιότητα JPG
Ορίστε το επίπεδο ποιότητας JPG (0‑100) πριν από την απόδοση του εγγράφου.

```java
import com.groupdocs.viewer.Viewer;
import com.groupdocs.viewer.options.PdfViewOptions;

public class FeatureAdjustQualityOfJpgImages {
    public static void run() {
        Path outputDirectory = FeatureResolveOutputDirectoryPath.getOutputDirectoryPath("YOUR_DOCUMENT_DIRECTORY");
        Path filePath = outputDirectory.resolve("output.pdf");

        PdfViewOptions viewOptions = new PdfViewOptions(filePath);
        
        // Set desired JPG quality (0-100 scale)
        byte quality = 10;
        viewOptions.setJpgQuality(quality);

        try (Viewer viewer = new Viewer("YOUR_DOCUMENT_DIRECTORY/source.pptx")) {
            viewer.view(viewOptions);
        }
    }
}
```

**Επεξήγηση:**  
- `setJpgQuality(byte quality)` λέει στο GroupDocs πόση συμπίεση να εφαρμόσει στις εικόνες JPG. Οι χαμηλότερες τιμές παράγουν μικρότερα αρχεία αλλά μπορεί να μειώσουν την οπτική ευκρίνεια.  
- Το παράδειγμα χρησιμοποιεί το `source.pptx` για να δείξει **convert pptx to pdf** ενώ ταυτόχρονα συμπιέζει τις εικόνες.

### Βήμα 3: Εκτελέστε τον Κώδικα και Επαληθεύστε το Αποτέλεσμα
Εκτελέστε το `FeatureAdjustQualityOfJpgImages.run()`. Το παραγόμενο `output.pdf` θα περιέχει εικόνες JPG στο επίπεδο ποιότητας που καθορίσατε, συμπιέζοντας αποτελεσματικά **PDF images** και μειώνοντας το συνολικό μέγεθος του αρχείου.
