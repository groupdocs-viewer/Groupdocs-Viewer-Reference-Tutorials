# Πώς να συμπιέσετε pdf σε Java – Βελτιστοποίηση Ποιότητας JPG

Η ισορροπία μεταξύ του μεγέθους του αρχείου και της οπτικής πιστότητας αποτελεί κοινή πρόκληση όταν εργάζεστε με PDF. Σε αυτό το σεμινάριο θα ανακαλύψετε **πώς να συμπιέσετε pdf** ρυθμίζοντας την ποιότητα εικόνας JPG μέσα σε έγγραφα PDF χρησιμοποιώντας το GroupDocs.Viewer for Java. Θα περάσουμε από τη ρύθμιση, την υλοποίηση κώδικα και πρακτικές συμβουλές, ώστε να μπορείτε με σιγουριά να συμπιέζετε τις εικόνες PDF χωρίς να θυσιάζετε την αναγνωσιμότητα.

![Optimize JPG Quality in PDFs with GroupDocs.Viewer for Java](/viewer/advanced-rendering/optimize-jpg-quality-in-pdfs.png)

## Γρήγορες Απαντήσεις
- **Τι σημαίνει “reduce PDF size Java”;** Ρύθμιση της ποιότητας εικόνας, συμπίεση και διαχείριση πόρων για τη δημιουργία μικρότερων αρχείων PDF σε εφαρμογές Java.  
- **Ποια ρύθμιση ελέγχει την ποιότητα JPG;** `PdfViewOptions.setJpgQuality(byte quality)` όπου η τιμή κυμαίνεται από 0 (χαμηλότερη) έως 100 (υψηλότερη).  
- **Μπορώ επίσης να μετατρέψω PPTX σε PDF στην ίδια ροή;** Ναι—απλώς κατευθύνετε το `Viewer` σε μια πηγή `.pptx` και οι ίδιες επιλογές ισχύουν.  
- **Ποιο επίπεδο ποιότητας είναι τυπικό για δημοσίευση στο web;** Μια τιμή γύρω στο **50‑70** προσφέρει καλή ισορροπία για τις περισσότερες διαδικτυακές περιπτώσεις.  
- **Χρειάζομαι άδεια για αυτή τη λειτουργία;** Μια δωρεάν δοκιμή λειτουργεί για αξιολόγηση· απαιτείται μόνιμη άδεια για παραγωγική χρήση.

## Οδηγός Υλοποίησης

### Βήμα 1: Επίλυση Διαδρομής Καταλόγου Εξόδου
Δημιουργήστε μια βοηθητική κλάση που δημιουργεί το φάκελο εξόδου όπου θα αποθηκευτεί το PDF.

```java
import java.nio.file.Path;
import java.nio.file.Paths;

public class FeatureResolveOutputDirectoryPath {
    public static Path getOutputDirectoryPath(String subdirectory) {
        String directory = Paths.get("YOUR_OUTPUT_DIRECTORY", "AdjustQualityOfJpgImages", subdirectory).toString();
        
        try {
            return Paths.get(directory);
        } catch (IOException e) {
            throw new RuntimeException("Failed to create output directory.", e);
        }
    }
}
```

### Βήμα 2: Διαμόρφωση `PdfViewOptions` με την Επιθυμητή Ποιότητα JPG
Ορίστε το επίπεδο ποιότητας JPG (0‑100) πριν από την απόδοση του εγγράφου.

```java
import com.groupdocs.viewer.Viewer;
import com.groupdocs.viewer.options.PdfViewOptions;

public class FeatureAdjustQualityOfJpgImages {
    public static void run() {
        Path outputDirectory = FeatureResolveOutputDirectoryPath.getOutputDirectoryPath("YOUR_DOCUMENT_DIRECTORY");
        Path filePath = outputDirectory.resolve("output.pdf");

        PdfViewOptions viewOptions = new PdfViewOptions(filePath);
        
        // Set desired JPG quality (0-100 scale)
        byte quality = 10;
        viewOptions.setJpgQuality(quality);

        try (Viewer viewer = new Viewer("YOUR_DOCUMENT_DIRECTORY/source.pptx")) {
            viewer.view(viewOptions);
        }
    }
}
```

**Επεξήγηση:**  
- `setJpgQuality(byte quality)` λέει στο GroupDocs πόση συμπίεση να εφαρμόσει στις εικόνες JPG. Οι χαμηλότερες τιμές παράγουν μικρότερα αρχεία αλλά μπορεί να μειώσουν την οπτική ευκρίνεια.  
- Το παράδειγμα χρησιμοποιεί το `source.pptx` για να δείξει **convert pptx to pdf** ενώ ταυτόχρονα συμπιέζει τις εικόνες.

### Βήμα 3: Εκτέλεση του Κώδικα και Επαλήθευση του Αποτελέσματος
Εκτελέστε `FeatureAdjustQualityOfJpgImages.run()`. Το παραγόμενο `output.pdf` θα περιέχει εικόνες JPG στο επίπεδο ποιότητας που καθορίσατε, **συμπιέζοντας τις εικόνες PDF** και μειώνοντας το συνολικό μέγεθος του αρχείου.
