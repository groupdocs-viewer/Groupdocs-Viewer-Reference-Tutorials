## Οδηγός Εφαρμογής
Αυτή η ενότητα σας καθοδηγεί στην υλοποίηση λειτουργιών για την απόδοση ενός εγγράφου με επίπεδο κειμένου χρησιμοποιώντας το GroupDocs.Viewer.

### Απόδοση εγγράφου με επίπεδο κειμένου
Αυτή η λειτουργία σάς επιτρέπει να εξαγάγετε κείμενο και να το επικαλύψετε σε μια εικόνα του εγγράφου σας, καθιστώντας το περιεχόμενο οπτικά ελκυστικό και αναζητήσιμο. Δείτε πώς:

#### Βήμα 1: Ορισμός καταλόγου εξόδου
Αρχικά, καθορίστε πού θα αποθηκευτούν οι εικόνες εξόδου σας ορίζοντας μια διαδρομή καταλόγου εξόδου.

```java
Path outputDirectory = Paths.get("YOUR_OUTPUT_DIRECTORY");
```

Βεβαιωθείτε ότι ο κατάλογος υπάρχει ή δημιουργείται κατά τη διάρκεια του χρόνου εκτέλεσης για να αποφύγετε σφάλματα.

#### Βήμα 2: Ρύθμιση παραμέτρων επιλογών προβολής
Στη συνέχεια, διαμορφώστε τις επιλογές προβολής για να αποδίδετε έγγραφα ως εικόνες PNG με ενεργοποιημένη την εξαγωγή κειμένου:

```java
Path pageFilePathFormat = outputDirectory.resolve("page_{0}.png");
PngViewOptions viewOptions = new PngViewOptions(pageFilePathFormat);
viewOptions.setExtractText(true);  // Ενεργοποίηση εξαγωγής κειμένου πάνω από την εικόνα
```

Εδώ, `PngViewOptions` καθορίζει ότι θέλουμε να αποδώσουμε εικόνες σε μορφή PNG. Η μέθοδος `setExtractText(true)` λέει στο GroupDocs.Viewer να επικαλύψει το εξαγόμενο κείμενο σε αυτές τις εικόνες.

#### Βήμα 3: Απόδοση του εγγράφου
Τέλος, χρησιμοποιήστε μια παρουσία Viewer για να εκτελέσετε τη λειτουργία απόδοσης:

```java
try (Viewer viewer = new Viewer("YOUR_DOCUMENT_DIRECTORY/SAMPLE_DOCX")) {
    viewer.view(viewOptions);  // Εκτέλεση λειτουργίας απόδοσης
}
```

Αυτό το μπλοκ κώδικα ανοίγει το έγγραφό σας και εφαρμόζει τις επιλογές προβολής που έχουν διαμορφωθεί προηγουμένως. `try-with-resources` Η δήλωση διασφαλίζει την ορθή διαχείριση των πόρων.

- **Προβλήματα δικαιωμάτων**: Επαληθεύστε τα δικαιώματα εγγραφής για τον κατάλογο εξόδου.
- **Διένεξεις εκδόσεων**Βεβαιωθείτε ότι έχετε την έκδοση GroupDocs.Viewer στο Maven σας `pom.xml` ταιριάζει με αυτό που σκοπεύετε να χρησιμοποιήσετε.
