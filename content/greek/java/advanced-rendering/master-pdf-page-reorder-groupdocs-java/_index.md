## Πώς να αλλάξετε τη σειρά σελίδων PDF χρησιμοποιώντας το GroupDocs.Viewer για Java

### Βήμα 1: Αρχικοποίηση Viewer και Options
Δημιουργήστε ένα αντικείμενο `Viewer` και διαμορφώστε το `PdfViewOptions` με τη διαδρομή του αρχείου προορισμού.

```java
import com.groupdocs.viewer.Viewer;
import com.groupdocs.viewer.options.PdfViewOptions;

import java.nio.file.Path;
import java.nio.file.Paths;

public class ReorderPagesFeature {
    public static void main(String[] args) {
        Path outputDirectory = Paths.get("YOUR_OUTPUT_DIRECTORY");
        Path outputFilePath = outputDirectory.resolve("output.pdf");

        PdfViewOptions viewOptions = new PdfViewOptions(outputFilePath);
```

### Βήμα 2: Ορισμός Επιθυμητής Σειράς Σελίδων
Περάστε τους αριθμούς σελίδων στη μέθοδο `view` με τη σειρά που θέλετε να εμφανιστούν. Σε αυτό το παράδειγμα αποδίδουμε πρώτα τη σελίδα 2, μετά τη σελίδα 1.

```java
try (Viewer viewer = new Viewer("YOUR_DOCUMENT_DIRECTORY/SAMPLE_DOCX")) {
    // Reorder pages: render page 2 first, then page 1
    viewer.view(viewOptions, 2, 1);
}
```

#### Επεξήγηση
- **`PdfViewOptions`** – ελέγχει τις ρυθμίσεις εξόδου για τη διαδικασία απόδοσης PDF.  
- **`viewer.view(viewOptions, 2, 1)`** – λέει στο Viewer να δημιουργήσει το PDF με τη σελίδα 2 πριν τη σελίδα 1, αλλάζοντας ουσιαστικά **τη σειρά σελίδων PDF**.

### Βήμα 3: Εκτέλεση και Επαλήθευση
Εκτελέστε τη μέθοδο `main`. Το παραγόμενο `output.pdf` θα περιέχει τις σελίδες στην προσαρμοσμένη ακολουθία που ορίσατε. Ανοίξτε το PDF σε οποιονδήποτε προβολέα για να επιβεβαιώσετε τη σειρά.

## Πώς εντάσσεται αυτό σε μια ροή εργασίας docx to pdf java;
