## Πώς να μετατρέψετε docx σε html java με το GroupDocs.Viewer

Η μετατροπή αρχείων DOCX σε HTML είναι συχνή απαίτηση όταν χρειάζεστε περιεχόμενο έτοιμο για το web ενώ διατηρείτε το στυλ, τις εικόνες και τους εξωτερικούς πόρους. Το GroupDocs.Viewer for Java απλοποιεί αυτή τη διαδικασία με μία κλήση API, επιτρέποντάς σας να εστιάσετε στην ενσωμάτωση αντί στην ανάλυση χαμηλού επιπέδου.

Τυπικά βήματα περιλαμβάνουν:

1. **Initialize the Viewer** – παρέχετε την άδειά σας και δημιουργήστε το αντικείμενο `Viewer`.  
2. **Load the DOCX file** – δώστε ένα `File` ή `InputStream`.  
3. **Configure rendering options** – ενεργοποιήστε τη διαχείριση εξωτερικών πόρων, ορίστε την ποιότητα εικόνας και επιλέξτε τη μορφή εξόδου.  
4. **Execute the conversion** – καλέστε `viewer.render` με `HtmlOptions`.  
5. **Process the result** – αποθηκεύστε τα αρχεία HTML και τυχόν εξαγόμενους πόρους στην επιθυμητή τοποθεσία.

Αυτά τα βήματα παρουσιάζονται στον πρώτο σύνδεσμο tutorial παρακάτω, ο οποίος δείχνει επίσης πώς να διαχειριστείτε εξωτερικές εικόνες και αρχεία CSS.

## Πώς να περιστρέψετε συγκεκριμένες σελίδες pdf χρησιμοποιώντας το GroupDocs.Viewer Java

Μερικές φορές χρειάζεται να περιστρέψετε μόνο ορισμένες σελίδες ενός PDF—π.χ. ένα σαρωμένο τιμολόγιο που είναι ανάποδα ή ένα σχέδιο που απαιτεί οριζόντια προσανατολισμό. Το GroupDocs.Viewer Java το κάνει απλό:

* Δημιουργήστε ένα αντικείμενο `PdfOptions`.  
* Χρησιμοποιήστε `setPages` για να ορίσετε τους αριθμούς σελίδων που θέλετε να περιστρέψετε.  
* Εφαρμόστε `setRotationAngle` (90°, 180° ή 270°) μόνο για αυτές τις σελίδες.  
* Καλέστε `viewer.render` με τις ρυθμισμένες επιλογές.

Αυτή η προσέγγιση αποφεύγει την επανα‑απόδοση ολόκληρου του εγγράφου και διατηρεί τον χρόνο επεξεργασίας χαμηλό—ιδανική για εφαρμογές κρίσιμης απόδοσης.

## Συχνές Ερωτήσεις

**Ε: Μπορώ να χρησιμοποιήσω το GroupDocs.Viewer για να μετατρέψω DOCX σε HTML σε εφαρμογή Spring Boot;**  
Α: Ναι. Αρχικοποιήστε το bean `Viewer` με την άδειά σας, στη συνέχεια καλέστε `viewer.render` με `HtmlOptions` μέσα σε οποιαδήποτε υπηρεσία ή controller.

**Ε: Πώς η βιβλιοθήκη διαχειρίζεται μεγάλα PDF όταν αποδίδει σε εικόνες;**  
Α: Χρησιμοποιήστε `PdfOptions` για να ενεργοποιήσετε την απόδοση σελίδα‑με‑σελίδα και ρυθμίστε `setCacheFolder` για αποθήκευση ενδιάμεσων αποτελεσμάτων, μειώνοντας την πίεση στη μνήμη.

**Ε: Είναι δυνατόν να αποδοθούν μόνο επιλεγμένες σελίδες ενός εγγράφου;**  
Α: Απόλυτα. Ορίστε τη συλλογή `pages` στο `RenderOptions` με τους συγκεκριμένους αριθμούς σελίδων που χρειάζεστε.

**Ε: Ποιες μορφές μπορούν να αποδοθούν σε HTML με ενσωματωμένους πόρους;**  
Α: Υποστηρίζονται DOCX, PPTX, XLSX, PDF και πολλά άλλα. Χρησιμοποιήστε `HtmlOptions.setResourcesPath` για να ελέγξετε πού αποθηκεύονται οι εικόνες και τα CSS.

**Ε: Το GroupDocs.Viewer υποστηρίζει πολυ‑νηματική απόδοση;**  
Α: Ναι, αλλά κάθε instance του `Viewer` πρέπει να χρησιμοποιείται ανά νήμα ή πρέπει να εφαρμόσετε σωστό συγχρονισμό για να αποφύγετε συνθήκες αγώνα.
