## Οδηγός Εφαρμογής

### Διαίρεση σχεδίου σε πλακίδια
Αυτή η ενότητα δείχνει πώς να χωρίσετε ένα σχέδιο CAD σε μικρότερα πλακίδια για πιο αποτελεσματικό χειρισμό και απόδοση. Κάθε πλακίδιο θα έχει το ένα τέταρτο του αρχικού μεγέθους.

#### Βήμα 1: Ορισμός διαδρομής καταλόγου εξόδου
Ξεκινήστε ορίζοντας πού θα αποθηκευτούν οι εικόνες που έχετε αποδώσει:
```java
import java.nio.file.Path;

Path outputDirectory = Utils.getOutputDirectoryPath("SplitDrawingIntoTiles");
Path pageFilePathFormat = outputDirectory.resolve("page_{0}.png");
```
Αυτή η ρύθμιση χρησιμοποιεί μια μέθοδο βοηθητικού προγράμματος για να λάβει τη διαδρομή, εξασφαλίζοντας επαναχρησιμοποίηση και σαφήνεια.

#### Βήμα 2: Ρύθμιση παραμέτρων επιλογών προβολής
Ορίστε επιλογές για την απόδοση κάθε ενότητας ξεχωριστά:
```java
import com.groupdocs.viewer.options.PngViewOptions;
import com.groupdocs.viewer.options.ViewInfoOptions;

PngViewOptions viewOptions = new PngViewOptions(pageFilePathFormat);
ViewInfoOptions viewInfoOptions = ViewInfoOptions.forPngView(false);
```
Αυτό το απόσπασμα κώδικα διαμορφώνει την απόδοση σε μορφή PNG χωρίς να επεξεργάζεται όλες τις σελίδες ταυτόχρονα.

#### Βήμα 3: Υπολογισμός διαστάσεων πλακιδίων
Προσδιορίστε τις διαστάσεις για κάθε πλακίδιο:
```java
import com.groupdocs.viewer.results.ViewInfo;
import com.groupdocs.viewer.options.Tile;

ViewInfo viewInfo = new Viewer("path/to/your/drawing.dwg").getViewer().getViewInfo(viewInfoOptions);
int width = viewInfo.getPages().get(0).getWidth();
int height = viewInfo.getPages().get(0).getHeight();

// Κάθε πλακίδιο έχει το ένα τέταρτο του συνολικού μεγέθους.
int tileWidth = width / 2;
int tileHeight = height / 2;

Tile[] tiles = {
    new Tile(0, 0, tileWidth, tileHeight),
    new Tile(tileWidth, 0, tileWidth, tileHeight),
    new Tile(0, tileHeight, tileWidth, tileHeight),
    new Tile(tileWidth, tileHeight, tileWidth, tileHeight)
};
```

#### Βήμα 4: Απόδοση και αποθήκευση πλακιδίων
Προσθέστε κάθε υπολογισμένο πλακίδιο στις επιλογές απόδοσης και αποδώστε:
```java
viewOptions.getCadOptions().getTiles().addAll(java.util.Arrays.asList(tiles));

try (Viewer viewer = new Viewer("path/to/your/drawing.dwg")) {
    viewer.view(viewOptions);
}
```
Αυτό το τελικό βήμα αποδίδει το έγγραφο με βάση τα καθορισμένα πλακίδια, αποθηκεύοντας το καθένα ως ξεχωριστό αρχείο PNG.
