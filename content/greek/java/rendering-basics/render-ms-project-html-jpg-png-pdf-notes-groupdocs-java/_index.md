## Ρύθμιση του GroupDocs.Viewer για Java

Για να χρησιμοποιήσετε το GroupDocs.Viewer για Java, πρέπει να το προσθέσετε ως εξάρτηση στο έργο Maven σας. Αυτή η ρύθμιση περιλαμβάνει τη διαμόρφωση του αποθετηρίου GroupDocs και τον καθορισμό της έκδοσης της βιβλιοθήκης που θέλετε να χρησιμοποιήσετε.

**Διαμόρφωση Maven:**

Προσθέστε τα ακόλουθα αποσπάσματα στο δικό σας `pom.xml` αρχείο:

```xml
<repositories>
   <repository>
      <id>repository.groupdocs.com</id>
      <name>GroupDocs Repository</name>
      <url>https://releases.groupdocs.com/viewer/java/</url>
   </repository>
</repositories>
<dependencies>
   <dependency>
      <groupId>com.groupdocs</groupId>
      <artifactId>groupdocs-viewer</artifactId>
      <version>25.2</version>
   </dependency>
</dependencies>
```

Αυτή η ρύθμιση παραμέτρων επιτρέπει στο Maven να ανακτήσει το GroupDocs.Viewer από το καθορισμένο αποθετήριο.

## Οδηγός Εφαρμογής

Θα εξερευνήσουμε τον τρόπο απόδοσης εγγράφων του MS Project σε μορφές HTML, JPG, PNG και PDF, συμπεριλαμβάνοντας σημειώσεις. Κάθε ενότητα είναι αφιερωμένη σε μία μορφή, περιγράφοντας λεπτομερώς κάθε βήμα που απαιτείται για την επιτυχή υλοποίηση.

### Απόδοση εγγράφου MS Project σε HTML με σημειώσεις

**Επισκόπηση:**
Αυτή η λειτουργία σάς επιτρέπει να μετατρέψετε αρχεία MS Project σε μια εύκολα κοινοποιήσιμη μορφή HTML, μαζί με σημειώσεις έργου.

#### Βήμα προς βήμα εφαρμογή:
1. **Ρύθμιση διαδρομών:**
   Αντικαθιστώ `YOUR_DOCUMENT_DIRECTORY` και `YOUR_OUTPUT_DIRECTORY` με τις πραγματικές διαδρομές αρχείων σας.

   ```java
   Path YOUR_DOCUMENT_DIRECTORY = Path.of("YOUR_DOCUMENT_DIRECTORY");
   Path outputDirectory = Path.of(YOUR_OUTPUT_DIRECTORY);
   Path pageFilePathFormat = outputDirectory.resolve("mpp_result.html");
   ```

2. **Αρχικοποίηση Προβολέα:**
   Δημιουργήστε ένα `Viewer` αντικείμενο για το αρχείο MS Project.

   ```java
   try (Viewer viewer = new Viewer(YOUR_DOCUMENT_DIRECTORY.resolve("Sample_MPP.mpp"))) {
       // Προχωρήστε στα βήματα απόδοσης
   }
   ```

3. **Ρύθμιση παραμέτρων επιλογών HTML:**
   Χρήση `HtmlViewOptions` για να ορίσετε τον τρόπο απόδοσης του εγγράφου, συμπεριλαμβανομένων των σημειώσεων.

   ```java
   HtmlViewOptions options = HtmlViewOptions.forEmbeddedResources(pageFilePathFormat);
   options.setRenderNotes(true); // Συμπερίληψη σημειώσεων έργου στο αποτέλεσμα
   ```

4. **Απόδοση εγγράφου:**
   Εκτελέστε τη διαδικασία απόδοσης με τις διαμορφωμένες επιλογές.

   ```java
   viewer.view(options);
   ```

#### Συμβουλές αντιμετώπισης προβλημάτων:
- Βεβαιωθείτε ότι οι διαδρομές αρχείων είναι σωστές και προσβάσιμες.
- Βεβαιωθείτε ότι έχετε τα απαραίτητα δικαιώματα για την ανάγνωση/εγγραφή αρχείων.

### Απόδοση εγγράφου MS Project σε JPG, PNG και PDF με σημειώσεις

Για κάθε μία από αυτές τις μορφές, ακολουθήστε μια παρόμοια προσέγγιση όπως περιγράφεται παραπάνω. Οι βασικές διαφορές έγκεινται στη χρήση `JpgViewOptions`, `PngViewOptions`, ή `PdfViewOptions` και ορισμός κατάλληλων μορφών διαδρομής αρχείου για τα αρχεία εξόδου. Για παράδειγμα:

- **JPG:** Χρήση `pageFilePathFormat = outputDirectory.resolve("mpp_{0}_result.jpg");`
- **PNG:** Χρήση `pageFilePathFormat = outputDirectory.resolve("mpp_{0}_result.png");`
- **PDF:** Χρήση `pageFilePathFormat = outputDirectory.resolve("mpp_result.pdf");`

Για κάθε μορφή, βεβαιωθείτε ότι έχετε ορίσει την επιλογή σημειώσεων απόδοσης όπως υποδεικνύεται.
