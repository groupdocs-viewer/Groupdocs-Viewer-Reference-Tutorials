### Φόρτωση εγγράφου από URL

Αυτή η λειτουργία σάς επιτρέπει να κατεβάσετε ένα έγγραφο απευθείας από μια καθορισμένη διεύθυνση URL και να το αποδώσετε σε μορφή HTML χρησιμοποιώντας το GroupDocs.Viewer. Δείτε πώς:

#### Βήμα 1: Ανοίξτε ένα InputStream από τη διεύθυνση URL

Ξεκινήστε δημιουργώντας ένα `InputStream` που συνδέεται με τη διεύθυνση URL προορισμού σας. Αυτή η ροή θα χρησιμοποιηθεί ως είσοδος για απόδοση.

```java
String url = "https://cms.admin.containerize.com/templates/groupdocs/images/logos/groupdocs-logo.png";
try (InputStream fileStream = new URL(url).openStream()) {
    // Συνέχεια με τη ρύθμιση προβολής εγγράφων
} catch (Exception e) {
    throw new RuntimeException("Failed to open stream from the URL", e);
}
```

#### Βήμα 2: Ρύθμιση παραμέτρων επιλογών προβολής HTML

Στη συνέχεια, διαμορφώστε `HtmlViewOptions` για απόδοση. Καθορίστε πού και πώς θέλετε να αποθηκεύεται το περιεχόμενο που αποδίδεται.

```java
Path outputDirectory = Paths.get("YOUR_OUTPUT_DIRECTORY");
Path pageFilePathFormat = outputDirectory.resolve("page_{0}.html");
HtmlViewOptions viewOptions = HtmlViewOptions.forEmbeddedResources(pageFilePathFormat);
```

#### Βήμα 3: Δημιουργήστε μια παρουσία προβολής και κάντε απόδοση

Τέλος, δημιουργήστε μια παρουσία του `Viewer` με τη ροή εισόδου της διεύθυνσης URL και χρησιμοποιήστε την για να αποδώσετε το έγγραφό σας σε μορφή HTML.

```java
try (Viewer viewer = new Viewer(fileStream)) {
    viewer.view(viewOptions);
}
```
