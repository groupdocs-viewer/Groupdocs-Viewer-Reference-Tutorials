# Πώς να Λάβετε το ViewInfo και να Ανακτήσετε τις Δομές Αρχείων σε Java

Η αποτελεσματική διαχείριση αρχείων αρχειοθήκης απαιτεί σαφή κατανόηση της δομής τους. Σε αυτό το σεμινάριο θα μάθετε **πώς να λάβετε viewinfo** για οποιοδήποτε αρχείο αρχειοθήκης και θα δείτε πώς σας βοηθά να εργάζεστε με τον **java archive file type**. Θα περάσουμε από τη ρύθμιση του GroupDocs.Viewer, την εξαγωγή πληροφοριών προβολής, και την αναδρομική ανάγνωση των ιεραρχιών φακέλων ώστε να μπορείτε να ενσωματώσετε τη λύση σε πραγματικά έργα.

## Γρήγορες Απαντήσεις
- **Τι παρέχει το “viewinfo”;** Επιστρέφει τον τύπο αρχείου, τον αριθμό σελίδων και τις λίστες φακέλων της αρχειοθήκης.  
- **Ποιες αρχειοθήκες υποστηρίζονται;** ZIP, RAR, TAR, και άλλες κοινές μορφές.  
- **Χρειάζομαι άδεια;** Μια δωρεάν δοκιμή λειτουργεί για αξιολόγηση· απαιτείται εμπορική άδεια για παραγωγή.  
- **Μπορώ να επεξεργαστώ μεγάλες αρχειοθήκες;** Ναι – χρησιμοποιήστε streaming και σωστή διαχείριση μνήμης όπως φαίνεται αργότερα.  
- **Απαιτείται το Maven;** Το Maven είναι η συνιστώμενη μέθοδος διαχείρισης της εξάρτησης GroupDocs.Viewer.

## Τι είναι το “how to get viewinfo”;
`getViewInfo` είναι μια μέθοδος στο GroupDocs.Viewer API που εξάγει μεταδεδομένα σχετικά με ένα έγγραφο ή μια αρχειοθήκη χωρίς να αποδίδει το πλήρες περιεχόμενο. Για αρχειοθήκες, αποκαλύπτει το εσωτερικό δέντρο φακέλων, επιτρέποντάς σας να αποφασίσετε ποια μέρη θα αποδώσετε ή θα επεξεργαστείτε περαιτέρω.

## Γιατί να Ανακτήσετε τις Δομές Τύπου Αρχείου Java Archive;
Η κατανόηση της εσωτερικής διάταξης ενός **java archive file type** (π.χ., ZIP, RAR) σας επιτρέπει να:
- Εντοπίζετε γρήγορα τα απαιτούμενα αρχεία χωρίς να εξάγετε τα πάντα.  
- Δημιουργείτε αυτοματοποιημένες γραμμές επεξεργασίας που επεξεργάζονται μόνο τους σχετικούς υπο‑φακέλους.  
- Ενσωματώνετε την πλοήγηση στην αρχειοθήκη σε συστήματα διαχείρισης περιεχομένου ή εισαγωγής δεδομένων.

## Οδηγός Υλοποίησης

### Πώς να Λάβετε ViewInfo για Αρχειοθήκες

Αυτή η ενότητα δείχνει τα ακριβή βήματα για την κλήση του `getViewInfo` και την ανάγνωση της ιεραρχίας φακέλων.

#### Αρχικοποίηση του Viewer

```java
try (Viewer viewer = new Viewer("YOUR_DOCUMENT_DIRECTORY/SAMPLE_ZIP_WITH_FOLDERS")) {
    // Additional steps will follow here.
}
```

#### Ανάκτηση Πληροφοριών Προβολής

```java
ViewInfo viewInfo = viewer.getViewInfo(ViewInfoOptions.forHtmlView());
System.out.println("File type: " + viewInfo.getFileType());
System.out.println("Pages count: " + viewInfo.getPages().size());
```

#### Εμφάνιση Δομής Φακέλου

```java
String rootFolder = "";
readFolders(viewer, rootFolder);
```

##### Αναδρομική Ανάγνωση Φακέλου

```java
private static void readFolders(Viewer viewer, String folder) {
    ViewInfoOptions viewInfoOptions = ViewInfoOptions.forHtmlView();
    viewInfoOptions.getArchiveOptions().setFolder(folder);
    
    ArchiveViewInfo viewInfo = (ArchiveViewInfo) viewer.getViewInfo(viewInfoOptions);
    
    for (String subFolder : viewInfo.getFolders()) {
        System.out.println(" - " + subFolder);
        readFolders(viewer, subFolder);
    }
}
```

### Χαρακτηριστικό 2: Ανάκτηση Δομής Φακέλου Αρχειοθήκης

Αυτή η λειτουργία εστιάζει στην εκτύπωση της δομής φακέλου ενός αρχείου αρχειοθήκης. Είναι παρόμοια με το πρώτο χαρακτηριστικό αλλά τονίζει την αναδρομική εξερεύνηση.

#### Ρύθμιση Επιλογών View Info

```java
ViewInfoOptions viewInfoOptions = ViewInfoOptions.forHtmlView();
viewInfoOptions.getArchiveOptions().setFolder(folder);
```

#### Αναδρομική Εξερεύνηση

```java
for (String subFolder : viewInfo.getFolders()) {

```
