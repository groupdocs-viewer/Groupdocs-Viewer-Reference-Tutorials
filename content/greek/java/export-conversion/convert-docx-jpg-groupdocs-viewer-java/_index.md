### Απόδοση Εγγράφου σε JPG με Ρύθμιση Ποιότητας
#### Επισκόπηση
Αυτό το τμήμα σας καθοδηγεί στη ρύθμιση της ποιότητας εξόδου JPG κατά τη μετατροπή εγγράφων. Αυτός ο έλεγχος επιτρέπει την εξισορρόπηση μεταξύ καθαρότητας εικόνας και μεγέθους αρχείου ανάλογα με τις ανάγκες σας.

#### Βήμα 1: Διαμόρφωση Καταλόγου Εξόδου
Ορίστε πού θα αποθηκευτούν οι εικόνες εξόδου:

```java
Path outputDirectory = Paths.get("YOUR_OUTPUT_DIRECTORY", "AdjustQualityWhenRenderingToJpg");
```

#### Βήμα 2: Ρύθμιση Μορφής Διαδρομής Αρχείου
Καθορίστε τη μορφή ονομασίας των αρχείων εξόδου:

```java
Path pageFilePathFormat = outputDirectory.resolve("page_{0}.jpg");
```

#### Βήμα 3: Αρχικοποίηση JpgViewOptions
Διαμορφώστε τις επιλογές απόδοσης, συμπεριλαμβανομένης της διαδρομής αρχείου και των ρυθμίσεων ποιότητας:

```java
JpgViewOptions viewOptions = new JpgViewOptions(pageFilePathFormat);
```

#### Βήμα 4: Ρύθμιση Ποιότητας Εικόνας
Ορίστε την επιθυμητή ποιότητα εικόνας (1‑100):

```java
byte quality = 50; // Adjust based on your needs.
viewOptions.setQuality(quality);
```

#### Βήμα 5: Απόδοση Εγγράφου σε JPG
Φορτώστε το έγγραφό σας και αποδώστε το χρησιμοποιώντας τις διαμορφωμένες επιλογές:

```java
try (Viewer viewer = new Viewer("YOUR_DOCUMENT_DIRECTORY/SAMPLE_DOCX")) {
    viewer.view(viewOptions);
}
```
