# Εξαγωγή κειμένου από docx χρησιμοποιώντας το GroupDocs.Viewer για Java

## Πώς να εξάγετε κείμενο από docx – Οδηγός Βήμα‑βήμα

### 1. Εξαγωγή Μεταδεδομένων Σελίδας
Τα μεταδεδομένα σελίδας όπως ο αριθμός σελίδας είναι ουσιώδη όταν χρειάζεται να δημιουργήσετε δομές πλοήγησης ή να αναφέρετε συγκεκριμένα τμήματα.

```java
try (Viewer viewer = new Viewer("YOUR_DOCUMENT_DIRECTORY/SAMPLE_DOCX")) {
    ViewInfoOptions viewInfoOptions = ViewInfoOptions.forPngView(true);
    ViewInfo viewInfo = viewer.getViewInfo(viewInfoOptions);
```

```java
    for (Page page : viewInfo.getPages()) {
        int pageNumber = page.getNumber();
        System.out.println("Page: " + pageNumber); // Outputs the page number
    }
}
```

- `ViewInfoOptions.forPngView(true)`: Οδηγεί το API να συλλέξει πληροφορίες σελίδας κατά την προετοιμασία της απόδοσης PNG.  
- `viewInfo.getPages()`: Επιστρέφει μια συλλογή όπου κάθε αντικείμενο `Page` περιέχει τον αριθμό του και άλλα μεταδεδομένα.

**Συμβουλή:** Αποδεσμεύστε το `Viewer` μέσα σε μπλοκ try‑with‑resources για αυτόματη απελευθέρωση των εγγενών πόρων.

### 2. Εξαγωγή Γραμμών Κειμένου από Σελίδες
Τώρα που μπορείτε να εντοπίσετε κάθε σελίδα, ας εξάγουμε τις πραγματικές γραμμές κειμένου.

```java
try (Viewer viewer = new Viewer("YOUR_DOCUMENT_DIRECTORY/SAMPLE_DOCX")) {
    ViewInfoOptions viewInfoOptions = ViewInfoOptions.forPngView(true);
    ViewInfo viewInfo = viewer.getViewInfo(viewInfoOptions);
```

```java
    for (Page page : viewInfo.getPages()) {
        System.out.println("Page: " + page.getNumber());
        System.out.println("Text lines:");
        
        for (Line line : page.getLines()) {
            String lineText = line.getValue();
            System.out.print(lineText + "\t");
        }
    }
}
```

- `page.getLines()`: Επιστρέφει μια λίστα από αντικείμενα `Line`, το καθένα αντιπροσωπεύει μια μοναδική γραμμή κειμένου όπως εμφανίζεται στη σελίδα.  
- Ο εσωτερικός βρόχος εκτυπώνει κάθε γραμμή, διαχωρισμένη με καρτέλες για ευανάγνωστη παρουσίαση.

### Συχνά Προβλήματα & Λύσεις
| Συμπτωμα | Πιθανή Αιτία | Διόρθωση |
|---------|--------------|----------|
| `null` αριθμοί σελίδας | Το έγγραφο δεν φορτώθηκε σωστά | Επαληθεύστε τη διαδρομή του αρχείου και βεβαιωθείτε ότι το αρχείο υπάρχει. |
| Δεν επιστράφηκαν γραμμές κειμένου | Μη υποστηριζόμενη μορφή αρχείου | Ελέγξτε ότι η έκδοση DOCX υποστηρίζεται· αναβαθμίστε το GroupDocs αν χρειάζεται. |
| `OutOfMemoryError` σε μεγάλα αρχεία | Το Viewer κρατάει πάρα πολλές σελίδες στη μνήμη | Επεξεργαστείτε τις σελίδες σε μικρότερες παρτίδες ή επαναχρησιμοποιήστε την ίδια παρουσία `Viewer`. |
