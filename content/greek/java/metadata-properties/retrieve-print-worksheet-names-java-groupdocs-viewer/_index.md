## Πώς να ανακτήσετε τα ονόματα φύλλων Excel (λίστα φύλλων λογιστικού φύλλου)

Παρακάτω βρίσκεται ένας οδηγός βήμα‑βήμα που σας καθοδηγεί στην εξαγωγή ονομάτων φύλλων εργασίας. Ο κώδικας παραμένει αμετάβλητος από το αρχικό παράδειγμα, εξασφαλίζοντας ότι λειτουργεί όπως είναι.

### Βήμα 1: Αρχικοποίηση του Viewer

Ξεκινήστε αρχικοποιώντας το `Viewer` με τη διαδρομή του εγγράφου σας:

```java
try (Viewer viewer = new Viewer("YOUR_DOCUMENT_DIRECTORY/three_sheets.xlsx")) {
    // Initialization code here...
}
```

Αυτό το τμήμα ρυθμίζει το Viewer για εργασία με ένα συγκεκριμένο αρχείο, εξασφαλίζοντας σωστή διαχείριση πόρων χρησιμοποιώντας το try‑with‑resources.

### Βήμα 2: Διαμόρφωση του ViewInfoOptions

Ορίστε το `ViewInfoOptions` για ανάκτηση πληροφοριών προβολής HTML:

```java
ViewInfoOptions viewInfoOptions = ViewInfoOptions.forHtmlView();
viewInfoOptions.setSpreadsheetOptions(SpreadsheetOptions.forOnePagePerSheet());
```

Αυτή η διαμόρφωση εξασφαλίζει ότι κάθε φύλλο εργασίας αποδίδεται ξεχωριστά, διευκολύνοντας την επανάληψη πάνω σε μεμονωμένα φύλλα.

### Βήμα 3: Ανάκτηση και Εμφάνιση Ονομάτων Φύλλων Εργασίας

Αποκτήστε το αντικείμενο `ViewInfo` για να λάβετε λεπτομέρειες σχετικά με τις σελίδες του εγγράφου (φύλλα εργασίας):

```java
ViewInfo viewInfo = viewer.getViewInfo(viewInfoOptions);

for (Page page : viewInfo.getPages()) {
    System.out.println(" - Worksheet " + page.getNumber() + ": '" + page.getName() + "'");
}
```

Αυτός ο βρόχος επαναλαμβάνει κάθε φύλλο εργασίας, εκτυπώνοντας το δείκτη και το όνομά του. Είναι ο πυρήνας της λειτουργίας **java extract xlsx sheets**.
