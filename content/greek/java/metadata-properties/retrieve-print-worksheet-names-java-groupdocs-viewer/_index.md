# Εξαγωγή και εμφάνιση ονομάτων φύλλων εργασίας σε Java χρησιμοποιώντας το GroupDocs.Viewer API

## Εισαγωγή

Η διαχείριση πολλαπλών φύλλων εργασίας μέσα σε αρχεία υπολογιστικών φύλλων μπορεί να είναι δύσκολη, ειδικά όταν χειρίζεστε μεγάλα σύνολα δεδομένων ή αυτοματοποιείτε τη δημιουργία αναφορών. Το API GroupDocs.Viewer για Java απλοποιεί αυτήν την εργασία επιτρέποντάς σας να ανακτάτε ονόματα φύλλων εργασίας μέσω προγραμματισμού, εξοικονομώντας χρόνο και βελτιώνοντας τις ροές εργασίας αυτοματοποίησης. Αυτό το σεμινάριο θα σας καθοδηγήσει στη διαδικασία χρήσης του GroupDocs.Viewer για Java για την εξαγωγή και εμφάνιση ονομάτων φύλλων εργασίας από ένα έγγραφο υπολογιστικού φύλλου.

**Βασικά σημεία:**
- Ρύθμιση του περιβάλλοντός σας με το GroupDocs.Viewer
- Αρχικοποίηση του Viewer και ρύθμιση παραμέτρων επιλογών
- Τεχνικές για την αποτελεσματική ανάκτηση και επανάληψη φύλλων εργασίας
- Βέλτιστες πρακτικές για τη βελτιστοποίηση της απόδοσης

## Οδηγός Εφαρμογής

### Χαρακτηριστικό: Εξαγωγή ονομάτων φύλλων εργασίας

Αυτή η λειτουργία δείχνει πώς να εξαγάγετε ονόματα φύλλων εργασίας από ένα υπολογιστικό φύλλο χρησιμοποιώντας το GroupDocs.Viewer.

#### Βήμα 1: Αρχικοποίηση του προγράμματος προβολής

Ξεκινήστε με την αρχικοποίηση `Viewer` με τη διαδρομή του εγγράφου σας:

```java
try (Viewer viewer = new Viewer("YOUR_DOCUMENT_DIRECTORY/three_sheets.xlsx")) {
    // Κώδικας αρχικοποίησης εδώ...
}
```

Αυτό το μπλοκ ρυθμίζει το πρόγραμμα προβολής ώστε να λειτουργεί με ένα συγκεκριμένο αρχείο, διασφαλίζοντας την κατάλληλη διαχείριση πόρων χρησιμοποιώντας την εντολή try-with-resources.

#### Βήμα 2: Ρύθμιση παραμέτρων ViewInfoOptions

Σειρά `ViewInfoOptions` για ανάκτηση πληροφοριών προβολής HTML:

```java
ViewInfoOptions viewInfoOptions = ViewInfoOptions.forHtmlView();
viewInfoOptions.setSpreadsheetOptions(SpreadsheetOptions.forOnePagePerSheet());
```

Αυτή η διαμόρφωση διασφαλίζει ότι κάθε φύλλο εργασίας αποδίδεται ξεχωριστά, διευκολύνοντας την επανάληψη σε μεμονωμένα φύλλα.

#### Βήμα 3: Ανάκτηση και εμφάνιση ονομάτων φύλλων εργασίας

Αποκτήστε το `ViewInfo` αντικείμενο για να λάβετε λεπτομέρειες σχετικά με σελίδες εγγράφων (φύλλα εργασίας):

```java
ViewInfo viewInfo = viewer.getViewInfo(viewInfoOptions);

for (Page page : viewInfo.getPages()) {
    System.out.println(" - Worksheet " + page.getNumber() + ": '" + page.getName() + "'");
}
```

Αυτός ο βρόχος επαναλαμβάνεται σε κάθε φύλλο εργασίας, εκτυπώνοντας τον δείκτη και το όνομά του.
