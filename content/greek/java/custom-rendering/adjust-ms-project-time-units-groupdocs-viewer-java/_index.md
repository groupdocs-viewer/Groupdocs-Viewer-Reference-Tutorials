### Επισκόπηση Χαρακτηριστικού: Ρύθμιση Μονάδων Χρόνου σε Έγγραφα MS Project
Αυτή η δυνατότητα σας επιτρέπει να αλλάξετε τη ρύθμιση μονάδας χρόνου διαχείρισης έργου από την προεπιλογή (συνήθως λεπτά) σε ημέρες, καθιστώντας το παραγόμενο HTML πιο φιλικό προς τον χρήστη και ευθυγραμμισμένο με τα τυπικά πρότυπα αναφοράς.

#### Βήμα 1: Ορισμός Καταλόγου Εξόδου και Μορφής Διαδρομής Αρχείου Σελίδας
Πρώτα, καθορίστε πού θα αποθηκευτούν τα παραγόμενα αρχεία HTML:

```java
import java.nio.file.Path;
// Specify the output directory for HTML files
Path outputDirectory = Utils.getOutputDirectoryPath("YOUR_OUTPUT_DIRECTORY");
```

Χρησιμοποιήστε αυτόν τον κατάλογο για να επιλύετε δυναμικά τις διαδρομές αρχείων για κάθε σελίδα του εγγράφου MS Project:

```java
// Define a format for each rendered page's file path
Path pageFilePathFormat = outputDirectory.resolve("page_{0}.html");
```

#### Βήμα 2: Αρχικοποίηση Επιλογών Προβολής
Δημιουργήστε επιλογές προβολής με ενσωματωμένους πόρους, που σας επιτρέπουν να καθορίσετε πώς θα προβληθεί και θα αποδοθεί το έργο:

```java
import com.groupdocs.viewer.options.HtmlViewOptions;
// Set up HTML view options for rendering
HtmlViewOptions viewOptions = HtmlViewOptions.forEmbeddedResources(pageFilePathFormat);
```

#### Βήμα 3: Ρύθμιση Μονάδας Χρόνου
Καθορίστε ότι η μονάδα χρόνου για τη διαχείριση έργου ορίζεται σε ημέρες, κάτι που συχνά είναι πιο κατάλληλο για παρουσιάσεις και αναφορές:

```java
import com.groupdocs.viewer.options.TimeUnit;
// Change the project management time unit to DAYS
viewOptions.getProjectManagementOptions().setTimeUnit(TimeUnit.DAYS);
```

#### Βήμα 4: Απόδοση Εγγράφου MS Project
Τέλος, χρησιμοποιήστε την κλάση Viewer για να αποδώσετε το έγγραφό σας με τις καθορισμένες επιλογές προβολής:

```java
import com.groupdocs.viewer.Viewer;
try (Viewer viewer = new Viewer("YOUR_DOCUMENT_DIRECTORY/SAMPLE_MPP")) {
    // Render the project document as HTML using set view options
    viewer.view(viewOptions);
}
```
