# Πώς να ανακτήσετε και να αποθηκεύσετε συνημμένα έγγραφα χρησιμοποιώντας το GroupDocs.Viewer για Java: Ένας πλήρης οδηγός

Παρακολουθώντας, θα αποκτήσετε πρακτικές δεξιότητες στην αποτελεσματική διαχείριση δεδομένων εγγράφων σε περιβάλλον Java.

## Ρύθμιση του GroupDocs.Viewer για Java
Για να ξεκινήσετε να χρησιμοποιείτε το GroupDocs.Viewer API στο έργο σας, πρέπει να το εγκαταστήσετε μέσω του Maven. Προσθέστε την ακόλουθη διαμόρφωση στο έργο σας `pom.xml` αρχείο:

```xml
<repositories>
   <repository>
      <id>repository.groupdocs.com</id>
      <name>GroupDocs Repository</name>
      <url>https://releases.groupdocs.com/viewer/java/</url>
   </repository>
</repositories>
<dependencies>
   <dependency>
      <groupId>com.groupdocs</groupId>
      <artifactId>groupdocs-viewer</artifactId>
      <version>25.2</version>
   </dependency>
</dependencies>
```

**Βήματα Απόκτησης Άδειας Χρήσης:**
- **Δωρεάν δοκιμή**Ξεκινήστε με μια δωρεάν δοκιμή για να εξερευνήσετε τις λειτουργίες.
- **Προσωρινή Άδεια**Αποκτήστε προσωρινή άδεια για εκτεταμένη περίοδο αξιολόγησης.
- **Αγορά**Για χρήση σε παραγωγική χρήση, θα χρειαστεί να αγοράσετε μια άδεια χρήσης.

### Βασική Αρχικοποίηση και Ρύθμιση
Μόλις το GroupDocs.Viewer προστεθεί ως εξάρτηση στο έργο σας, αρχικοποιήστε το στην εφαρμογή Java. Δείτε πώς:

```java
import com.groupdocs.viewer.Viewer;
import java.nio.file.Path;

public class InitializeViewer {
    public static void main(String[] args) {

            e.printStackTrace();
        }
    }
}
```

Αυτή η βασική ρύθμιση αρχικοποιεί το GroupDocs.Viewer και το προετοιμάζει για την ανάκτηση συνημμένων.

## Οδηγός Εφαρμογής

### Ανάκτηση συνημμένων από ένα έγγραφο
Ας αναλύσουμε πώς μπορείτε να ανακτήσετε συνημμένα χρησιμοποιώντας το GroupDocs.Viewer. Αυτή η λειτουργία σάς επιτρέπει να εξαγάγετε μεταδεδομένα σχετικά με κάθε συνημμένο που υπάρχει στο έγγραφό σας.

#### Επισκόπηση
Η ανάκτηση συνημμένων περιλαμβάνει την πρόσβαση στο `getAttachments` μέθοδος, η οποία επιστρέφει μια λίστα αντικειμένων Συνημμένου που περιέχουν λεπτομέρειες όπως το όνομα και το μέγεθος του αρχείου.

#### Βήματα Υλοποίησης
1. **Δημιουργία στιγμιότυπου προβολής**
   Αρχικοποίηση του `Viewer` κλάση με τη διαδρομή προς το έγγραφό σας:

   ```java
   import com.groupdocs.viewer.Viewer;
   import com.groupdocs.viewer.examples.TestFiles;
   import com.groupdocs.viewer.results.Attachment;

   public class FeatureRetrieveAttachments {
       public static void main(String[] args) {
           try (Viewer viewer = new Viewer(TestFiles.SAMPLE_MSG_WITH_ATTACHMENTS)) {
               // Συνέχεια στην ανάκτηση συνημμένων
           } catch (Exception e) {
               throw new RuntimeException(e);
           }
       }
   }
   ```

2. **Ανάκτηση συνημμένων**
   Χρησιμοποιήστε το `getAttachments` μέθοδος:

   ```java
   List<Attachment> attachments = viewer.getAttachments();
   for (Attachment attachment : attachments) {
       System.out.println(attachment.getFileName());
   }
   ```

3. **Κατανόηση Παραμέτρων και Μεθόδων**
   - `Viewer`: Λαμβάνει μια διαδρομή αρχείου ή μια ροή προς το έγγραφο.
   - `getAttachments()`: Ανακτά μια λίστα συνημμένων αρχείων, παρέχοντας λεπτομέρειες όπως ονόματα.

### Αποθήκευση συνημμένων εγγράφων σε έναν κατάλογο
Τώρα που μάθατε πώς να ανακτάτε συνημμένα, ας τα αποθηκεύσουμε στον επιθυμητό κατάλογο χρησιμοποιώντας το GroupDocs.Viewer API.

#### Επισκόπηση
Αυτή η λειτουργία περιλαμβάνει την αποθήκευση κάθε ανακτημένου συνημμένου αρχείου σε έναν καθορισμένο κατάλογο εξόδου. 

#### Βήματα Υλοποίησης
1. **Ορίστε τον κατάλογο εξόδου**
   Ρύθμιση ενός `outputDirectory` διαδρομή όπου θα αποθηκευτούν τα αρχεία:

   ```java
   import java.nio.file.Path;
   import java.nio.file.Paths;

   Path outputDirectory = Paths.get("YOUR_OUTPUT_DIRECTORY");
   ```

2. **Αποθήκευση συνημμένων**
   Χρησιμοποιήστε έναν βρόχο για να αποθηκεύσετε κάθε συνημμένο χρησιμοποιώντας το `saveAttachment` μέθοδος:

   ```java
   try (Viewer viewer = new Viewer(TestFiles.SAMPLE_MSG_WITH_ATTACHMENTS)) {
       List<Attachment> attachments = viewer.getAttachments();
       
       for (Attachment attachment : attachments) {
           final Path file = outputDirectory.resolve(attachment.getFileName());
           try (FileOutputStream outputStream = new FileOutputStream(file.toFile())) {
               viewer.saveAttachment(attachment, outputStream);
           }
       }
   } catch (Exception e) {
       throw new RuntimeException(e);
   }
   ```

3. **Εξηγήστε Παραμέτρους και Μεθόδους**
   - `saveAttachment`: Δέχεται ένα αντικείμενο Συνημμένου και μια ροή εξόδου αρχείου για να αποθηκεύσει το συνημμένο.
   - `FileOutputStream`: Διαχειρίζεται την εγγραφή δεδομένων σε αρχεία.
