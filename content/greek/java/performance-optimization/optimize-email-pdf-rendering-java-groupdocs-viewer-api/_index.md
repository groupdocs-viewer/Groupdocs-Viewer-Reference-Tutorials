## Οδηγός Εφαρμογής

### Προσαρμογή μεγέθους σελίδας για απόδοση email

Αυτή η λειτουργία εστιάζει στην προσαρμογή του μεγέθους σελίδας κατά τη μετατροπή μηνυμάτων email σε PDF. Από προεπιλογή, τα email ενδέχεται να εμφανίζονται σε διάφορες μορφές. Ωστόσο, ο ορισμός ενός συγκεκριμένου μεγέθους σελίδας διασφαλίζει τη συνέπεια σε όλα τα έγγραφα.

#### Βήμα 1: Ορισμός καταλόγου εξόδου και διαδρομής αρχείου
Αρχικά, καθορίστε πού θα αποθηκευτεί το αποτυπωμένο έγγραφό σας:

```java
import java.nio.file.Path;
import java.nio.file.Paths;

Path YOUR_OUTPUT_DIRECTORY = Paths.get("YOUR_OUTPUT_DIRECTORY");
Path filePath = YOUR_OUTPUT_DIRECTORY.resolve("output.pdf");
```

#### Βήμα 2: Ρύθμιση παραμέτρων του PdfViewOptions
Ορίστε επιλογές για να προσαρμόσετε τη διαδικασία απόδοσης, ορίζοντας συγκεκριμένα το μέγεθος σελίδας:

```java
import com.groupdocs.viewer.options.PdfViewOptions;
import com.groupdocs.viewer.options.PageSize;

PdfViewOptions viewOptions = new PdfViewOptions(filePath);
viewOptions.getEmailOptions().setPageSize(PageSize.A4); // Προσαρμόστε το μέγεθος σελίδας για μηνύματα email
```

#### Βήμα 3: Απόδοση του μηνύματος ηλεκτρονικού ταχυδρομείου σε μορφή PDF

Τέλος, αποδώστε το μήνυμα ηλεκτρονικού ταχυδρομείου σας χρησιμοποιώντας τις διαμορφωμένες επιλογές:

```java
try (Viewer viewer = new Viewer("YOUR_DOCUMENT_DIRECTORY/SAMPLE_MSG")) {
    viewer.view(viewOptions);
}
// Το έγγραφο που αποδίδεται αποθηκεύεται στον κατάλογο YOUR_OUTPUT_DIRECTORY
```

### Επεξήγηση παραμέτρων και μεθόδων κώδικα
- **Επιλογές Προβολής Pdf:** Διαχειρίζεται τον τρόπο μετατροπής των email σε PDF, επιτρέποντας τη διαμόρφωση του μεγέθους σελίδας.
- **ορισμόςΜέγεθοςΣελίδας(ΜέγεθοςΣελίδας.A4):** Προσαρμόζει την απόδοση σε μέγεθος χαρτιού A4 για συνέπεια.
