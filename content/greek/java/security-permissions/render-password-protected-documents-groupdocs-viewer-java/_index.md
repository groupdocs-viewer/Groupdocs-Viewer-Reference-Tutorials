## Οδηγός Εφαρμογής

Τώρα, ας εφαρμόσουμε τη λειτουργία για την απόδοση εγγράφων που προστατεύονται με κωδικό πρόσβασης.

### Απόδοση εγγράφων που προστατεύονται με κωδικό πρόσβασης

#### Επισκόπηση

Αυτή η ενότητα δείχνει πώς να φορτώσετε ένα έγγραφο που προστατεύεται από κωδικό πρόσβασης χρησιμοποιώντας το GroupDocs.Viewer. Θα ρυθμίσουμε την εφαρμογή ώστε να το μετατρέψει σε μορφή HTML για εύκολη προβολή.

#### Οδηγίες βήμα προς βήμα

##### Φόρτωση επιλογών και ορισμός κωδικού πρόσβασης

Για να αποκτήσετε πρόσβαση σε ένα έγγραφο που προστατεύεται με κωδικό πρόσβασης, δημιουργήστε `LoadOptions` και καθορίστε τον κωδικό πρόσβασης:

```java
import com.groupdocs.viewer.options.LoadOptions;

// Ορίστε τη διαδρομή του εγγράφου και τον κατάλογο εξόδου
String inputFilePath = "YOUR_DOCUMENT_DIRECTORY/SAMPLE_DOCX_WITH_PASSWORD";
Path outputDirectory = java.nio.file.Paths.get("YOUR_OUTPUT_DIRECTORY");
Path pageFilePathFormat = outputDirectory.resolve("page_{0}.html");

LoadOptions loadOptions = new LoadOptions();
loadOptions.setPassword("12345"); // Αντικατάσταση με τον πραγματικό κωδικό πρόσβασης εγγράφου
```

##### Ρύθμιση παραμέτρων HtmlViewOptions

Στήνω `HtmlViewOptions` για να προσδιορίσετε πού θα αποθηκευτούν οι σελίδες HTML που εμφανίζονται:

```java
import com.groupdocs.viewer.options.HtmlViewOptions;

HtmlViewOptions viewOptions = HtmlViewOptions.forEmbeddedResources(pageFilePathFormat);
```

##### Δημιουργία στιγμιότυπου προβολής και απόδοση εγγράφου

Χρησιμοποιήστε μια εντολή try-with-resources για να δημιουργήσετε ένα `Viewer` για παράδειγμα, διασφαλίζοντας την ορθή διαχείριση των πόρων:

```java
try (Viewer viewer = new Viewer(inputFilePath, loadOptions)) {
    // Απόδοση του εγγράφου χρησιμοποιώντας καθορισμένες επιλογές προβολής
    viewer.view(viewOptions);
}
```

### Εξήγηση

- **Επιλογές Φόρτωσης**Αυτή η κλάση σάς επιτρέπει να καθορίσετε παραμέτρους για τη φόρτωση εγγράφων. Εδώ, χρησιμοποιείται για την παροχή του κωδικού πρόσβασης.
- **Επιλογές προβολής Html**: Ρυθμίζει τον τρόπο και την τοποθεσία αποθήκευσης των αρχείων HTML εξόδου.
- **Θεατής**: Η κύρια κλάση που χειρίζεται λειτουργίες απόδοσης.

#### Συμβουλές αντιμετώπισης προβλημάτων

- Βεβαιωθείτε ότι η διαδρομή του εγγράφου και ο κωδικός πρόσβασης είναι σωστά. Διαφορετικά, θα δημιουργηθεί μια εξαίρεση.
- Ελέγξτε τα δικαιώματα αρχείων τόσο για τους καταλόγους εισόδου όσο και για τους καταλόγους εξόδου για να αποφύγετε σφάλματα εισόδου/εξόδου.
