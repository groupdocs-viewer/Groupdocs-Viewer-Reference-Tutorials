Για να ξεκινήσετε να χρησιμοποιείτε το GroupDocs.Viewer, ενσωματώστε το στο έργο Java σας μέσω του Maven:

**Ρύθμιση Maven**
```xml
<repositories>
   <repository>
      <id>repository.groupdocs.com</id>
      <name>GroupDocs Repository</name>
      <url>https://releases.groupdocs.com/viewer/java/</url>
   </repository>
</repositories>
<dependencies>
   <dependency>
      <groupId>com.groupdocs</groupId>
      <artifactId>groupdocs-viewer</artifactId>
      <version>25.2</version>
   </dependency>
</dependencies>
```

## Οδηγός Εφαρμογής
### Λειτουργία 1: Έλεγχος κρυπτογράφησης αρχείων
#### Επισκόπηση
Αυτή η λειτουργία σάς επιτρέπει να προσδιορίσετε εάν ένα έγγραφο είναι κρυπτογραφημένο, διασφαλίζοντας ότι τα ευαίσθητα δεδομένα παραμένουν ασφαλή.

#### Βήμα προς βήμα εφαρμογή
##### Εισαγωγή απαιτούμενων κλάσεων
Ξεκινήστε εισάγοντας τις απαραίτητες κλάσεις:
```java
import com.groupdocs.viewer.Viewer;
import com.groupdocs.viewer.results.FileInfo;
```

##### Αρχικοποίηση προγράμματος προβολής και έλεγχος κρυπτογράφησης
Αντικαθιστώ `'YOUR_DOCUMENT_DIRECTORY'` με τη διαδρομή του εγγράφου σας:
```java
String filePath = "YOUR_DOCUMENT_DIRECTORY/ENCRYPTED";
try (Viewer viewer = new Viewer(filePath)) {
    FileInfo fileInfo = viewer.getFileInfo();
    boolean isEncrypted = fileInfo.isEncrypted();

    if(isEncrypted) {
        System.out.println("The file is encrypted.");
    } else {
        System.out.println("The file is not encrypted.");
    }
}
```
- **Παράμετροι & Σκοπός μεθόδου:** `viewer.getFileInfo()` ανακτά μεταδεδομένα, συμπεριλαμβανομένης της κατάστασης κρυπτογράφησης.
- **Συμβουλή αντιμετώπισης προβλημάτων:** Βεβαιωθείτε ότι η διαδρομή του εγγράφου είναι σωστή για να αποφύγετε `FileNotFoundException`.

### Χαρακτηριστικό 2: Ανίχνευση τύπου αρχείου
#### Επισκόπηση
Προσδιορίστε γρήγορα τον τύπο ενός αρχείου για να προσαρμόσετε τα βήματα επεξεργασίας ανάλογα.

#### Βήμα προς βήμα εφαρμογή
##### Λήψη και εξαγωγή τύπου αρχείου
Χρησιμοποιήστε την ίδια αρχική ρύθμιση όπως παραπάνω για την εισαγωγή κλάσεων:
```java
String filePath = "YOUR_DOCUMENT_DIRECTORY/ANY_FILE";
try (Viewer viewer = new Viewer(filePath)) {
    FileInfo fileInfo = viewer.getFileInfo();
    String fileType = fileInfo.getFileType();

    System.out.println("The file type is: " + fileType);
}
```
- **Παράμετροι & Σκοπός μεθόδου:** `fileInfo.getFileType()` επιστρέφει τη μορφή του εγγράφου.
- **Συμβουλή αντιμετώπισης προβλημάτων:** Τα μη υποστηριζόμενα αρχεία ενδέχεται να επιστρέψουν null ή ένα μη αναμενόμενο αποτέλεσμα.
