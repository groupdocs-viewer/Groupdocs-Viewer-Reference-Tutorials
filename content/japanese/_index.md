# GroupDocs.Viewerでドキュメントにウォーターマークを追加する

GroupDocs.Viewerチュートリアルハブへようこそ。ここでは、.NET と Java 用の強力なドキュメントビューア API を習得するための、包括的なチュートリアルとガイドのコレクションをご覧いただけます。Web アプリケーション、デスクトップアプリ、モバイルソリューションのいずれを構築していても、GroupDocs.Viewer を使用すると、PDF、Microsoft Office（Word、Excel、PowerPoint）、CAD、画像など、さまざまなドキュメント形式をシームレスにレンダリングおよび表示できます。**ドキュメントに簡単にウォーターマークを追加でき**、コンテンツを保護し、ブランドを強化します。

## GroupDocs.Viewer を使用してドキュメントにウォーターマークを追加する方法

ウォーターマークの追加は、機密情報を保護したり、エクスポートされた各ページにブランドを付ける必要がある企業にとって一般的な要件です。GroupDocs.Viewer を使用すると、元のファイルを変更せずに、レンダリングプロセス中にテキストまたは画像のウォーターマークを適用できます。このアプローチは PDF、Word ファイル、スプレッドシート、さらには CAD 図面にも対応しており、すべてのサポート対象フォーマットで一貫した保護層を提供します。

以下では、チュートリアルライブラリの主要セクションをご案内します。各セクションにはステップバイステップのガイド、コードスニペット、ベストプラクティスのヒントが含まれており、アプリケーションでドキュメントのレンダリング、変換、ウォーターマーク付与をすぐに開始できます。
