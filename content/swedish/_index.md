# Lägg till vattenstämpel i dokument med GroupDocs.Viewer

Välkommen till GroupDocs.Viewer‑handledningshubben. Oavsett om du bygger en webb‑, skrivbords‑ eller mobilapplikation kommer du snart att upptäcka hur enkelt det är att **lägga till vattenstämpel i dokument**‑filer och rendera dem felfritt på alla plattformar. Med stöd för PDF, Microsoft Office, CAD‑ritningar, e‑postmeddelanden och mer låter GroupDocs.Viewer dig konvertera Word till PDF, rendera PDF i Java och visa CAD‑ritningar — allt medan dina dokument hålls säkra och varumärkeskonsekventa.

## Varför lägga till vattenstämpel i dokument med GroupDocs.Viewer?

Att lägga till en vattenstämpel skyddar immateriella rättigheter, stärker varumärket och uppfyller efterlevnadskrav. GroupDocs.Viewer erbjuder ett enkelt API som låter dig överlagra text‑ eller bildvattenstämplar under renderingsprocessen, så att originalfilen förblir orörd medan varje renderad sida bär ditt anpassade märke.
