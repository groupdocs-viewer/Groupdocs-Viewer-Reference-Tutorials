# GroupDocs.Viewer Tutoriály

Vítejte v hubu tutoriálů GroupDocs.Viewer. Zde najdete komplexní sbírku tutoriálů a průvodců, které vám pomohou **přidat vodoznak do dokumentu** a ovládnout naše výkonné API pro prohlížení dokumentů pro .NET a Java. Ať už vytváříte webovou aplikaci, desktopovou aplikaci nebo mobilní řešení, GroupDocs.Viewer vám umožňuje bezproblémově renderovat a zobrazovat širokou škálu formátů dokumentů, včetně PDF, Microsoft Office (Word, Excel, PowerPoint), CAD, obrázků a dalších.

## Quick Answers
- **Co znamená “add watermark document”?** Jedná se o vložení poloprůhledného textového nebo obrázkového překryvu na každou stránku renderovaného dokumentu.  
- **Které formáty podporují vodoznaky?** Všechny formáty renderované GroupDocs.Viewer, včetně PDF, DOCX, XLSX, CAD a e‑mailových zpráv.  
- **Potřebuji licenci?** Pro produkční použití je vyžadována platná licence GroupDocs.Viewer; je k dispozici bezplatná zkušební verze.  
- **Mohu kombinovat vodoznaky s dalšími funkcemi?** Ano — vodoznaky lze použít společně s cachováním, vlastním renderováním a bezpečnostními nastaveními.  
- **Je kompatibilní s .NET Core?** Naprosto — GroupDocs.Viewer funguje s .NET Framework, .NET Core a .NET 5/6+.

## Jak přidat vodoznak do dokumentu pomocí GroupDocs.Viewer

Přidání vodoznaku je často vyžadováno pro značkování, důvěrnost nebo splnění regulačních požadavků. S GroupDocs.Viewer můžete aplikovat vodoznak během renderování dokumentu do HTML, PDF nebo obrazových formátů. Proces je jednoduchý:

1. **Vytvořte objekt `WatermarkOptions`** a nastavte text, barvu, neprůhlednost a rotaci.  
2. **Předajte možnosti** příslušné metodě renderování (např. `RenderToPdf`, `RenderToHtml` nebo `RenderToImage`).  
3. **Renderujte dokument** — výstup bude obsahovat vodoznak na každé stránce.

Tento přístup funguje napříč všemi podporovanými typy souborů, včetně scénářů **render pdf .net**, CAD výkresů a e‑mailových zpráv.

## Často kladené otázky

**Q: Mohu přidat vodoznak do dokumentů renderovaných jako obrázky?**  
A: Ano — použijte `WatermarkOptions` spolu s `RenderToImage` k vložení vodoznaků na každou vygenerovanou stránku obrázku.

**Q: Ovlivňuje přidání vodoznaku výkon renderování?**  
A: Dopad je minimální; GroupDocs.Viewer optimalizuje proces překrytí, zejména při kombinaci s cachováním.

**Q: Jsou vodoznaky podporovány při renderování e‑mailových zpráv?**  
A: Rozhodně — vodoznaky lze aplikovat na HTML nebo PDF výstup renderovaných e‑mailových zpráv.

**Q: Jak mohu přizpůsobit vzhled vodoznaku?**  
A: Můžete nastavit rodinu písma, velikost, barvu, neprůhlednost, úhel rotace a dokonce použít obrázek jako vodoznak.

**Q: Je možné použít různé vodoznaky na různé stránky?**  
A: Standardní API aplikuje jednotný vodoznak, ale můžete implementovat vlastní logiku renderování pro různé vodoznaky na jednotlivých stránkách.
