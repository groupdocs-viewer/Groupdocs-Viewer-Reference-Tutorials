# Dodaj znak wodny do dokumentu za pomocą GroupDocs.Viewer

Witamy w centrum samouczków GroupDocs.Viewer. Niezależnie od tego, czy tworzysz aplikację internetową, desktopową czy mobilną, wkrótce odkryjesz, jak łatwo **dodawać znak wodny do dokumentu** i renderować go bezbłędnie na różnych platformach. Dzięki obsłudze PDF, Microsoft Office, rysunków CAD, wiadomości e‑mail i wielu innych, GroupDocs.Viewer umożliwia konwersję Word do PDF, renderowanie PDF w Java oraz wyświetlanie rysunków CAD — wszystko przy zachowaniu bezpieczeństwa i spójności marki.

## Dlaczego dodawać znak wodny do dokumentu przy użyciu GroupDocs.Viewer?

Dodanie znaku wodnego chroni własność intelektualną, wzmacnia branding i spełnia wymogi zgodności. GroupDocs.Viewer udostępnia prostą API, która pozwala nakładać tekstowe lub graficzne znaki wodne w trakcie procesu renderowania, dzięki czemu oryginalny plik pozostaje niezmieniony, a każda wyrenderowana strona zawiera Twój własny znak.

{{% alert color="primary" %}}
Wzmacniaj swoje aplikacje .NET dzięki wysokiej jakości możliwościom podglądu dokumentów. Nasze samouczki GroupDocs.Viewer dla .NET dostarczają wszystkiego, co potrzebne, aby zintegrować potężny podgląd dokumentów. Dowiedz się, jak renderować ponad 170 formatów dokumentów do HTML, JPEG, PNG i PDF. Poznaj zaawansowane tematy, takie jak renderowanie konkretnych układów w rysunkach CAD, obsługa załączników dokumentów oraz optymalizacja wydajności. Rozpocznij budowanie solidnych i profesjonalnych doświadczeń podglądu dokumentów w C#, ASP.NET i innych frameworkach .NET.
{{% /alert %}}
