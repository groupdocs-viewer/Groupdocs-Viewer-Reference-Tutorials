Each feature is detailed below.\n\n### Rendering CMX Document to HTML\n\nConverting a CMX document to an HTML file allows for easy web embedding and sharing.\n\n#### Step-by-Step Guide:\n\n**1. Set Up Output Directory**\n\nDefine where your rendered files will be saved:\n```csharp\nstring outputDirectory = \"YOUR_OUTPUT_DIRECTORY\";\nstring pageFilePathFormat = Path.Combine(outputDirectory, \"cmx_result_{0}.html\");\n```\n\n**2. Initialize Viewer and Configure HTML Options**\n\nCreate a `Viewer` object and configure it to render as embedded resources in HTML:\n\n```csharp\nusing (Viewer viewer = new Viewer(\"@\" + YOUR_DOCUMENT_DIRECTORY + \"/sample.cmx\"))\n{\n    HtmlViewOptions options = HtmlViewOptions.ForEmbeddedResources(pageFilePathFormat);\n    viewer.View(options);\n}\n```\n\n- **HtmlViewOptions**: Configures how the document will be rendered in HTML. The `ForEmbeddedResources` method ensures all resources (images, styles) are embedded within the HTML file.\n\n### Rendering CMX Document to JPG\n\nRendering to JPG is useful for creating image previews of documents.\n\n#### Step-by-Step Guide:\n\n**1. Initialize Viewer and Configure JPG Options**\n\n```csharp\nusing (Viewer viewer = new Viewer(\"@\" + YOUR_DOCUMENT_DIRECTORY + \"/sample.cmx\"))\n{\n    JpgViewOptions options = new JpgViewOptions(pageFilePathFormat);\n    viewer.View(options);\n}\n```\n\n- **JpgViewOptions**: This sets up the rendering process to produce JPG images. You can further customize image quality and size if needed.\n\n### Rendering CMX Document to PNG\n\nPNG format provides high-quality, lossless document previews.\n\n#### Step-by-Step Guide:\n\n**1.
