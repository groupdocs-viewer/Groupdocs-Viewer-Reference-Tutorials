# Ajouter un filigrane à un document avec GroupDocs.Viewer

Bienvenue sur le hub de tutoriels GroupDocs.Viewer. Vous y trouverez une collection complète de tutoriels et de guides pour vous aider à maîtriser nos puissantes API de visualisation de documents pour .NET et Java. Que vous développiez une application web, une application de bureau ou une solution mobile, GroupDocs.Viewer vous permet de rendre et d'afficher de manière transparente une grande variété de formats de documents, notamment PDF, Microsoft Office (Word, Excel, PowerPoint), CAD, images, et plus encore. **Vous pouvez également ajouter facilement un filigrane à un document** pour protéger votre contenu et renforcer votre image de marque.

## Comment ajouter un filigrane à un document avec GroupDocs.Viewer

L'ajout d'un filigrane est une exigence courante pour les entreprises qui doivent protéger des informations propriétaires ou marquer chaque page exportée. Avec GroupDocs.Viewer, vous pouvez appliquer des filigranes texte ou image pendant le processus de rendu sans modifier le fichier original. Cette approche fonctionne pour les PDF, les fichiers Word, les feuilles de calcul et même les dessins CAD, vous offrant une couche de protection cohérente sur tous les formats pris en charge.

Ci-dessous, nous parcourrons les principales sections de notre bibliothèque de tutoriels. Chaque section contient des guides étape par étape, des extraits de code et des conseils de bonnes pratiques, afin que vous puissiez rapidement commencer à rendre, convertir et ajouter des filigranes aux documents dans vos applications.
