# Aggiungere filigrana a un documento con GroupDocs.Viewer

Benvenuti al hub dei tutorial di GroupDocs.Viewer. Qui troverete una raccolta completa di tutorial e guide per aiutarvi a padroneggiare le nostre potenti API di visualizzazione documenti per .NET e Java. Che stiate creando un'applicazione web, un'app desktop o una soluzione mobile, GroupDocs.Viewer vi consente di rendere e visualizzare senza problemi un'ampia varietà di formati di documento, tra cui PDF, Microsoft Office (Word, Excel, PowerPoint), CAD, immagini e altro. **È possibile aggiungere facilmente una filigrana al documento** per proteggere i contenuti e rafforzare il branding.

## Come aggiungere una filigrana a un documento usando GroupDocs.Viewer

Aggiungere una filigrana è una necessità comune per le aziende che devono proteggere informazioni proprietarie o marchiare ogni pagina esportata. Con GroupDocs.Viewer è possibile applicare filigrane di testo o immagine durante il processo di rendering senza modificare il file originale. Questo approccio funziona per PDF, file Word, fogli di calcolo e anche disegni CAD, fornendo uno strato di protezione coerente su tutti i formati supportati.

Di seguito vi guideremo attraverso le sezioni principali della nostra libreria di tutorial. Ogni sezione contiene guide passo‑passo, frammenti di codice e consigli sulle migliori pratiche, così potrete iniziare rapidamente a renderizzare, convertire e aggiungere filigrane ai documenti nelle vostre applicazioni.
