# GroupDocs Viewer tutorial – Rendering e visualizzazione dei documenti

Benvenuti al hub del **GroupDocs Viewer tutorial**. In questo tutorial troverete una raccolta completa di guide che vi aiuteranno a padroneggiare le nostre potenti API di visualizzazione documenti per .NET e Java. Che stiate creando un'app web, una soluzione desktop o un'esperienza mobile, GroupDocs Viewer consente di renderizzare e visualizzare senza problemi un'ampia varietà di formati di documento, tra cui PDF, Microsoft Office (Word, Excel, PowerPoint), CAD, immagini e altro.

## Risposte rapide
- **Che cos'è il tutorial GroupDocs Viewer?** Una guida passo‑passo per renderizzare, convertire e visualizzare oltre 170 formati di file utilizzando GroupDocs Viewer per .NET e Java.  
- **Quali piattaforme sono supportate?** .NET (Framework, Core, .NET 5/6) e Java (8+).  
- **Posso renderizzare PDF e immagini?** Sì – è possibile esportare in HTML, JPEG, PNG e PDF.

## Domande frequenti

**Q: È necessario installare software aggiuntivo per utilizzare GroupDocs Viewer?**  
A: Non è richiesto alcun software esterno; l'API gestisce tutto il rendering internamente.

**Q: Posso renderizzare documenti protetti da password?**  
A: Sì, è possibile fornire la password durante il caricamento del documento tramite l'API.

**Q: In che modo la cache migliora le prestazioni?**  
A: La cache memorizza le pagine o le immagini renderizzate, riducendo il tempo di elaborazione per le richieste successive.

**Q: È possibile aggiungere filigrane alle pagine renderizzate?**  
A: Assolutamente—tutorial dedicati mostrano come sovrapporre filigrane di testo o immagine durante il rendering.

**Q: Quali formati di file sono supportati nativamente?**  
A: Oltre 170 formati, tra cui PDF, DOCX, XLSX, PPTX, DWG, DWF, SVG e molti tipi di immagine.
