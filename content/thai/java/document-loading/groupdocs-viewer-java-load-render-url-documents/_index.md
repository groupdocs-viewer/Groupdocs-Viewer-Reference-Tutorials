## คู่มือการใช้งาน

ตอนนี้สภาพแวดล้อมของคุณได้รับการตั้งค่าเรียบร้อยแล้ว ลองใช้งานฟังก์ชันการโหลดและแสดงเอกสารจาก URL กัน

### โหลดเอกสารจาก URL

ฟีเจอร์นี้ช่วยให้คุณดาวน์โหลดเอกสารโดยตรงจาก URL ที่ระบุ และแสดงผลในรูปแบบ HTML โดยใช้ GroupDocs.Viewer ดังต่อไปนี้:

#### ขั้นตอนที่ 1: เปิด InputStream จาก URL

เริ่มต้นด้วยการสร้าง `InputStream` ที่เชื่อมต่อกับ URL เป้าหมายของคุณ สตรีมนี้จะใช้เป็นอินพุตสำหรับการเรนเดอร์

```java
String url = "https://cms.admin.containerize.com/templates/groupdocs/images/logos/groupdocs-logo.png";
try (InputStream fileStream = new URL(url).openStream()) {
    // ดำเนินการตามการตั้งค่าการดูเอกสาร
} catch (Exception e) {
    throw new RuntimeException("Failed to open stream from the URL", e);
}
```

#### ขั้นตอนที่ 2: กำหนดค่าตัวเลือกมุมมอง HTML

ขั้นตอนต่อไปคือการกำหนดค่า `HtmlViewOptions` สำหรับการเรนเดอร์ ระบุว่าคุณต้องการบันทึกเนื้อหาที่เรนเดอร์ไว้ที่ไหนและอย่างไร

```java
Path outputDirectory = Paths.get("YOUR_OUTPUT_DIRECTORY");
Path pageFilePathFormat = outputDirectory.resolve("page_{0}.html");
HtmlViewOptions viewOptions = HtmlViewOptions.forEmbeddedResources(pageFilePathFormat);
```

#### ขั้นตอนที่ 3: สร้างอินสแตนซ์ของโปรแกรมดูและเรนเดอร์

สุดท้ายสร้างอินสแตนซ์ของ `Viewer` ด้วยสตรีมอินพุตของ URL และใช้ในการเรนเดอร์เอกสารของคุณเป็นรูปแบบ HTML

```java
try (Viewer viewer = new Viewer(fileStream)) {
    viewer.view(viewOptions);
}
```

### เคล็ดลับการแก้ไขปัญหา

- **ปัญหาการเชื่อมต่อ:** ตรวจสอบให้แน่ใจว่า URL ถูกต้องและสามารถเข้าถึงได้ ข้อจำกัดของเครือข่ายอาจขัดขวางการเข้าถึง URL บางส่วน
- **IOข้อยกเว้น:** จัดการข้อยกเว้นที่เกี่ยวข้องกับการดำเนินการไฟล์อย่างเรียบร้อยและมีการส่งข้อความแสดงข้อผิดพลาดที่ให้ข้อมูล
