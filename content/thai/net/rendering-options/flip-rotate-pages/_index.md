# พลิกและหมุนหน้า

## การแนะนำ
ในบทช่วยสอนนี้ เราจะเจาะลึกฟังก์ชันการทำงานของ Groupdocs.Viewer สำหรับ .NET โดยเน้นที่การพลิกและหมุนหน้าโดยเฉพาะ Groupdocs.Viewer สำหรับ .NET เป็นเครื่องมืออันทรงพลังที่ออกแบบมาเพื่อแสดงเอกสารในรูปแบบต่างๆ ในแอปพลิเคชัน .NET ไม่ว่าคุณจะกำลังพัฒนาระบบการจัดการเอกสารหรือต้องการผสานรวมความสามารถในการดูเอกสารเข้ากับซอฟต์แวร์ของคุณ Groupdocs.Viewer สำหรับ .NET ก็มีโซลูชันที่มีประสิทธิภาพ
## ข้อกำหนดเบื้องต้น
ก่อนที่เราจะเริ่ม ให้แน่ใจว่าคุณได้ตั้งค่าข้อกำหนดเบื้องต้นดังต่อไปนี้:
### การติดตั้ง Groupdocs.Viewer สำหรับ .NET
หากต้องการใช้ Groupdocs.Viewer สำหรับ .NET คุณต้องติดตั้งแพ็กเกจผ่านตัวจัดการแพ็กเกจ NuGet คุณสามารถดูคำแนะนำการติดตั้งโดยละเอียดได้ใน [เอกสารประกอบ](https://tutorials-groupdocs.com/viewer/net/).

## นำเข้าเนมสเปซ
ตรวจสอบให้แน่ใจว่าคุณมีเนมสเปซที่จำเป็นนำเข้าในโครงการของคุณเพื่อใช้ Groupdocs.Viewer สำหรับ .NET ได้อย่างมีประสิทธิภาพ
```csharp
using System;
using System.IO;
using GroupDocs.Viewer.Options;
```

มาแบ่งกระบวนการพลิกและหมุนหน้าโดยใช้ Groupdocs.Viewer สำหรับ .NET ออกเป็นขั้นตอนง่าย ๆ กัน:
## ขั้นตอนที่ 1: ตั้งค่าไดเรกทอรีเอาต์พุตและเส้นทางไฟล์
กำหนดไดเร็กทอรีที่คุณต้องการบันทึกไฟล์เอาต์พุต และระบุเส้นทางไฟล์เอาต์พุต
```csharp
string outputDirectory = "Your Document Directory";
string outputFilePath = Path.Combine(outputDirectory, "output.pdf");
```
## ขั้นตอนที่ 2: เริ่มต้น Viewer Object
สร้างอินสแตนซ์ของคลาส Viewer โดยส่งเส้นทางไปยังเอกสารที่คุณต้องการดู
```csharp
using (Viewer viewer = new Viewer(TestFiles.SAMPLE_DOCX))
```
## ขั้นตอนที่ 3: กำหนดค่าตัวเลือกมุมมอง
ตั้งค่าตัวเลือกการดู เช่น การระบุรูปแบบไฟล์เอาต์พุตและการตั้งค่าเพิ่มเติมเช่นการหมุนหน้า
```csharp
PdfViewOptions viewOptions = new PdfViewOptions(outputFilePath);
viewOptions.RotatePage(1, Rotation.On90Degree);
```
## ขั้นตอนที่ 4: เรนเดอร์เอกสาร
เรียกใช้วิธีการมุมมองของวัตถุ Viewer และส่งตัวเลือกมุมมอง
```csharp
viewer.View(viewOptions);
```
## ขั้นตอนที่ 5: แสดงข้อความแสดงว่าสำเร็จ
แจ้งให้ผู้ใช้ทราบว่าเอกสารได้รับการแสดงสำเร็จแล้ว และระบุไดเรกทอรีเอาต์พุตสำหรับการตรวจสอบ
```csharp
Console.WriteLine($"\nSource document rendered successfully.\nCheck output in {outputDirectory}.");
```

## บทสรุป
โดยสรุป Groupdocs.Viewer สำหรับ .NET นำเสนอความสามารถอันทรงพลังสำหรับการแสดงเอกสาร รวมถึงการพลิกและหมุนหน้า ด้วยการทำตามขั้นตอนที่ระบุไว้ในบทช่วยสอนนี้ คุณสามารถผสานรวมคุณสมบัติเหล่านี้เข้ากับแอปพลิเคชัน .NET ของคุณได้อย่างราบรื่น ช่วยเพิ่มประสบการณ์การดูเอกสารให้กับผู้ใช้ของคุณ
