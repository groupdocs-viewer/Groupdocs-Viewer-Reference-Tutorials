# ตั้งค่าใบอนุญาตจากสตรีม

## การแนะนำ
คุณกำลังมองหาวิธีเพิ่มประสิทธิภาพให้แอปพลิเคชั่น .NET ของคุณด้วยความสามารถในการดูเอกสารขั้นสูงอยู่หรือไม่ GroupDocs.Viewer สำหรับ .NET นำเสนอโซลูชันที่ครอบคลุมเพื่อผสานรวมฟังก์ชันการดูเอกสารเข้ากับโปรเจ็กต์ของคุณได้อย่างราบรื่น ในบทช่วยสอนนี้ เราจะเจาะลึกถึงกระบวนการใช้ประโยชน์จาก GroupDocs.Viewer สำหรับ .NET เพื่อเสริมประสิทธิภาพให้แอปพลิเคชั่นของคุณด้วยความสามารถในการดูเอกสารอันทรงพลัง 

![ตั้งค่าใบอนุญาตจาก Stream ด้วย GroupDocs.Viewer สำหรับ .NET](/viewer/getting-started/set-license-from-stream.png)

## ข้อกำหนดเบื้องต้น
ก่อนที่จะเจาะลึกลงไปในกระบวนการบูรณาการ ให้แน่ใจว่าคุณมีข้อกำหนดเบื้องต้นดังต่อไปนี้:
1. ความรู้พื้นฐานในการพัฒนา .NET: ความคุ้นเคยกับ C# และ .NET framework ถือเป็นสิ่งสำคัญที่ต้องปฏิบัติตามพร้อมกับบทช่วยสอนนี้
   
2. แพ็กเกจ GroupDocs.Viewer สำหรับ .NET: ตรวจสอบให้แน่ใจว่าคุณได้ดาวน์โหลดและติดตั้งแพ็กเกจ GroupDocs.Viewer สำหรับ .NET แล้ว คุณสามารถรับได้จาก [ลิงค์ดาวน์โหลด](https://releases-groupdocs.com/viewer/net/).
3. การเข้าถึงเอกสาร GroupDocs: เก็บ [เอกสารประกอบ](https://tutorials.groupdocs.com/viewer/net/) มีประโยชน์สำหรับการสอนในระหว่างกระบวนการบูรณาการ

## นำเข้าเนมสเปซ
ในการเริ่มต้น ให้นำเข้าเนมสเปซที่จำเป็นลงในแอปพลิเคชัน .NET ของคุณ ทำตามขั้นตอนเหล่านี้:
### ขั้นตอนที่ 1: เปิดโครงการ .NET ของคุณ
ตรวจสอบให้แน่ใจว่าคุณได้เปิดโครงการ .NET ไว้ในสภาพแวดล้อมการพัฒนาที่คุณต้องการ
### ขั้นตอนที่ 2: เพิ่มเนมสเปซ GroupDocs.Viewer
ในไฟล์โค้ดของคุณ เพิ่มเนมสเปซต่อไปนี้เพื่อเข้าถึงฟังก์ชันการทำงานของ GroupDocs.Viewer:
```csharp
using System;
using System.IO;
```
## ตั้งค่าใบอนุญาตจากสตรีม
ขั้นตอนต่อไปคือการตั้งค่าใบอนุญาตจากสตรีม ทำตามขั้นตอนโดยละเอียดเหล่านี้:
### ขั้นตอนที่ 1: กำหนดไดเรกทอรีเอาต์พุต
ตั้งค่าไดเร็กทอรีที่จะเก็บเอกสารของคุณโดยการกำหนดไดเร็กทอรีเอาท์พุต:
```csharp
string outputDirectory = "Your Document Directory";
```
### ขั้นตอนที่ 2: ตรวจสอบการมีอยู่ของไฟล์ใบอนุญาต
ตรวจสอบว่าไฟล์ใบอนุญาตมีอยู่ในไดเร็กทอรีโครงการของคุณหรือไม่:
```csharp
if (File.Exists(Utils.LicensePath))
```
### ขั้นตอนที่ 3: ตั้งค่าใบอนุญาต
หากมีไฟล์ใบอนุญาตอยู่ ให้ตั้งค่าใบอนุญาตโดยใช้สตรีมที่ให้มา:
```csharp
using (FileStream stream = File.OpenRead(Utils.LicensePath))
{
    License license = new License();
    license.SetLicense(stream);
}
```
### ขั้นตอนที่ 4: จัดการการขาดใบอนุญาต
หากไม่พบไฟล์ใบอนุญาต โปรดให้คำแนะนำในการรับใบอนุญาต:
```csharp
else
{
    Console.WriteLine("\nWe do not ship any license with this example. " +
                      "\nVisit the GroupDocs site to obtain either a temporary or permanent license. " +
                      "\nLearn more about licensing at https://purchase.groupdocs.com/faqs/licensing " +
                      "\nLearn how to request a temporary license at https://purchase.groupdocs.com/temporary-license.");
}
```

## บทสรุป
ขอแสดงความยินดี! คุณได้เรียนรู้วิธีการผสานรวม GroupDocs.Viewer สำหรับ .NET เข้ากับแอปพลิเคชันของคุณสำเร็จแล้ว ด้วยเครื่องมืออันทรงพลังนี้ คุณสามารถดูรูปแบบเอกสารต่างๆ ภายในโปรเจ็กต์ .NET ของคุณได้อย่างง่ายดาย ช่วยเพิ่มประสบการณ์ของผู้ใช้และประสิทธิภาพการทำงาน
## คำถามที่พบบ่อย
### ฉันต้องมีใบอนุญาตเพื่อใช้ GroupDocs.Viewer สำหรับ .NET หรือไม่
ใช่ คุณต้องมีใบอนุญาตเพื่อใช้ GroupDocs.Viewer สำหรับ .NET คุณสามารถขอรับใบอนุญาตชั่วคราวหรือถาวรได้จากเว็บไซต์ GroupDocs
### ฉันสามารถรวม GroupDocs.Viewer เข้ากับแอปพลิเคชัน ASP.NET ของฉันได้หรือไม่
แน่นอน! GroupDocs.Viewer สำหรับ .NET สามารถบูรณาการกับแอพพลิเคชันเดสก์ท็อปและเว็บ รวมถึง ASP.NET ได้อย่างราบรื่น
### รูปแบบเอกสารใดบ้างที่ได้รับการรองรับโดย GroupDocs.Viewer?
GroupDocs.Viewer รองรับรูปแบบเอกสารต่างๆ มากมาย รวมถึง PDF, Microsoft Office (Word, Excel, PowerPoint), รูปภาพ และอื่นๆ อีกมากมาย
### GroupDocs.Viewer เข้ากันได้กับ .NET Core ได้หรือไม่
ใช่ GroupDocs.Viewer สำหรับ .NET เข้ากันได้กับทั้ง .NET Framework และ .NET Core
### ฉันสามารถปรับแต่งอินเทอร์เฟซผู้ชมให้สอดคล้องกับธีมแอปพลิเคชันของฉันได้หรือไม่
ใช่ GroupDocs.Viewer มีตัวเลือกการปรับแต่งมากมาย ช่วยให้คุณปรับแต่งอินเทอร์เฟซการดูเพื่อให้ตรงกับธีมแอปพลิเคชันของคุณได้อย่างราบรื่น
