# รับข้อมูลการดูเอกสาร PDF

GroupDocs.Viewer สำหรับ .NET เป็นเครื่องมืออันทรงพลังที่ออกแบบมาเพื่อปรับปรุงการดูเอกสารภายในแอปพลิเคชัน .NET ไม่ว่าคุณจะกำลังจัดการกับ PDF เอกสาร Word สเปรดชีต Excel หรือการนำเสนอ PowerPoint ไลบรารีนี้จะทำให้กระบวนการเรนเดอร์และการโต้ตอบกับรูปแบบไฟล์ต่างๆ ง่ายขึ้น ในบทช่วยสอนนี้ เราจะเน้นที่การใช้ประโยชน์จากความสามารถของ GroupDocs.Viewer โดยเฉพาะสำหรับการดึงข้อมูลมุมมองจากเอกสาร PDF

ก่อนจะเริ่มบทช่วยสอนนี้ ให้แน่ใจว่าคุณมีข้อกำหนดเบื้องต้นดังต่อไปนี้:
1. การติดตั้ง GroupDocs.Viewer สำหรับ .NET: ตรวจสอบให้แน่ใจว่าคุณได้ดาวน์โหลดและติดตั้งไลบรารี GroupDocs.Viewer แล้ว คุณสามารถรับได้จาก [ลิงค์ดาวน์โหลด](https://releases-groupdocs.com/viewer/net/).   
2. ความรู้พื้นฐานเกี่ยวกับ C#: ความคุ้นเคยกับภาษาการเขียนโปรแกรม C# ถือเป็นสิ่งสำคัญในการทำความเข้าใจและนำตัวอย่างโค้ดที่ให้มาไปใช้
3. การเข้าถึงเอกสาร PDF: เตรียมเอกสาร PDF ที่คุณจะใช้ในการแยกข้อมูลมุมมอง

## นำเข้าเนมสเปซ
ในโครงการ C# ของคุณ นำเข้าเนมสเปซที่จำเป็นเพื่อใช้ฟังก์ชันการทำงานของ GroupDocs.Viewer

```csharp
using System;
using GroupDocs.Viewer.Options;
using GroupDocs.Viewer.Results;
```


ตอนนี้ เรามาดูขั้นตอนในการดึงข้อมูลมุมมองจากเอกสาร PDF โดยใช้ GroupDocs.Viewer สำหรับ .NET กัน
## ขั้นตอนที่ 1: เริ่มต้นวัตถุ Viewer
สร้างวัตถุ Viewer และระบุเส้นทางไปยังเอกสาร PDF เป็นพารามิเตอร์
```csharp
using (Viewer viewer = new Viewer("path/to/your/sample.pdf"))
{
```
## ขั้นตอนที่ 2: กำหนด ViewInfoOptions
ระบุตัวเลือกมุมมอง เช่น มุมมอง HTML เพื่อดึงข้อมูลมุมมอง
```csharp
	ViewInfoOptions options = ViewInfoOptions.ForHtmlView();
```
## ขั้นตอนที่ 3: รับข้อมูลการดู
เรียกใช้เมธอด GetViewInfo เพื่อดึงข้อมูลมุมมองจากเอกสาร PDF
```csharp
	PdfViewInfo info = viewer.GetViewInfo(options) as PdfViewInfo;
```
## ขั้นตอนที่ 4: แสดงข้อมูลมุมมองเอาท์พุต
แสดงข้อมูลมุมมองที่แยกออกมา เช่น ประเภทเอกสาร จำนวนหน้า และสิทธิ์ในการพิมพ์
```csharp
	Console.WriteLine("Document type is: " + info.FileType);
	Console.WriteLine("Pages count: " + info.Pages.Count);
	Console.WriteLine("Printing allowed: " + info.PrintingAllowed);
}
```
