# แสดงผล HTML แบบตอบสนอง

## การแนะนำ
Groupdocs.Viewer สำหรับ .NET เป็นไลบรารีที่มีประสิทธิภาพที่ช่วยให้ผู้พัฒนาสามารถเรนเดอร์เอกสารในรูปแบบต่างๆ ในรูปแบบ HTML ที่ตอบสนองได้ บทช่วยสอนนี้จะแนะนำคุณตลอดกระบวนการเรนเดอร์ HTML ที่ตอบสนองได้โดยใช้ Groupdocs.Viewer สำหรับ .NET เมื่อสิ้นสุดบทช่วยสอนนี้ คุณจะสามารถแปลงเอกสารเป็น HTML ที่ปรับให้เข้ากับขนาดหน้าจอต่างๆ ได้อย่างราบรื่น ช่วยให้มั่นใจได้ถึงประสบการณ์การรับชมที่เหมาะสมที่สุดบนอุปกรณ์ต่างๆ
## ข้อกำหนดเบื้องต้น
ก่อนที่คุณจะเริ่มต้น ให้แน่ใจว่าคุณมีสิ่งต่อไปนี้:
1. Groupdocs.Viewer สำหรับไลบรารี .NET: ดาวน์โหลดและติดตั้งไลบรารีจาก [เว็บไซต์](https://releases-groupdocs.com/viewer/net/).
2. สภาพแวดล้อมการพัฒนา: ตรวจสอบให้แน่ใจว่าคุณมีการตั้งค่าสภาพแวดล้อมการพัฒนาที่เหมาะสมสำหรับการพัฒนา .NET
3. ไฟล์เอกสาร: เตรียมไฟล์เอกสารที่คุณต้องการเรนเดอร์เป็น HTML ที่ตอบสนอง

## นำเข้าเนมสเปซ
หากต้องการเริ่มเรนเดอร์ HTML แบบตอบสนอง ให้นำเข้าเนมสเปซที่จำเป็นลงในโครงการของคุณ:
```csharp
using System;
using System.IO;
using GroupDocs.Viewer.Options;
```

มาแบ่งกระบวนการเรนเดอร์ออกเป็นหลายขั้นตอน:
## ขั้นตอนที่ 1: ตั้งค่าไดเรกทอรีผลลัพธ์
กำหนดไดเรกทอรีที่คุณต้องการบันทึกหน้า HTML ที่แสดงผล:
```csharp
string outputDirectory = "Your Document Directory";
```
## ขั้นตอนที่ 2: กำหนดรูปแบบเส้นทางไฟล์เพจ
ระบุรูปแบบการตั้งชื่อไฟล์ HTML สำหรับแต่ละหน้า:
```csharp
string pageFilePathFormat = Path.Combine(outputDirectory, "page_{0}.html");
```
## ขั้นตอนที่ 3: เริ่มต้น Viewer Object
สร้างอินสแตนซ์ของคลาส Viewer และระบุเอกสารที่ต้องการเรนเดอร์:
```csharp
using (Viewer viewer = new Viewer(TestFiles.SAMPLE_DOCX))
{
    // โค้ดการเรนเดอร์จะอยู่ที่นี่
}
```
## ขั้นตอนที่ 4: กำหนดค่าตัวเลือกมุมมอง HTML
ตั้งค่าตัวเลือกมุมมอง HTML รวมถึงการเปิดใช้งานการแสดงผลแบบตอบสนอง:
```csharp
HtmlViewOptions options = HtmlViewOptions.ForEmbeddedResources(pageFilePathFormat);
options.RenderResponsive = true;
```
## ขั้นตอนที่ 5: เรนเดอร์เอกสารเป็น HTML
ใช้เมธอด View ของอ็อบเจ็กต์ Viewer เพื่อเรนเดอร์เอกสารเป็น HTML:
```csharp
viewer.View(options);
```
## ขั้นตอนที่ 6: ส่งข้อความแสดงความสำเร็จ
แสดงข้อความแจ้งว่าเอกสารได้รับการแสดงสำเร็จแล้ว:
```csharp
Console.WriteLine($"\nSource document rendered successfully.\nCheck output in {outputDirectory}.");
```
