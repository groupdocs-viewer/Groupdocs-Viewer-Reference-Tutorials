# เรนเดอร์เส้นกริด

## การแนะนำ
ยินดีต้อนรับสู่คู่มือทีละขั้นตอนเกี่ยวกับการใช้ GroupDocs.Viewer สำหรับ .NET เพื่อแสดงเส้นกริดในเอกสารของคุณ ไม่ว่าคุณจะเป็นนักพัฒนาที่มีประสบการณ์หรือเป็นผู้ใช้ใหม่ของกรอบงาน .NET บทช่วยสอนนี้จะแนะนำคุณตลอดกระบวนการด้วยคำอธิบายโดยละเอียดและตัวอย่างที่ทำตามได้ง่าย
## ข้อกำหนดเบื้องต้น
ก่อนจะเริ่มบทช่วยสอนนี้ ให้แน่ใจว่าคุณมีข้อกำหนดเบื้องต้นดังต่อไปนี้:
- GroupDocs.Viewer สำหรับ .NET: ดาวน์โหลดและติดตั้งไลบรารีจาก [เว็บไซต์อย่างเป็นทางการ](https://releases-groupdocs.com/viewer/net/).
- ไดเร็กทอรีเอกสารของคุณ: ตรวจสอบให้แน่ใจว่าคุณมีไดเร็กทอรีที่กำหนดไว้สำหรับเอกสารของคุณ และแทนที่ "ไดเร็กทอรีเอกสารของคุณ" ในชิ้นส่วนโค้ดที่ให้มาด้วยเส้นทางจริง
ตอนนี้คุณได้ตั้งค่าทุกอย่างเรียบร้อยแล้ว มาเริ่มกันเลย
## นำเข้าเนมสเปซ
ในโครงการ .NET ของคุณ เริ่มต้นด้วยการนำเข้าเนมสเปซที่จำเป็น:
```csharp
using System;
using System.IO;
using GroupDocs.Viewer.Options;
```
## ขั้นตอนที่ 1: ตั้งค่าไดเรกทอรีเอกสาร
เริ่มต้นด้วยการระบุเส้นทางไปยังไดเร็กทอรีเอกสารของคุณ:
```csharp
string outputDirectory = "Your Document Directory";
```
แทนที่ "ไดเรกทอรีเอกสารของคุณ" ด้วยเส้นทางจริงที่คุณจัดเก็บเอกสารของคุณ
## ขั้นตอนที่ 2: กำหนดเส้นทางไฟล์และรูปแบบเอาท์พุต HTML
สร้างตัวแปรเพื่อจัดเก็บรูปแบบเส้นทางไฟล์สำหรับแต่ละหน้าและรูปแบบ HTML เอาท์พุต:
```csharp
string pageFilePathFormat = Path.Combine(outputDirectory, "page_{0}.html");
```
บรรทัดนี้สร้างเส้นทางไฟล์สำหรับแต่ละหน้าในรูปแบบที่ระบุ
## ขั้นตอนที่ 3: เริ่มต้น GroupDocs.Viewer
สร้างอินสแตนซ์ของคลาส Viewer ด้วยเอกสารที่คุณต้องการดู:
```csharp
using (Viewer viewer = new Viewer(outputDirectory + "SAMPLE.XLSX"))
{
    // ขั้นตอนต่อไปจะดำเนินการภายในบล็อคนี้
}
```
อย่าลืมแทนที่ "SAMPLE.XLSX" ด้วยชื่อเอกสารจริงของคุณ
## ขั้นตอนที่ 4: กำหนดค่าตัวเลือกมุมมอง HTML
ตั้งค่าตัวเลือกมุมมอง HTML โดยเฉพาะการเปิดใช้งานการเรนเดอร์เส้นตาราง:
```csharp
HtmlViewOptions options = HtmlViewOptions.ForEmbeddedResources(pageFilePathFormat);
options.SpreadsheetOptions.RenderGridLines = true;
```
โค้ดตัวอย่างนี้กำหนดค่าตัวเลือกมุมมอง HTML เพื่อฝังทรัพยากรและแสดงเส้นตารางสำหรับเอกสารสเปรดชีต
## ขั้นตอนที่ 5: เรนเดอร์เส้นกริด
เรียกใช้ `View` วิธีการเรนเดอร์เอกสารด้วยตัวเลือกที่ระบุสำหรับหน้า 1, 2 และ 3:
```csharp
viewer.View(options, 1, 2, 3);
```
ปรับหมายเลขหน้าตามความต้องการของคุณ
เสร็จเรียบร้อย! คุณได้เรนเดอร์เส้นตารางโดยใช้ GroupDocs.Viewer สำหรับ .NET สำเร็จแล้ว
