# เรนเดอร์พื้นที่พิมพ์ด้วย GroupDocs.Viewer สำหรับ .NET

## การแนะนำ
ยินดีต้อนรับสู่คู่มือที่ครอบคลุมนี้เกี่ยวกับการใช้ประโยชน์จาก GroupDocs.Viewer สำหรับ .NET เพื่อเรนเดอร์พื้นที่พิมพ์ในเอกสารของคุณ หากคุณเป็นนักพัฒนา .NET ที่กำลังมองหาโซลูชันที่มีประสิทธิภาพสำหรับการเรนเดอร์เอกสาร คุณมาถูกที่แล้ว ในบทช่วยสอนนี้ เราจะแนะนำคุณเกี่ยวกับกระบวนการเรนเดอร์พื้นที่พิมพ์โดยใช้ GroupDocs.Viewer เพื่อให้แน่ใจว่าคุณจะได้รับประสบการณ์ที่ราบรื่นในแอปพลิเคชันของคุณ
## ข้อกำหนดเบื้องต้น
ก่อนจะเริ่มบทช่วยสอนนี้ ให้แน่ใจว่าคุณมีข้อกำหนดเบื้องต้นดังต่อไปนี้:
- ความรู้พื้นฐานเกี่ยวกับการพัฒนา C# และ .NET
- GroupDocs.Viewer สำหรับ .NET ได้รับการติดตั้งแล้ว คุณสามารถดาวน์โหลดได้ [ที่นี่](https://releases-groupdocs.com/viewer/net/).
- เอกสารตัวอย่าง (เช่น "SAMPLE.XLSX") ในไดเร็กทอรีเอกสารที่คุณระบุ
## นำเข้าเนมสเปซ
ตรวจสอบให้แน่ใจว่าคุณได้นำเข้าเนมสเปซที่จำเป็นในโค้ด C# ของคุณสำหรับการใช้งานอย่างถูกต้อง:
```csharp
using System;
using System.IO;
using GroupDocs.Viewer.Options;
```
## ขั้นตอนที่ 1: ตั้งค่าไดเรกทอรีเอกสาร
เริ่มต้นโดยระบุไดเรกทอรีเอาท์พุตสำหรับหน้า HTML ที่จะแสดงผล:
```csharp
string outputDirectory = "Your Document Directory";
```
## ขั้นตอนที่ 2: กำหนดรูปแบบเส้นทางไฟล์เพจ
สร้างรูปแบบสำหรับเส้นทางไฟล์เพจ โดยรวมไดเร็กทอรีเอาท์พุตและตัวแทนสำหรับหมายเลขหน้า:
```csharp
string pageFilePathFormat = Path.Combine(outputDirectory, "page_{0}.html");
```
## ขั้นตอนที่ 3: เริ่มต้น GroupDocs.Viewer
สร้างอินสแตนซ์ของคลาส Viewer พร้อมด้วยเส้นทางไปยังเอกสารตัวอย่างของคุณ:
```csharp
using (Viewer viewer = new Viewer("SAMPLE.XLSX"))
{
```
## ขั้นตอนที่ 4: กำหนดค่าตัวเลือกมุมมอง HTML
กำหนดค่าตัวเลือกมุมมอง HTML โดยระบุรูปแบบเส้นทางไฟล์หน้าและเปิดใช้งานตัวเลือกสำหรับการแสดงพื้นที่พิมพ์:
```csharp
HtmlViewOptions options = HtmlViewOptions.ForEmbeddedResources(pageFilePathFormat);
options.SpreadsheetOptions = SpreadsheetOptions.ForRenderingPrintArea();
```
## ขั้นตอนที่ 5: เรนเดอร์เอกสาร
เรียกใช้ `View` วิธีการเรนเดอร์เอกสารด้วยตัวเลือกที่ระบุ:
```csharp
viewer.View(options);
```
## ขั้นตอนที่ 6: แสดงข้อความแสดงว่าสำเร็จ
พิมพ์ข้อความแสดงความสำเร็จ ระบุว่าเอกสารต้นฉบับได้รับการแสดงผลสำเร็จแล้ว:
```csharp
Console.WriteLine($"\nSource document rendered successfully.\nCheck output in {outputDirectory}.");
```
