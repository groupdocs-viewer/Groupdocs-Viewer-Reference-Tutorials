## คู่มือการใช้งาน
ตอนนี้คุณได้ตั้งค่า GroupDocs.Viewer .NET เรียบร้อยแล้ว มาลองดูวิธีการแสดงเอกสาร Apple Numbers เป็นรูปแบบต่างๆ กัน แต่ละส่วนจะแนะนำคุณเกี่ยวกับฟีเจอร์เฉพาะ

### เรนเดอร์เอกสารหมายเลข Apple เป็น HTML
**ภาพรวม:** แปลงไฟล์ Apple Numbers ของคุณเป็นเอกสาร HTML ที่มีทรัพยากรที่ฝังไว้เพื่อการดูและการแชร์ที่ง่ายดาย

#### ขั้นตอนที่ 1: กำหนดไดเรกทอรีผลลัพธ์
```csharp
string outputDirectory = "YOUR_OUTPUT_DIRECTORY";
```

#### ขั้นตอนที่ 2: ตั้งค่าอินสแตนซ์ของผู้ชม
```csharp
using (Viewer viewer = new Viewer("YOUR_DOCUMENT_DIRECTORY/SAMPLE_NUMBERS"))
{
    // ตัวอย่างโค้ดยังคงดำเนินต่อไป...
}
```
*ทำไม* การใช้ `using` คำชี้แจงนี้ช่วยให้แน่ใจว่ามีการกำจัดทรัพยากรอย่างเหมาะสมและป้องกันการรั่วไหลของหน่วยความจำ

#### ขั้นตอนที่ 3: กำหนดค่าตัวเลือกมุมมอง HTML
```csharp
HtmlViewOptions options = HtmlViewOptions.ForEmbeddedResources(pageFileFullPath);
```

#### ขั้นตอนที่ 4: เรนเดอร์เอกสาร
```csharp
viewer.View(options);
```
*ทำไม* ขั้นตอนนี้จะแปลงเอกสารของคุณเป็นรูปแบบเว็บที่สามารถเข้าถึงได้

**เคล็ดลับการแก้ไขปัญหา:** ตรวจสอบให้แน่ใจว่าเส้นทางไฟล์ทั้งหมดถูกต้องและสามารถเข้าถึงได้เพื่อหลีกเลี่ยงข้อผิดพลาดรันไทม์

### เรนเดอร์เอกสารหมายเลข Apple เป็น JPG
**ภาพรวม:** แปลงไฟล์ Apple Numbers ของคุณเป็นภาพ JPEG คุณภาพสูง เหมาะสำหรับการนำเสนอหรือการแชร์

#### ขั้นตอนที่ 1: กำหนดเส้นทางเอาต์พุต
```csharp
string pageFileFullPath = Path.Combine("YOUR_OUTPUT_DIRECTORY", "Numbers_result.jpg");
```

#### ขั้นตอนที่ 2: ตั้งค่าตัวแสดงและตัวเลือก
```csharp
using (Viewer viewer = new Viewer("YOUR_DOCUMENT_DIRECTORY/SAMPLE_NUMBERS"))
{
    JpgViewOptions options = new JpgViewOptions(pageFileFullPath);
    // ตัวอย่างโค้ดยังคงดำเนินต่อไป...
}
```

#### ขั้นตอนที่ 3: เรนเดอร์เอกสาร
```csharp
viewer.View(options);
```
*ทำไม* การแปลงเป็น JPG ช่วยให้มั่นใจถึงความเข้ากันได้ระหว่างแพลตฟอร์มและอุปกรณ์ต่างๆ

### เรนเดอร์เอกสารหมายเลข Apple เป็น PNG
**ภาพรวม:** แปลงเอกสารของคุณเป็นภาพ PNG เพื่อคุณภาพที่ไม่สูญเสียข้อมูล

#### ขั้นตอน:
1. **กำหนดเส้นทางเอาต์พุต**
2. **ตั้งค่าตัวแสดงและตัวเลือก**
3. **แสดงผลเอกสาร**

```csharp
PngViewOptions options = new PngViewOptions(pageFileFullPath);
viewer.View(options);
```

### เรนเดอร์เอกสารหมายเลข Apple เป็น PDF
**ภาพรวม:** สร้างไฟล์ PDF แบบพกพาจากเอกสาร Apple Numbers ของคุณ

#### ขั้นตอน:
1. **กำหนดเส้นทางเอาต์พุต**
2. **ตั้งค่าตัวแสดงและตัวเลือก**
3. **แสดงผลเอกสาร**

```csharp
PdfViewOptions options = new PdfViewOptions(pageFileFullPath);
viewer.View(options);
```
