# เรนเดอร์เค้าโครงทั้งหมดในรูปแบบ CAD

## การแนะนำ
GroupDocs.Viewer สำหรับ .NET ถือเป็นโซลูชันที่มีความยืดหยุ่นสูงในด้านการจัดการและการแสดงภาพเอกสาร ช่วยให้นักพัฒนาสามารถแสดงผลเอกสารประเภทต่างๆ ได้อย่างง่ายดายภายในแอปพลิเคชัน .NET ความสามารถมากมายของ GroupDocs.Viewer สำหรับ .NET คือการเรนเดอร์แบบ CAD ได้อย่างมีประสิทธิภาพ รวมถึงเลย์เอาต์ที่ซับซ้อนที่เกี่ยวข้อง ในบทช่วยสอนนี้ เราจะเจาะลึกถึงกระบวนการใช้ประโยชน์จาก GroupDocs.Viewer สำหรับ .NET ในการเรนเดอร์เลย์เอาต์ทั้งหมดที่มีอยู่ในแบบ CAD 
## ข้อกำหนดเบื้องต้น
ก่อนที่จะเริ่มบทช่วยสอนนี้ ให้แน่ใจว่าคุณมีข้อกำหนดเบื้องต้นดังต่อไปนี้:
1. ความเข้าใจพื้นฐานในการพัฒนา .NET: ความคุ้นเคยกับหลักพื้นฐานการพัฒนา .NET จะเป็นประโยชน์ในการทำความเข้าใจขั้นตอนการใช้งานที่ระบุไว้ในบทช่วยสอนนี้
2. การติดตั้ง GroupDocs.Viewer สำหรับ .NET: ตรวจสอบให้แน่ใจว่าคุณได้ติดตั้งไลบรารี GroupDocs.Viewer สำหรับ .NET แล้ว คุณสามารถดาวน์โหลดได้จาก [เว็บไซต์](https://releases-groupdocs.com/viewer/net/).
3. ไฟล์ CAD Drawing: รับไฟล์ CAD Drawing ที่คุณต้องการเรนเดอร์ ซึ่งอาจรวมถึงไฟล์ DWG ที่มีเลย์เอาต์หลายแบบ
4. สภาพแวดล้อมการพัฒนา: ตั้งค่าสภาพแวดล้อมการพัฒนาที่คุณต้องการด้วยเครื่องมือและการอ้างอิงที่จำเป็น

## นำเข้าเนมสเปซ
ขั้นแรก ตรวจสอบให้แน่ใจว่าคุณได้นำเข้าเนมสเปซที่จำเป็นไปยังโปรเจ็กต์ .NET ของคุณแล้ว เนมสเปซเหล่านี้ช่วยให้เข้าถึงฟังก์ชันต่างๆ ที่จำเป็นสำหรับการเรนเดอร์ภาพวาด CAD ด้วย GroupDocs.Viewer ได้

```csharp
using System;
using System.IO;
using GroupDocs.Viewer.Options;
```
## ขั้นตอนที่ 2: นำเข้าเนมสเปซ System.IO
```csharp
using System.IO;
```
## ขั้นตอนที่ 1: ระบุไดเรกทอรีผลลัพธ์
```csharp
string outputDirectory = "Your Document Directory";
```
กำหนดไดเร็กทอรีที่คุณต้องการบันทึกเอาท์พุตที่แสดงผล
## ขั้นตอนที่ 2: กำหนดรูปแบบเส้นทางไฟล์เพจ
```csharp
string pageFilePathFormat = Path.Combine(outputDirectory, "page_{0}.html");
```
ตั้งค่ารูปแบบเส้นทางไฟล์ของหน้าที่แสดงผล ในกรณีนี้ หน้าจะถูกบันทึกเป็นไฟล์ HTML
## ขั้นตอนที่ 3: สร้างอินสแตนซ์ของวัตถุ Viewer
```csharp
using (Viewer viewer = new Viewer(TestFiles.SAMPLE_DWG_WITH_LAYOUTS_AND_LAYERS))
```
สร้างอินสแตนซ์ของคลาส Viewer โดยส่งเส้นทางไปยังไฟล์รูปวาด CAD เป็นพารามิเตอร์
## ขั้นตอนที่ 4: กำหนดค่าตัวเลือกมุมมอง HTML
```csharp
HtmlViewOptions options = HtmlViewOptions.ForEmbeddedResources(pageFilePathFormat);
options.CadOptions.RenderLayouts = true;
```
กำหนดค่าตัวเลือกมุมมอง HTML โดยระบุว่าควรแสดงเค้าโครงสำหรับรูปวาด CAD
## ขั้นตอนที่ 5: เรนเดอร์ภาพวาด CAD
```csharp
viewer.View(options);
```
เรียกใช้วิธีการมุมมองของวัตถุ Viewer โดยส่งตัวเลือกที่กำหนดค่าไว้เพื่อเรนเดอร์รูปวาด CAD
## ขั้นตอนที่ 6: แสดงไดเรกทอรีผลลัพธ์
```csharp
Console.WriteLine($"\nSource document rendered successfully.\nCheck output in {outputDirectory}.");
```
แจ้งให้ผู้ใช้ทราบเกี่ยวกับการเรนเดอร์ที่ประสบความสำเร็จและตำแหน่งของไดเร็กทอรีเอาต์พุต

## บทสรุป
ในบทช่วยสอนนี้ เราได้ศึกษาวิธีการใช้ GroupDocs.Viewer สำหรับ .NET เพื่อแสดงเค้าโครงทั้งหมดที่มีอยู่ในแบบ CAD โดยปฏิบัติตามคำแนะนำทีละขั้นตอนและนำตัวอย่างโค้ดที่ให้มาไปใช้ คุณสามารถผสานฟังก์ชันนี้เข้ากับแอปพลิเคชัน .NET ของคุณได้อย่างราบรื่น จึงช่วยเพิ่มความสามารถในการแสดงภาพเอกสาร
## คำถามที่พบบ่อย
### GroupDocs.Viewer เข้ากันได้กับรูปแบบ CAD ต่างๆ หรือไม่
ใช่ GroupDocs.Viewer รองรับการเรนเดอร์รูปวาด CAD ในรูปแบบเช่น DWG และ DXF
### ฉันสามารถปรับแต่งเอาท์พุตการเรนเดอร์ให้สอดคล้องกับข้อกำหนดของแอปพลิเคชันของฉันได้หรือไม่
แน่นอนว่า GroupDocs.Viewer นำเสนอตัวเลือกต่างๆ มากมายในการปรับแต่งผลลัพธ์การเรนเดอร์ รวมถึงคุณภาพของภาพ ขนาดหน้า และอื่นๆ อีกมากมาย
### GroupDocs.Viewer ต้องมีใบอนุญาตเพิ่มเติมสำหรับการใช้งานเชิงพาณิชย์หรือไม่
ใช่ สำหรับการใช้งานเชิงพาณิชย์ คุณอาจจำเป็นต้องขอใบอนุญาต คุณสามารถขอใบอนุญาตชั่วคราวเพื่อวัตถุประสงค์ในการทดสอบ หรือซื้อใบอนุญาตเชิงพาณิชย์จากเว็บไซต์ได้
### ฉันสามารถเรนเดอร์รูปวาด CAD แบบอะซิงโครนัสด้วย GroupDocs.Viewer ได้หรือไม่
ใช่ GroupDocs.Viewer ให้ความสามารถในการเรนเดอร์แบบอะซิงโครนัส ช่วยให้สามารถจัดการรูปวาด CAD ขนาดใหญ่ได้อย่างมีประสิทธิภาพโดยไม่บล็อกเธรดหลัก
### GroupDocs.Viewer ให้การสนับสนุนการแก้ไขปัญหาและความช่วยเหลือด้านเทคนิคหรือไม่
แน่นอน คุณสามารถขอความช่วยเหลือและการสนับสนุนจากฟอรัมชุมชน GroupDocs.Viewer ได้ [ที่นี่](https://forum-groupdocs.com/c/viewer/9).
