# ปรับขนาดภาพเอาท์พุตสำหรับรูปวาด CAD

## การแนะนำ
แบบวาด CAD มักต้องมีการปรับแต่งเฉพาะเพื่อให้ดูและนำเสนอได้อย่างเหมาะสมที่สุด GroupDocs.Viewer สำหรับ .NET มอบชุดเครื่องมืออันทรงพลังสำหรับจัดการและปรับแต่งผลลัพธ์แบบวาด CAD ในบทช่วยสอนนี้ เราจะแนะนำคุณตลอดกระบวนการปรับขนาดภาพเอาต์พุตสำหรับแบบวาด CAD ทีละขั้นตอน
## ข้อกำหนดเบื้องต้น
ก่อนที่คุณจะเริ่มต้น ให้แน่ใจว่าคุณมีข้อกำหนดเบื้องต้นดังต่อไปนี้:
1. GroupDocs.Viewer สำหรับ .NET: ดาวน์โหลดและติดตั้ง GroupDocs.Viewer สำหรับ .NET จาก [ที่นี่](https://releases-groupdocs.com/viewer/net/).
2. ไดเรกทอรีเอกสาร: เตรียมไดเรกทอรีที่เอกสารของคุณตั้งอยู่
3. ความเข้าใจพื้นฐาน: ทำความคุ้นเคยกับแนวคิดพื้นฐานของการเขียนโปรแกรม .NET

## นำเข้าเนมสเปซ
ก่อนอื่น ตรวจสอบให้แน่ใจว่าคุณได้นำเข้าเนมสเปซที่จำเป็นเพื่อเข้าถึงฟังก์ชันการทำงานของ GroupDocs.Viewer:
```csharp
using System;
using System.IO;
using GroupDocs.Viewer.Options;
```
## ขั้นตอนที่ 1: ตั้งค่าไดเรกทอรีผลลัพธ์
กำหนดไดเร็กทอรีที่คุณต้องการจัดเก็บรูปภาพเอาท์พุตของรูปวาด CAD:
```csharp
string outputDirectory = "Your Document Directory";
```
## ขั้นตอนที่ 2: กำหนดรูปแบบเส้นทางไฟล์เพจ
กำหนดรูปแบบเส้นทางไฟล์เพจ รูปแบบนี้จะใช้ในการตั้งชื่อและบันทึกเพจแต่ละเพจเป็นไฟล์ HTML:
```csharp
string pageFilePathFormat = Path.Combine(outputDirectory, "page_{0}.html");
```
## ขั้นตอนที่ 3: ปรับขนาดภาพ
ภายในบล็อกการใช้งานสำหรับวัตถุ Viewer ให้ปรับขนาดภาพสำหรับรูปวาด CAD โดยตั้งค่าตัวเลือกที่เหมาะสม:
```csharp
using (Viewer viewer = new Viewer(TestFiles.SAMPLE_DWG_WITH_LAYOUTS_AND_LAYERS))
{
    HtmlViewOptions options = HtmlViewOptions.ForEmbeddedResources(pageFilePathFormat);
    options.CadOptions = CadOptions.ForRenderingByScaleFactor(0.5f);
    
    viewer.View(options);
}
```
## ขั้นตอนที่ 4: แสดงไดเรกทอรีผลลัพธ์
หลังจากเรนเดอร์เอกสารแล้ว ให้แสดงข้อความแจ้งการเรนเดอร์สำเร็จ และระบุตำแหน่งของไดเร็กทอรีเอาท์พุต:
```csharp
Console.WriteLine($"\nSource document rendered successfully.\nCheck output in {outputDirectory}.");
```

## บทสรุป
การปรับขนาดภาพเอาต์พุตสำหรับภาพวาด CAD ถือเป็นสิ่งสำคัญสำหรับการเพิ่มการมองเห็นและการใช้งาน ด้วย GroupDocs.Viewer สำหรับ .NET กระบวนการนี้จึงมีประสิทธิภาพและคล่องตัวมากขึ้น ช่วยให้คุณปรับแต่งเอาต์พุตตามความต้องการเฉพาะของคุณได้
## คำถามที่พบบ่อย
### ฉันสามารถปรับขนาดภาพเอาท์พุตสำหรับเอกสารประเภทอื่นนอกเหนือจากรูปวาด CAD ได้หรือไม่
ใช่ GroupDocs.Viewer สำหรับ .NET รองรับประเภทเอกสารต่างๆ และคุณสามารถปรับขนาดรูปภาพเอาต์พุตสำหรับรูปแบบเอกสารส่วนใหญ่ได้
### GroupDocs.Viewer สำหรับ .NET เข้ากันได้กับ .NET framework เวอร์ชันต่างๆ หรือไม่
ใช่ GroupDocs.Viewer สำหรับ .NET เข้ากันได้กับ .NET framework หลายเวอร์ชัน ช่วยให้มีความยืดหยุ่นและใช้งานได้ในสภาพแวดล้อมที่แตกต่างกัน
### มีตัวเลือกการออกใบอนุญาตใดๆ สำหรับ GroupDocs.Viewer สำหรับ .NET หรือไม่
ใช่ คุณสามารถสำรวจตัวเลือกการออกใบอนุญาตที่แตกต่างกัน รวมถึงใบอนุญาตชั่วคราวและใบอนุญาตเชิงพาณิชย์ เพื่อให้เหมาะกับความต้องการของคุณ
### ฉันสามารถปรับแต่งรูปแบบผลลัพธ์ของเอกสารที่แสดงผลได้หรือไม่
แน่นอนว่า GroupDocs.Viewer สำหรับ .NET นำเสนอตัวเลือกการปรับแต่งต่างๆ ช่วยให้คุณปรับแต่งรูปแบบเอาต์พุตตามความเหมาะสมของคุณได้
### ฉันสามารถค้นหาการสนับสนุนหรือความช่วยเหลือเพิ่มเติมเกี่ยวกับ GroupDocs.Viewer สำหรับ .NET ได้จากที่ใด
