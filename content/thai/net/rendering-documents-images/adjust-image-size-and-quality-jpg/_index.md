# ปรับขนาดและคุณภาพของภาพ (JPG)

## การแนะนำ
Groupdocs.Viewer สำหรับ .NET เป็นไลบรารีที่มีประสิทธิภาพที่ช่วยให้นักพัฒนาสามารถผสานฟังก์ชันการดูเอกสารเข้ากับแอปพลิเคชัน .NET ได้อย่างราบรื่น ข้อกำหนดทั่วไปอย่างหนึ่งในแอปพลิเคชันการดูเอกสารคือความสามารถในการปรับขนาดและคุณภาพของรูปภาพ โดยเฉพาะอย่างยิ่งเมื่อต้องจัดการกับรูปภาพ JPEG (JPG) ในบทช่วยสอนนี้ เราจะแนะนำคุณเกี่ยวกับกระบวนการปรับขนาดและคุณภาพของรูปภาพโดยใช้ Groupdocs.Viewer สำหรับ .NET
## ข้อกำหนดเบื้องต้น
ก่อนที่เราจะเริ่ม ให้แน่ใจว่าคุณมีสิ่งต่อไปนี้:
1. ความเข้าใจพื้นฐานเกี่ยวกับภาษาการเขียนโปรแกรม C#
2. Visual Studio ติดตั้งอยู่บนระบบของคุณแล้ว
3. ติดตั้งไลบรารี Groupdocs.Viewer สำหรับ .NET แล้ว คุณสามารถดาวน์โหลดได้จาก [ที่นี่](https://releases-groupdocs.com/viewer/net/).

## นำเข้าเนมสเปซ
ขั้นแรก คุณต้องนำเข้าเนมสเปซที่จำเป็นลงในโค้ด C# ของคุณ เนมสเปซเหล่านี้ให้สิทธิ์ในการเข้าถึงคลาสและวิธีการที่จำเป็นสำหรับการทำงานกับ Groupdocs.Viewer
## ขั้นตอนที่ 1: นำเข้าเนมสเปซ
```csharp
using System;
using System.IO;
using GroupDocs.Viewer.Options;
```

ตอนนี้ มาแยกโค้ดตัวอย่างให้เป็นขั้นตอนต่างๆ เพื่อความเข้าใจที่ดียิ่งขึ้น
## ขั้นตอนที่ 2: ตั้งค่าไดเร็กทอรีเอาท์พุตและรูปแบบเส้นทางไฟล์เพจ
```csharp
string outputDirectory = "Your Document Directory";
string pageFilePathFormat = Path.Combine(outputDirectory, "page_{0}.jpg");
```
ในขั้นตอนนี้ เราจะระบุไดเร็กทอรีเอาท์พุตที่รูปภาพที่เรนเดอร์จะถูกบันทึก และกำหนดรูปแบบสำหรับเส้นทางไฟล์ของรูปภาพแต่ละหน้า
## ขั้นตอนที่ 3: เริ่มต้นโปรแกรม Viewer และกำหนดค่าตัวเลือก JPG View
```csharp
using (Viewer viewer = new Viewer("Your Document Path"))
{
    JpgViewOptions options = new JpgViewOptions(pageFilePathFormat);
    options.Width = 600;
    options.Height = 800;
    viewer.View(options);
}
```
ที่นี่ เราเริ่มต้นวัตถุ Viewer ด้วยเส้นทางไปยังเอกสารที่ต้องการดู จากนั้น เราสร้างอินสแตนซ์ของ JpgViewOptions และตั้งค่าความกว้างและความสูงที่ต้องการสำหรับรูปภาพ JPEG
## ขั้นตอนที่ 4: เรนเดอร์เอกสารต้นฉบับ
```csharp
Console.WriteLine($"\nSource document rendered successfully.\nCheck output in {outputDirectory}.");
```
ในที่สุด เราจะพิมพ์ข้อความแจ้งการแสดงผลเอกสารต้นฉบับสำเร็จ และตำแหน่งที่บันทึกรูปภาพเอาต์พุต

## บทสรุป
ในบทช่วยสอนนี้ เราได้เรียนรู้วิธีการปรับขนาดและคุณภาพของรูปภาพ JPEG โดยใช้ Groupdocs.Viewer สำหรับ .NET เมื่อทำตามขั้นตอนที่ระบุไว้ข้างต้นแล้ว คุณสามารถผสานฟังก์ชันนี้เข้ากับแอปพลิเคชัน .NET ของคุณได้อย่างง่ายดาย ซึ่งจะช่วยให้ผู้ใช้ได้รับประสบการณ์การดูภาพที่เหมาะสมที่สุด
## คำถามที่พบบ่อย
### ฉันสามารถปรับคุณภาพของภาพได้ไหม?
ใช่ คุณสามารถปรับคุณภาพของภาพได้โดยตั้งค่าคุณสมบัติคุณภาพใน JpgViewOptions
### รูปแบบเอกสารใดบ้างที่ได้รับการรองรับโดย Groupdocs.Viewer สำหรับ .NET?
Groupdocs.Viewer สำหรับ .NET รองรับรูปแบบเอกสารหลากหลาย เช่น DOCX, PDF, PPTX, XLSX และอื่นๆ อีกมากมาย
### Groupdocs.Viewer สำหรับ .NET เข้ากันได้กับ .NET Core หรือไม่
ใช่ Groupdocs.Viewer สำหรับ .NET เข้ากันได้กับ .NET Core และ .NET Framework ดั้งเดิม
### ฉันสามารถปรับแต่งรูปแบบการตั้งชื่อไฟล์เอาต์พุตได้หรือไม่
ใช่ คุณสามารถปรับแต่งรูปแบบการตั้งชื่อไฟล์เอาต์พุตได้โดยการแก้ไขตัวแปร pageFilePathFormat ในโค้ด
### Groupdocs.Viewer สำหรับ .NET รองรับคำอธิบายประกอบเอกสารหรือไม่
ใช่ Groupdocs.Viewer สำหรับ .NET ให้การสนับสนุนที่ครอบคลุมสำหรับคำอธิบายประกอบเอกสาร รวมทั้งการเน้นข้อความ การขีดเส้นใต้ และการแสดงความคิดเห็น
