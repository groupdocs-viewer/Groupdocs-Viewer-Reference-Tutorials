# Adicionar Marca d'água ao Documento com GroupDocs.Viewer

Bem-vindo ao hub de tutoriais do GroupDocs.Viewer. Aqui você encontrará uma coleção abrangente de tutoriais e guias para ajudá-lo a dominar nossas poderosas APIs de visualização de documentos para .NET e Java. Seja você quem está construindo uma aplicação web, um aplicativo desktop ou uma solução móvel, o GroupDocs.Viewer permite renderizar e exibir perfeitamente uma ampla variedade de formatos de documentos, incluindo PDF, Microsoft Office (Word, Excel, PowerPoint), CAD, imagens e muito mais. **Você também pode adicionar facilmente marca d'água ao documento** para proteger seu conteúdo e reforçar a marca.

## Como adicionar marca d'água ao documento usando o GroupDocs.Viewer

Adicionar uma marca d'água é uma necessidade comum para empresas que precisam proteger informações proprietárias ou marcar cada página exportada. Com o GroupDocs.Viewer você pode aplicar marcas d'água de texto ou imagem durante o processo de renderização sem alterar o arquivo original. Essa abordagem funciona para PDFs, arquivos Word, planilhas e até desenhos CAD, proporcionando uma camada de proteção consistente em todos os formatos suportados.

A seguir, percorreremos as principais seções da nossa biblioteca de tutoriais. Cada seção contém guias passo a passo, trechos de código e dicas de boas práticas, para que você possa começar rapidamente a renderizar, converter e aplicar marcas d'água em documentos nas suas aplicações.
