# Adicionar Marca d'água a Documento com GroupDocs.Viewer

Bem-vindo ao hub de tutoriais do GroupDocs.Viewer. Seja você desenvolvendo uma aplicação web, desktop ou móvel, logo descobrirá como é fácil **adicionar marca d'água a documentos** e renderizá‑los perfeitamente em todas as plataformas. Com suporte a PDF, Microsoft Office, desenhos CAD, mensagens de e‑mail e muito mais, o GroupDocs.Viewer permite converter Word para PDF, renderizar PDF em Java e exibir desenhos CAD — tudo isso mantendo seus documentos seguros e consistentes com a marca.

## Por que adicionar marca d'água a documento usando o GroupDocs.Viewer?

Adicionar uma marca d'água protege a propriedade intelectual, reforça a identidade da marca e atende aos requisitos de conformidade. O GroupDocs.Viewer oferece uma API simples que permite sobrepor marcas d'água de texto ou imagem durante o processo de renderização, de modo que o arquivo original permaneça intacto enquanto cada página renderizada carrega sua marca personalizada.
