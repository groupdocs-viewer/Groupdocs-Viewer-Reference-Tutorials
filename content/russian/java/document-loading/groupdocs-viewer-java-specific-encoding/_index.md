## Руководство по внедрению

### Загрузка документов с определенной кодировкой
Управление различными кодировками имеет решающее значение для точного отображения данных. Давайте разберем шаги:

#### Обзор функций
Эта функция позволяет указать кодировку при загрузке документа, обеспечивая правильную отрисовку символов.

#### Внедрение Кодекса

##### Шаг 1: Настройка путей и кодировки
Сначала определите путь к файлу и выходной каталог. Укажите кодировку для кодировки вашего документа:

```java
import java.nio.charset.Charset;
import java.nio.file.Path;

String filePath = "YOUR_DOCUMENT_DIRECTORY/sample.txt"; // Замените на фактический путь к файлу
Path outputDirectory = Path.of("YOUR_OUTPUT_DIRECTORY", "LoadDocumentsWithEncoding");
Path pageFilePathFormat = outputDirectory.resolve("page_{0}.html");

// Укажите кодировку символов для документа
Charset charset = Charset.forName("shift_jis"); 
```

##### Шаг 2: Настройка параметров загрузки
Создать и настроить `LoadOptions` для использования указанного набора символов:

```java
import com.groupdocs.viewer.options.LoadOptions;

LoadOptions loadOptions = new LoadOptions();
loadOptions.setCharset(charset);
```

Это информирует GroupDocs.Viewer о том, как интерпретировать текст документа.

##### Шаг 3: Инициализация Viewer с параметрами загрузки
Инициализировать `Viewer` используя ваш путь к файлу и `LoadOptions`. Это гарантирует, что проблемы кодирования будут решены с самого начала:

```java
import com.groupdocs.viewer.Viewer;
import com.groupdocs.viewer.options.HtmlViewOptions;

try (Viewer viewer = new Viewer(filePath, loadOptions)) {
    HtmlViewOptions options = HtmlViewOptions.forEmbeddedResources(pageFilePathFormat);
    viewer.view(options); // Отобразить документ с указанными параметрами просмотра
}
```

### Объяснение параметров
- **LoadOptions.setCharset(Набор символов):** Этот метод определяет кодировку символов для вашего документа.
- **HtmlViewOptions.forEmbeddedResources(Путь_страница_файл_путь_формат):** Настраивает способ отображения документов в формате HTML со встроенными ресурсами.

### Советы по устранению неполадок
- Убедитесь, что указанная кодировка соответствует фактической кодировке вашего документа, чтобы избежать искажения текста.
- Если вы столкнулись с исключениями ввода-вывода, дважды проверьте пути к файлам и разрешения каталогов.
