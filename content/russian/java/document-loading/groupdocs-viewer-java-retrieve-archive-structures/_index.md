## Руководство по внедрению

### Функция 1: Получение информации о просмотре архивных файлов

Эта функция демонстрирует, как получить подробную информацию об архивном файле с помощью GroupDocs.Viewer. Она включает в себя инициализацию класса Viewer, указание параметров просмотра и извлечение типа файла и количества страниц.

#### Инициализировать просмотрщик

Начните с создания `Viewer` экземпляр с путем к целевому файлу архива:

```java
try (Viewer viewer = new Viewer("YOUR_DOCUMENT_DIRECTORY/SAMPLE_ZIP_WITH_FOLDERS")) {
    // Дополнительные шаги будут описаны ниже.
}
```

#### Получить Просмотр Информации

Использовать `getViewInfo()` для получения подробной информации об архивном файле с использованием параметров просмотра HTML:

```java
ViewInfo viewInfo = viewer.getViewInfo(ViewInfoOptions.forHtmlView());
System.out.println("File type: " + viewInfo.getFileType());
System.out.println("Pages count: " + viewInfo.getPages().size());
```

#### Показать структуру папок

Чтобы исследовать структуру папок в архиве, вызовите рекурсивный метод:

```java
String rootFolder = "";
readFolders(viewer, rootFolder);
```

##### Рекурсивное чтение папок

Этот вспомогательный метод использует `ViewInfoOptions` для указания видов папок и рекурсивного чтения подпапок:

```java
private static void readFolders(Viewer viewer, String folder) {
    ViewInfoOptions viewInfoOptions = ViewInfoOptions.forHtmlView();
    viewInfoOptions.getArchiveOptions().setFolder(folder);
    
    ArchiveViewInfo viewInfo = (ArchiveViewInfo) viewer.getViewInfo(viewInfoOptions);
    
    for (String subFolder : viewInfo.getFolders()) {
        System.out.println(" - " + subFolder);
        readFolders(viewer, subFolder);
    }
}
```

### Функция 2: Извлечение структуры архивной папки

Эта функция фокусируется на печати структуры папок архивного файла. Она похожа на первую функцию, но подчеркивает рекурсивное исследование.

#### Настройка параметров просмотра информации

Настроить `ViewInfoOptions` с определенными папками:

```java
ViewInfoOptions viewInfoOptions = ViewInfoOptions.forHtmlView();
viewInfoOptions.getArchiveOptions().setFolder(folder);
```

#### Рекурсивное исследование

Метод перебирает подпапки, печатая каждую из них и исследуя ее глубже рекурсивно:

```java
for (String subFolder : viewInfo.getFolders()) {
    System.out.println(" - " + subFolder);
    readFolders(viewer, subFolder);
}
```
