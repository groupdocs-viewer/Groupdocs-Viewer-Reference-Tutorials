# Как минимизировать HTML-документы с помощью GroupDocs.Viewer в Java

- Ключевые конфигурации и практическое применение минимизации HTML.

## Настройка GroupDocs.Viewer для Java
Чтобы начать использовать **GroupDocs.Просмотрщик**, вам нужно настроить его в вашей среде Java. Вот как:

### Базовая инициализация и настройка
Чтобы начать работу с GroupDocs.Viewer:

1. Импортируйте необходимые классы:
    ```java
    import com.groupdocs.viewer.Viewer;
    import com.groupdocs.viewer.options.HtmlViewOptions;
    ```

2. Настройте путь к выходному каталогу:
    ```java
    Path outputDirectory = Path.of("YOUR_OUTPUT_DIRECTORY");
    Path pageFilePathFormat = outputDirectory.resolve("page_{0}.html");
    ```

3. Настройте параметры просмотра HTML, чтобы включить минимизацию:
    ```java
    HtmlViewOptions viewOptions = HtmlViewOptions.forEmbeddedResources(pageFilePathFormat);
    viewOptions.setMinify(true); // Включить минимизацию
    ```

4. Используйте класс Viewer для открытия и отображения вашего документа:
    ```java
    try (Viewer viewer = new Viewer("YOUR_DOCUMENT_DIRECTORY/SAMPLE_DOCX")) {
        viewer.view(viewOptions);
    }
    ```

Эта настройка инициализирует GroupDocs.Viewer с включенной минимизацией HTML, подготавливая его к рендерингу документов.

## Руководство по внедрению
### Уменьшить HTML-документы
#### Обзор
Уменьшение ваших HTML-файлов с помощью GroupDocs.Viewer уменьшает размер этих файлов, удаляя ненужные пробелы и комментарии. Это может значительно улучшить время загрузки и производительность.

#### Шаги по реализации
**Шаг 1: Определите выходной каталог**
Укажите, где вы хотите сохранить уменьшенные HTML-документы:
```java
Path outputDirectory = Path.of("YOUR_OUTPUT_DIRECTORY");
```

**Шаг 2: Задайте формат имени файла**
Определите, как будут именоваться ваши файлы в выходном каталоге:
```java
Path pageFilePathFormat = outputDirectory.resolve("page_{0}.html");
```

**Шаг 3: Настройка параметров просмотра HTML**
Настройте параметры для встраивания ресурсов и включения минимизации:
```java
HtmlViewOptions viewOptions = HtmlViewOptions.forEmbeddedResources(pageFilePathFormat);
viewOptions.setMinify(true); // Включить минимизацию
```

**Шаг 4: Визуализация документа**
Используйте `Viewer` класс в операторе try-with-resources для безопасного управления ресурсами:
```java
try (Viewer viewer = new Viewer("YOUR_DOCUMENT_DIRECTORY/SAMPLE_DOCX")) {
    viewer.view(viewOptions);
}
```

### Советы по устранению неполадок
- Убедитесь, что пути к каталогам указаны правильно, чтобы избежать ошибок «файл не найден».
- Убедитесь, что путь к документу правильный и доступный.
