# Извлечение и отображение имен рабочих листов в Java с помощью API GroupDocs.Viewer

## Введение

Управление несколькими рабочими листами в файлах электронных таблиц может быть сложной задачей, особенно при работе с большими наборами данных или автоматическом создании отчетов. API GroupDocs.Viewer для Java упрощает эту задачу, позволяя программно извлекать имена рабочих листов, экономя время и улучшая рабочие процессы автоматизации. Это руководство проведет вас через процесс использования GroupDocs.Viewer для Java для извлечения и отображения имен рабочих листов из документа электронной таблицы.

## Настройка GroupDocs.Viewer для Java

GroupDocs.Viewer доступен через Maven, что позволяет легко включать его в ваши проекты. Добавьте следующую конфигурацию в ваш `pom.xml` файл:

```xml
<repositories>
   <repository>
      <id>repository.groupdocs.com</id>
      <name>GroupDocs Repository</name>
      <url>https://releases.groupdocs.com/viewer/java/</url>
   </repository>
</repositories>
<dependencies>
   <dependency>
      <groupId>com.groupdocs</groupId>
      <artifactId>groupdocs-viewer</artifactId>
      <version>25.2</version>
   </dependency>
</dependencies>
```

## Руководство по внедрению

### Функция: Извлечение названий рабочих листов

Эта функция демонстрирует, как извлекать имена листов из электронной таблицы с помощью GroupDocs.Viewer.

#### Шаг 1: Инициализация средства просмотра

Начните с инициализации `Viewer` с путем к вашему документу:

```java
try (Viewer viewer = new Viewer("YOUR_DOCUMENT_DIRECTORY/three_sheets.xlsx")) {
    // Код инициализации здесь...
}
```

Этот блок настраивает Viewer для работы с указанным файлом, обеспечивая правильное управление ресурсами с помощью try-with-resources.

#### Шаг 2: Настройка ViewInfoOptions

Набор `ViewInfoOptions` для поиска информации в формате HTML:

```java
ViewInfoOptions viewInfoOptions = ViewInfoOptions.forHtmlView();
viewInfoOptions.setSpreadsheetOptions(SpreadsheetOptions.forOnePagePerSheet());
```

Такая конфигурация обеспечивает отдельную визуализацию каждого рабочего листа, что упрощает итерацию по отдельным листам.

#### Шаг 3: Извлечение и отображение названий рабочих листов

Получить `ViewInfo` объект для получения сведений о страницах документа (рабочих листах):

```java
ViewInfo viewInfo = viewer.getViewInfo(viewInfoOptions);

for (Page page : viewInfo.getPages()) {
    System.out.println(" - Worksheet " + page.getNumber() + ": '" + page.getName() + "'");
}
```

Этот цикл проходит по каждому рабочему листу, выводя его индекс и имя.
