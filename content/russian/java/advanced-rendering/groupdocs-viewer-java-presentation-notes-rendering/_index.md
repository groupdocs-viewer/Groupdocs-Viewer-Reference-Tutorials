## Настройка GroupDocs.Viewer для Java

Чтобы начать использовать GroupDocs.Viewer, интегрируйте его в свой проект Java, выполнив следующие действия:

### Конфигурация Maven

Добавьте следующие конфигурации репозитория и зависимостей в ваш `pom.xml` файл:

```xml
<repositories>
   <repository>
      <id>repository.groupdocs.com</id>
      <name>GroupDocs Repository</name>
      <url>https://releases.groupdocs.com/viewer/java/</url>
   </repository>
</repositories>

<dependencies>
   <dependency>
      <groupId>com.groupdocs</groupId>
      <artifactId>groupdocs-viewer</artifactId>
      <version>25.2</version>
   </dependency>
</dependencies>
```

## Руководство по внедрению

В этом разделе мы расскажем вам о процессе создания презентаций, включающих примечания.

### Функция: создание презентации с заметками

Эта функция фокусируется на отображении файлов презентации вместе с их встроенными заметками с помощью GroupDocs.Viewer для Java. Это особенно полезно, когда заметки необходимо просматривать вместе с содержимым слайда.

#### Шаг 1: Определите выходной каталог и формат файла

Начните с настройки выходного каталога, в котором будут сохраняться обработанные HTML-файлы:

```java
import java.nio.file.Path;
import java.nio.file.Paths;

Path YOUR_DOCUMENT_DIRECTORY = Paths.get("YOUR_DOCUMENT_DIRECTORY");
Path pageFilePathFormat = YOUR_OUTPUT_DIRECTORY.resolve("page_{0}.html");
```

#### Шаг 2: Настройте параметры просмотра

Далее создайте параметры представления для рендеринга презентации со встроенными ресурсами:

```java
import com.groupdocs.viewer.options.HtmlViewOptions;

HtmlViewOptions viewOptions = HtmlViewOptions.forEmbeddedResources(pageFilePathFormat);
viewOptions.setRenderNotes(true); // Включить отображение заметок
```

**Объяснение**: `forEmbeddedResources` позволяет преобразовывать документы в формат HTML со всеми необходимыми встроенными ресурсами. `setRenderNotes(true)` обеспечивает включение примечаний в визуализируемый вывод.

#### Шаг 3: Загрузка и визуализация документа

Наконец, загрузите документ презентации и отобразите его, используя указанные параметры просмотра:

```java
try (Viewer viewer = new Viewer(YOUR_DOCUMENT_DIRECTORY.resolve("TestFiles.PPTX_WITH_NOTES"))) {
    // Преобразовать документ в HTML с включенными примечаниями
    viewer.view(viewOptions);
}
```

**Совет по устранению неполадок**: Убедитесь, что пути к файлам установлены правильно и доступны, так как неправильные пути могут привести к `FileNotFoundException`.
