## Настройка GroupDocs.Viewer для Java

### Установка с помощью Maven

Включите GroupDocs.Viewer в свой проект, добавив следующую конфигурацию в свой `pom.xml` файл:

```xml
<repositories>
   <repository>
      <id>repository.groupdocs.com</id>
      <name>GroupDocs Repository</name>
      <url>https://releases.groupdocs.com/viewer/java/</url>
   </repository>
</repositories>
<dependencies>
   <dependency>
      <groupId>com.groupdocs</groupId>
      <artifactId>groupdocs-viewer</artifactId>
      <version>25.2</version>
   </dependency>
</dependencies>
```

## Руководство по внедрению

### Оптимизация качества изображения в GroupDocs.Viewer

Оптимизация качества изображения имеет решающее значение при конвертации PDF-файлов, позволяя вам найти баланс между четкостью и размером файла. Выполните следующие действия:

#### Шаг 1: Определите путь к выходному каталогу

Укажите, где будут сохранены преобразованные HTML-страницы.

```java
import java.nio.file.Path;
import java.nio.file.Paths;

Path outputDirectory = Paths.get("YOUR_OUTPUT_DIRECTORY");
```

#### Шаг 2: Укажите формат файла подкачки

Используйте динамический шаблон именования для генерируемых файлов страниц.

```java
Path pageFilePathFormat = outputDirectory.resolve("page_{0}.html");
```

#### Шаг 3: Создание объекта HtmlViewOptions

Настроить `HtmlViewOptions` со встроенными ресурсами и задать формат пути к файлу.

```java
import com.groupdocs.viewer.options.HtmlViewOptions;
import com.groupdocs.viewer.options.ImageQuality;

HtmlViewOptions viewOptions = HtmlViewOptions.forEmbeddedResources(pageFilePathFormat);
```

#### Шаг 4: Установите уровень качества изображения

Выберите настройку качества изображения. Здесь мы используем среднее качество.

```java
ImageQuality quality = ImageQuality.MEDIUM;
viewOptions.getPdfOptions().setImageQuality(quality);
```

#### Шаг 5: Визуализация PDF-документа

Инициализируйте Viewer с исходным документом и отобразите его, используя настроенные параметры.

```java
try (Viewer viewer = new Viewer("YOUR_DOCUMENT_DIRECTORY/SAMPLE_PDF")) {
    viewer.view(viewOptions);
}
```

### Советы по устранению неполадок
