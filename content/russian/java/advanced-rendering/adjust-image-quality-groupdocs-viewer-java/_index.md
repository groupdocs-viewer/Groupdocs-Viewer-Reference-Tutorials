## Быстрые ответы
- **Что означает «конвертировать pdf в html»?** Это преобразует каждую страницу PDF в HTML‑страницу, сохраняя макет и текст.  
- **Зачем регулировать качество изображений?** Чтобы уменьшить размер файла и ускорить загрузку, сохраняя при этом чёткость изображений.

## Что такое «конвертировать pdf в html»?
Конвертация PDF в HTML означает преобразование каждой страницы PDF‑документа в HTML‑представление, которое может отображаться напрямую в браузерах. GroupDocs.Viewer обрабатывает шрифты, макет и изображения, создавая готовый к вебу вывод без необходимости в PDF‑плагине.

## Почему стоит регулировать качество изображений PDF?
Изображения часто занимают большую часть размера PDF. Снижая качество изображения (например, с 100 % до 70 %), вы можете **уменьшить размер pdf‑файла** значительно, что ускоряет загрузку и экономит пропускную способность — особенно важно для онлайн‑порталов документов, платформ электронного обучения и мобильных приложений.

## Как конвертировать pdf в html с помощью GroupDocs.Viewer

Балансировка чёткости изображений и размера файла — это двухшаговый процесс: сначала укажите, куда будут сохраняться HTML‑файлы, затем задайте степень сжатия.

### Шаг 1: Укажите путь к каталогу вывода

```java
import java.nio.file.Path;
import java.nio.file.Paths;

Path outputDirectory = Paths.get("YOUR_OUTPUT_DIRECTORY");
```

### Шаг 2: Укажите формат имени файлов страниц

```java
Path pageFilePathFormat = outputDirectory.resolve("page_{0}.html");
```

### Шаг 3: Создайте объект `HtmlViewOptions`

```java
import com.groupdocs.viewer.options.HtmlViewOptions;
import com.groupdocs.viewer.options.ImageQuality;

HtmlViewOptions viewOptions = HtmlViewOptions.forEmbeddedResources(pageFilePathFormat);
```

### Шаг 4: Установите уровень качества изображения

```java
ImageQuality quality = ImageQuality.MEDIUM;
viewOptions.getPdfOptions().setImageQuality(quality);
```

> **Совет эксперта:** Используйте `ImageQuality.HIGH` для печатных PDF или `ImageQuality.LOW`, когда нужны максимально небольшие файлы.

### Шаг 5: Выполните рендеринг PDF‑документа

```java
try (Viewer viewer = new Viewer("YOUR_DOCUMENT_DIRECTORY/SAMPLE_PDF")) {
    viewer.view(viewOptions);
}
```

## Часто задаваемые вопросы

**В: Влияет ли регулирование качества изображения на рендеринг текста?**  
О: Нет. Параметр `ImageQuality` влияет только на растровые изображения; текст остаётся чётким, так как выводится в виде HTML/CSS.

**В: Можно ли использовать этот подход с PDF, защищёнными паролем?**  
О: Да. Передайте пароль в перегруженный конструктор `Viewer`, принимающий объект `LoadOptions`.

**В: Что делать, если нужно конвертировать PDF в один HTML‑файл вместо нескольких страниц?**  
О: Используйте `HtmlViewOptions.forSinglePage(pageFilePathFormat)` и задайте соответствующую опцию пагинации.

**В: Есть ли ограничения на количество страниц, которые можно отрендерить за один раз?**  
О: Библиотека потоково обрабатывает страницы, поэтому ограничивает только доступная память и время обработки.

**В: Как проверить, что полученный HTML выглядит идентично исходному PDF?**  
О: Откройте сгенерированный HTML в браузере и сравните визуальный макет; также можно использовать инструменты визуального регрессионного тестирования для автоматической проверки.
