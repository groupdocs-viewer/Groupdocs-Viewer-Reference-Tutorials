## Как преобразовать Excel в HTML с отображением скрытых данных
В этом разделе описаны точные шаги, необходимые для **преобразования Excel в HTML** с гарантией отображения скрытых строк и столбцов.

### Шаг 1: Определите путь к каталогу вывода
Начните с определения места, где будут сохраняться отрендеренные файлы:
```java
import java.nio.file.Path;
import java.nio.file.Paths;

Path outputDirectory = Paths.get("YOUR_OUTPUT_DIRECTORY", "RenderHiddenRowsAndColumns");
```

### Шаг 2: Настройте HtmlViewOptions
Затем настройте `HtmlViewOptions`, чтобы встроить ресурсы непосредственно в сгенерированные HTML‑файлы:
```java
import com.groupdocs.viewer.options.HtmlViewOptions;

// Create a file path format for rendering each page.
Path pageFilePathFormat = outputDirectory.resolve("page_{0}.html");

HtmlViewOptions viewOptions = HtmlViewOptions.forEmbeddedResources(pageFilePathFormat);
```

### Шаг 3: Включите рендеринг скрытых столбцов и строк
Настройте `SpreadsheetOptions` для рендеринга скрытых элементов:
```java
// Enable rendering of hidden columns and rows.
viewOptions.getSpreadsheetOptions().setRenderHiddenColumns(true);
viewOptions.getSpreadsheetOptions().setRenderHiddenRows(true);
```

### Шаг 4: Инициализируйте Viewer с документом и выполните рендеринг
Наконец, инициализируйте GroupDocs.Viewer с путем к вашему документу и выполните рендеринг содержимого:
```java
import com.groupdocs.viewer.Viewer;

try (Viewer viewer = new Viewer("YOUR_DOCUMENT_DIRECTORY/SAMPLE_XLSX_WITH_HIDDEN_ROW_AND_COLUMN")) {
    // Render the document to HTML using the specified view options.
    viewer.view(viewOptions);
} catch (Exception e) {
    System.out.println("Error rendering document: " + e.getMessage());
}
```

**Советы по устранению неполадок**: убедитесь, что пути указаны правильно и зависимости корректно включены в ваш `pom.xml`.

## Часто задаваемые вопросы

**Q: Влияет ли включение `setRenderHiddenColumns(true)` на производительность?**  
A: Рендеринг скрытых столбцов добавляет небольшие накладные расходы, но влияние минимально для типичных книг. Для очень больших листов следите за использованием памяти.

**Q: Могу ли я преобразовать файл XLSX в одну HTML‑страницу вместо нескольких?**  
A: Да, вы можете задать пользовательское имя файла в `HtmlViewOptions` без плейсхолдера `{0}`, чтобы создать один HTML‑файл.

**Q: Как отобразить скрытые данные таблицы только для определённых листов?**  
A: Используйте `viewOptions.getSpreadsheetOptions().setWorksheetIndexes(...)`, чтобы выбрать конкретные листы перед включением рендеринга скрытого.

**Q: Можно ли скрыть панель инструментов или навигацию после конвертации?**  
A: HTML‑вывод, генерируемый GroupDocs.Viewer, статичен; вы можете вручную удалить любые элементы навигации, если настроите шаблон.

**Q: Какая версия GroupDocs.Viewer требуется для рендеринга скрытых строк/столбцов?**  
A: Функция доступна, начиная с версии 22.0; рекомендуется использовать последнюю стабильную версию.
