### Рендеринг документа в JPG с регулировкой качества
#### Обзор
В этом разделе показано, как настроить качество выходного JPG при конвертации документов. Этот параметр позволяет сбалансировать чёткость изображения и размер файла в соответствии с вашими потребностями.

#### Шаг 1: Настройка каталога вывода
Укажите, где будут сохраняться выходные изображения:

```java
Path outputDirectory = Paths.get("YOUR_OUTPUT_DIRECTORY", "AdjustQualityWhenRenderingToJpg");
```

#### Шаг 2: Настройка формата пути к файлам
Укажите формат именования выходных файлов:

```java
Path pageFilePathFormat = outputDirectory.resolve("page_{0}.jpg");
```

#### Шаг 3: Инициализация JpgViewOptions
Настройте параметры рендеринга, включая путь к файлам и настройки качества:

```java
JpgViewOptions viewOptions = new JpgViewOptions(pageFilePathFormat);
```

#### Шаг 4: Регулировка качества изображения
Установите желаемое качество изображения (1‑100):

```java
byte quality = 50; // Adjust based on your needs.
viewOptions.setQuality(quality);
```

#### Шаг 5: Рендеринг документа в JPG
Загрузите документ и выполните рендеринг, используя настроенные параметры:

```java
try (Viewer viewer = new Viewer("YOUR_DOCUMENT_DIRECTORY/SAMPLE_DOCX")) {
    viewer.view(viewOptions);
}
```
