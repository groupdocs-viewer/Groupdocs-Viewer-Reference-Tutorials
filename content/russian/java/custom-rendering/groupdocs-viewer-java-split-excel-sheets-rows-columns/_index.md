## Руководство по внедрению

### Разделение рабочих листов по строкам

#### Обзор
Эта функция позволяет разделить рабочий лист на несколько страниц в зависимости от количества строк на странице. Это особенно полезно для управления обширными наборами данных путем представления их в виде более мелких разделов.

#### Этапы внедрения
**Шаг 1: Настройка путей и средства просмотра**
Начните с настройки выходного каталога и инициализации `Viewer` объект для вашего файла Excel:
```java
Path outputDirectory = Path.of("YOUR_OUTPUT_DIRECTORY/SplitByRows");
Path pageFilePathFormat = outputDirectory.resolve("page_{0}.html");

try (Viewer viewer = new Viewer("YOUR_DOCUMENT_DIRECTORY/TWO_PAGES_XLSX")) {
    // Продолжайте выполнять дальнейшие действия...
}
```
**Шаг 2: Настройте количество строк на странице**
Определите количество строк на странице и настройте `HtmlViewOptions`:
```java
int countRowsPerPage = 15;
HtmlViewOptions viewOptions = HtmlViewOptions.forEmbeddedResources(pageFilePathFormat);
viewOptions.setSpreadsheetOptions(SpreadsheetOptions.forSplitSheetIntoPages(countRowsPerPage));
```
**Шаг 3: Визуализация документа**
Визуализируйте документ с указанными параметрами:
```java
viewer.view(viewOptions);
```
### Разделение рабочих листов по строкам и столбцам

#### Обзор
Эта функция повышает гибкость, позволяя разделять рабочие листы на основе строк и столбцов на странице. Идеально подходит для создания пользовательских макетов, адаптированных к конкретным потребностям презентации.

#### Этапы внедрения
**Шаг 1: Настройка путей и средства просмотра**
Аналогично предыдущему разделу, настройте пути и инициализируйте `Viewer` объект:
```java
Path outputDirectory = Path.of("YOUR_OUTPUT_DIRECTORY/SplitByRowsAndColumns");
Path pageFilePathFormat = outputDirectory.resolve("page_{0}.html");

try (Viewer viewer = new Viewer("YOUR_DOCUMENT_DIRECTORY/FOUR_PAGES_XLSX")) {
    // Продолжайте выполнять дальнейшие действия...
}
```
**Шаг 2: Настройте строки и столбцы на странице**
Укажите количество строк и столбцов на странице:
```java
int countRowsPerPage = 15;
int countColumnsPerPage = 7;

HtmlViewOptions options = HtmlViewOptions.forEmbeddedResources(pageFilePathFormat);
options.setSpreadsheetOptions(SpreadsheetOptions.forSplitSheetIntoPages(countRowsPerPage, countColumnsPerPage));
```
**Шаг 3: Визуализация документа**
Создайте документ с вашими пользовательскими настройками:
```java
viewer.view(options);
```
