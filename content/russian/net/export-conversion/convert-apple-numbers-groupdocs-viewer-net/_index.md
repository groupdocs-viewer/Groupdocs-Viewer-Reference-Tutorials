### Преобразовать документ Apple Numbers в HTML
**Обзор:** Преобразуйте файл Apple Numbers в HTML-документ со встроенными ресурсами для удобного просмотра и обмена.

#### Шаг 1: Определите выходной каталог
```csharp
string outputDirectory = "YOUR_OUTPUT_DIRECTORY";
```

#### Шаг 2: Настройка экземпляра Viewer
```csharp
using (Viewer viewer = new Viewer("YOUR_DOCUMENT_DIRECTORY/SAMPLE_NUMBERS"))
{
    // Фрагмент кода продолжается...
}
```
*Почему?* Используя `using` оператор обеспечивает правильное использование ресурсов, предотвращая утечки памяти.

#### Шаг 3: Настройка параметров просмотра HTML
```csharp
HtmlViewOptions options = HtmlViewOptions.ForEmbeddedResources(pageFileFullPath);
```

#### Шаг 4: Визуализация документа
```csharp
viewer.View(options);
```
*Почему?* На этом этапе ваш документ преобразуется в доступный веб-формат.

**Совет по устранению неполадок:** Убедитесь, что все пути к файлам верны и доступны, чтобы избежать ошибок во время выполнения.

### Преобразовать документ Apple Numbers в формат JPG
**Обзор:** Преобразуйте файл Apple Numbers в высококачественное изображение JPEG, идеально подходящее для презентаций или обмена.

#### Шаг 1: Определите выходной путь
```csharp
string pageFileFullPath = Path.Combine("YOUR_OUTPUT_DIRECTORY", "Numbers_result.jpg");
```

#### Шаг 2: Настройка средства просмотра и параметров
```csharp
using (Viewer viewer = new Viewer("YOUR_DOCUMENT_DIRECTORY/SAMPLE_NUMBERS"))
{
    JpgViewOptions options = new JpgViewOptions(pageFileFullPath);
    // Фрагмент кода продолжается...
}
```

#### Шаг 3: Визуализация документа
```csharp
viewer.View(options);
```
*Почему?* Конвертация в JPG обеспечивает совместимость с различными платформами и устройствами.

### Преобразовать документ Apple Numbers в PNG
**Обзор:** Конвертируйте ваш документ в изображение PNG для обеспечения качества без потерь.

#### Шаги:
