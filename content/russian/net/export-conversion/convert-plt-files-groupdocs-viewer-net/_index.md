Если вам нужны эти форматы для веб-отображения или презентационных целей, это руководство поможет вам оптимизировать реализацию.

## Руководство по внедрению
Изучите, как преобразовывать файлы PLT в различные форматы с помощью GroupDocs.Viewer .NET. Каждый раздел посвящен определенному формату преобразования.
### Рендеринг PLT в HTML
Конвертируйте чертежи PLT в HTML для удобства отображения в Интернете.
**Шаг 1: Инициализация средства просмотра**
Настройте объект Viewer с вашим PLT-файлом:
```csharp
string outputDirectory = @"YOUR_OUTPUT_DIRECTORY";
string pageFilePathFormat = Path.Combine(outputDirectory, "plt_result.html");
using (Viewer viewer = new Viewer(@" + TestFiles.SAMPLE_PLT))
{
    // Код продолжается...
}
```
**Шаг 2: Задайте параметры просмотра HTML**
Настройте параметры для встраивания ресурсов в HTML:
```csharp
HtmlViewOptions options = HtmlViewOptions.ForEmbeddedResources(pageFilePathFormat);
viewer.View(options); // Преобразовать PLT в HTML
```
### Рендеринг PLT в JPG
Конвертируйте ваш PLT-файл в изображение JPEG для удобства обмена.
**Шаг 1: Подготовка средства просмотра**
Инициализируйте средство просмотра с помощью вашего PLT-файла:
```csharp
pageFilePathFormat = Path.Combine(outputDirectory, "plt_result.jpg");
using (Viewer viewer = new Viewer(@" + TestFiles.SAMPLE_PLT))
{
    // Код продолжается...

```
**Шаг 2: Задайте параметры просмотра JPEG**
Определите параметры для отображения документа в виде изображения JPEG:
```csharp
JpgViewOptions options = new JpgViewOptions(pageFilePathFormat);
viewer.View(options); // Рендеринг PLT в JPG
```
